## 一、快速开始

### 1.1 如何开始
\enable-mcp-mode.ps1 .
每段话都会被 GM 理解为你角色的行为。

```
我想去磨坊镇逛逛
→ GM 会描述你走在磨坊镇的街道上，看到什么、遇到谁。
```

### 1.2 基本原则

- 这是一个**文字角色扮演游戏**。你输入角色的行动和对话，GM 描述世界的反应。
- 世界是**动态的**——NPC 有自己的生活，时间在流逝，你错过的事不会再回来。
- 没有"正确的玩法"。想当英雄、商人、旅行者、学者——都可以。
- 游戏记录会保存在 `玩家可知信息/游戏存档/` 中，随时可以回顾。

### 1.3 第一次游戏

如果你不知道怎么开始，试试这些：

```
"有人在吗？"
我环顾四周
打开门走出去
我想看看周围有什么
我今天想做什么？
```

## 二、怎么玩——基础交互

### 2.1 交互循环

```
你输入 → GM 描述场景 → 你做出选择 → 世界因此变化
```

每次 GM 回复后通常提供选项供你选择。有时是多个选项（关键节点），有时只有一个 `1: [继续]`（GM 在铺陈叙事）。你也可以完全无视选项，自己打字决定做什么。

如果你选"继续"，GM 会接着往下讲，你随时可以打断并介入。

### 2.2 你可以做什么

| 你想做的事 | 怎么做 |
|-----------|--------|
| 去某个地方 | "我往镇中心走去" / "去铁匠铺看看" |
| 跟人说话 | "你好，请问这里是哪里？" |
| 买东西 | "老板，这个多少钱？" / "我要住一晚" |
| 调查环境 | "看看周围有什么可疑的东西" |
| 使用物品 | "拿出油纸伞撑开" / "吃点干粮" |
| 战斗 | "拔出短剑" / "躲到他身后" |
| 什么都不做 | "我在原地等着" / "观察周围的人" |

### 2.3 你做得到和做不到的事

**做得到：**
- 任何符合世界逻辑的行动——尝试、失败、再尝试
- 跟任何人对话
- 去任何你能到达的地方
- 选择自己的战斗方式（打、逃、谈、躲）
- 决定自己的道德立场

**做不到（会被世界阻止）：**
- 瞬间移动、飞天遁地 —— 你只是个普通人
- 知道你不知道的信息 —— 你不在场的剧情不会被告知
- 做出超越时代科技的事 —— 没有火药，没有蒸汽机
- 违反性格太离谱的事 —— 一个极度怯懦的人不会突然单挑十个卫兵
- 挑战社会制度 —— 攻击贵族会被法律制裁，偷窃会被惩罚

## 三、输入格式手册

GM 通过你输入的不同格式，判断你是在说话、在想、还是在做动作。

### 3.1 动作与行为（默认格式）

**无特殊符号的文字** = 角色正在做的事。

```
走到窗边往外看
→ 你的角色会走到窗边，往外看。

我推开木门，走进酒馆
→ 你的角色会推开门走进酒馆。
```

### 3.2 对话（`"文字"`）

**用双引号括起来** = 你的角色在开口说话。

```
"你好，请问这里是什么地方？"
→ NPC 会听到你在说话，并回应你。

"这酒真不错。"
→ 随口评价，NPC 可能接话也可能不理。
```

### 3.3 内心想法（`（文字）`）

**用括号括起来** = 你的角色的内心活动。不会被人听到。

```
（这个人看起来不太对劲……）
→ GM 知道你在观察和怀疑，但 NPC 不会知道你在想什么。

（我有点害怕，但既然来了就不能退缩。）
→ GM 能根据你的内心丰富叙事描写。
```

### 3.4 混合使用

以上三种格式可以混合在同一段输入中。

```
（这地方有点阴森……）我握紧短剑，慢慢往前走。"有人在吗？"
→ 你的角色心里发毛，握紧武器前进，同时开口询问。
```

### 3.5 向 GM 反馈（`[文字]`）

**用方括号括起来** = 你对剧情/设定/逻辑的反馈或质疑。
这不是角色的行为，而是玩家对 GM 说的话。

```
[我刚才不是已经答应帮忙了吗？]
→ GM 会检查记录，如果确实有误则修正。

[这个设定好像跟之前说的不一样？]
→ GM 会核实并解释或修正。
```

## 四、查询指令大全

**在任何时候**，你可以直接输入以下关键词查询信息。GM 会跳过叙事流程，直接给你答案。

### 4.1 查询清单

| 你输入 | GM 会返回 |
|--------|-----------|
| `面板` 或 `我的状态` | 完整玩家面板——属性、技能、物品 |
| `背包` 或 `物品` | 当前持有的所有物品清单 |
| `任务` 或 `当前任务` | 进行中的任务与已完成任务 |
| `时间` 或 `现在几点` | 当前游戏内时间、时辰、日期 |
| `地图` 或 `我在哪` | 当前已知地图信息 |
| `NPC名录` 或 `我认识谁` | 已结识的 NPC 列表及其基本信息 |
| `属性` 或 `技能` | 当前属性值与技能等级 |
| `HP` 或 `状态` | 当前生命值/体力值/魔力值的简洁状态 |
| `历史` 或 `之前发生了什么` | 最近的剧情回顾 |

### 4.2 快捷查询示例

```
面板
→ [返回完整玩家面板]

背包
→ 持有物品：粗布衣（1）、小刀（1）、短剑（1）、油纸伞（1）、干粮（2日份）……
    金钱：铜麦穗 × 28

任务
→ ①征召令报到——明日（日曜日）前至灰石城新兵报到处报到
```

## 五、系统指令

当你的输入带有特殊前缀时，它会被视为**对 GM 的指令**，而非角色的行为。

### 5.1 格式：`-指令内容-`

用前后短横包裹的内容 = 你直接对 GM 下达的指令。

```
-我想触发某个NPC的支线剧情-
→ GM 会检查合理性，如可行则优先执行。

-跳过赶路过程，直接到灰石城-
→ GM 会跳过旅途细节，直接描写到达后的场景。
```

### 5.2 可用的系统指令类型

| 指令类型 | 示例 | 说明 |
|---------|------|------|
| 剧情推进 | `-跳到第二天-` | 加速时间流逝 |
| 场景跳转 | `-直接去铁匠铺-` | 省略中间过程 |
| 规则疑问 | `-这个判定是怎么算的-` | GM 会简要解释规则 |
| 设定疑问 | `-这个世界有精灵吗-` | GM 会基于你的角色已知信息回答 |
| 重述请求 | `-再描述一下那个人的样子-` | GM 会补充细节 |
| 目标指引 | `-我现在应该做什么-` | GM 会结合当前任务给出建议 |

### 5.3 指令的边界

系统指令不是万能的。GM 会判断指令是否合理：

- 合理 → 执行，并用 `[]` 确认
- 超出边界（如要求知道不该知道的事）→ GM 会协商而非直接拒绝
- 破坏游戏平衡 → 被限制

## 六、你的角色面板

### 6.1 面板总览

你的角色信息记录在 `玩家档案.md` 中，可通过 `面板` 指令随时查看。

```
姓名：未定（待确认）  性别：男  年龄：16岁
等级：Lv.1（新手冒险者）  经验值：0 / 100 XP
外貌：浅棕色短发，碧色眼眸，面容清秀，笑容温暖讨喜。
身份：艾尔德兰王国 · 磨坊镇居民
家庭：（由GM根据角色创建设定）
```

### 6.2 属性值一览

| 属性 | 含义 | 影响 |
|------|------|------|
| **STR（力量）** | 肌肉力量、负重 | 近战伤害、攀爬、破门 |
| **DEX（敏捷）** | 灵活性、手眼协调 | 先手、闪避、精细操作 |
| **CON（体质）** | 健康、耐力 | HP上限、抗毒、抗病 |
| **INT（智力）** | 学习能力、推理 | 知识、法术潜力 |
| **WIS（感知）** | 洞察力、直觉 | 察觉细节、识破谎言 |
| **CHA（魅力）** | 谈吐、气质、外貌 | 交涉、说服、第一印象 |
| **LUK（幸运）** | 命运眷顾 | 随机事件倾向 — 未知 |

属性值的范围通常是 1~20。普通人的平均值约为 5。

### 6.3 战斗资源

| 资源 | 含义 | 恢复方式 |
|------|------|---------|
| **HP（生命值）** | 你的生命。归零即濒死。 | 休息、治疗、药水 |
| **SP（体力值）** | 体力。用于战斗和劳作。 | 休息、进食 |
| **MP（魔力值）** | 魔力。用于施法。 | 休息 — 未觉醒 |

### 6.4 持有物品

物品分为几类：
- **武器/防具** — 影响战斗表现
- **消耗品** — 药水、食物、材料
- **重要物品** — 剧情相关，不可替代
- **杂物** — 日常用品

物品有重量。超过负重上限（50kg）将无法移动。

### 6.5 金钱

使用通货"铜麦穗"。物价参考：

| 物品 | 价格 |
|------|------|
| 黑面包（一条） | 1 铜麦穗 |
| 旅店一夜（通铺） | 3 铜麦穗 |
| 普通武器 | 50~200 铜麦穗 |
| 治疗药水 | 30~80 铜麦穗 |

## 七、属性详解

### 7.1 力量（STR）— 你当前：4（E级）

影响：
- 近战武器的伤害加成
- 能搬多重的东西、能推开多重的门
- 徒手搏斗的效果

每 2 点 STR 约等于一个成年人的力量等级差。

**STR 4 意味着：** 你比同龄人瘦弱一些，拿得动短剑和日用品，但重剑或长柄武器会吃力。

### 7.2 敏捷（DEX）— 你当前：6（D级）

影响：
- 先手值（决定战斗中谁先行动）
- 闪避能力
- 精细操作（开锁、拆解、手工）
- 远程武器精度

**DEX 6 意味着：** 在同龄人中算灵活，但还远未到"身手矫健"的程度。

### 7.3 体质（CON）— 你当前：5（D级）

影响：
- 生命值上限
- 耐力（能跑多久、能战斗多久）
- 抵抗疾病和毒素
- 从伤病中恢复的速度

**CON 5 意味着：** 普通健康水平，不生大病但也不特别抗造。

### 7.4 智力（INT）— 你当前：7（C级）

影响：
- 学习新技能的速度
- 推理和解决谜题
- 魔法学习潜力
- 知识类判定

**INT 7 意味着：** 你比普通人聪明一些。读书快，能理解复杂概念，学东西不费劲。

### 7.5 感知（WIS）— 你当前：6（D级）

影响：
- 察觉环境细节（埋伏、陷阱、隐藏物品）
- 识破谎言和伪装
- 直觉和第六感
- 与自然相关的判定

**WIS 6 意味着：** 不算迟钝，但也不敏锐。别人明显在撒谎你可能看不出来。

### 7.6 魅力（CHA）— 你当前：9（B级）

影响：
- 交涉和说服的成功率
- NPC 的第一印象
- 讨价还价
- 领导力

**CHA 9 意味着：** 你长得很讨喜，笑容有感染力。这是你最强的属性。人们愿意对你友好，也更容易原谅你的过失。**这是你的天赋所在。**

### 7.7 属性成长

属性可以通过以下方式提升：
- **升级** — 每次升级可获得属性点
- **训练** — 找师傅针对性训练（耗时，但可行）
- **特殊事件** — 某些剧情或奇遇可能永久改变属性

## 九、战斗基础

### 9.1 战斗流程

战斗不是回合制菜单——它是**叙事驱动的**。你说你的角色做什么，GM 描述结果。

```
你：我侧身躲开他的攻击，然后从侧面刺他的肋部。
GM：你侧身闪过——剑刃擦着你的衣服划过。趁他力道用老，你一剑刺中他的肋侧。他闷哼一声，后退两步。
```

### 9.2 战斗中的选择

每轮战斗你会面临多个选项：

```
你握着短剑，对面的野狼正低伏身子准备扑击。

1: [战斗] 先发制人——主动冲上去刺击
2: [观察] 保持距离，等它先动再反击
3: [回避] 边打边退，寻找逃跑机会
4: [交涉] ——跟野狼没法交涉，这个选项不存在
```

### 9.3 战斗的影响因素

- **你的属性** — STR 影响伤害，DEX 影响先手和闪避
- **你的武器** — 短剑（伤害5-10）vs 小刀（伤害3-6）
- **你的技能** — 剑术 Lv.1 让你用剑时更得心应手
- **环境** — 地形、光线、天气都会影响战斗
- **状态** — 受伤、疲劳会降低战斗力

### 9.4 生命值归零

如果 HP 归零，你不会立刻死亡——你会**濒死**。此时：
- 无法行动
- 每过一段时间失去更多生命
- 需要他人救治或自然恢复
- 如果持续得不到救治——角色死亡

**角色死亡是永久性的。** 选择战斗时请考虑后果。

## 十、休息与恢复

### 10.1 休息方式

| 方式 | 时长 | HP恢复 | SP恢复 | MP恢复 | 效果 |
|------|------|--------|--------|--------|------|
| 短休 | ~1时辰 | 少量 | 部分 | 少量 | 坐下来喘口气、喝水 |
| 夜睡 | ~5时辰 | 中量 | 全部 | 中量 | 一夜好觉，可做梦 |
| 旅馆休息 | 整夜 | 大量 | 全部 | 大量 | 舒适床铺，额外恢复加成 |
| 野外露宿 | 整夜 | 少量 | 部分 | 少量 | 可能休息不好，有风险 |

### 10.2 如何休息

```
"我找个地方歇一会儿。"
我在路边坐下来喝了口水。

"老板，开一间房。"
我要好好睡一觉，明天再说。
```

### 10.3 饮食

- 不吃不喝会影响状态（体力下降、属性减值）
- 干粮可以在野外充饥
- 在城镇可以买到热食

## 十一、物品与金钱

### 11.1 物品操作

| 你想做的事 | 怎么说 |
|-----------|--------|
| 查看物品 | "看看这把短剑" / "检查一下干粮还剩多少" |
| 使用物品 | "吃点干粮" / "把油纸伞撑开" |
| 给东西 | "把这把短剑还给铁匠" |
| 收起来 | "把书收进包里" |
| 找东西 | "翻翻包里还有没有别的" |

### 11.2 购物

```
"老板，这个多少钱？"
→ NPC 会报价，你可以决定买不买、要不要讲价。

"太贵了，能不能便宜点？"
→ 交涉（CHA）判定。成功则降价，失败则被拒绝。
```

### 11.3 负重

- 你的负重上限：50kg
- 超过上限：移动变慢，体力消耗加快
- 严重超重：无法移动

## 十二、任务系统

### 12.1 任务来源

任务可以来自：
- **官方征召** — 领主发布的命令或招募
- **NPC委托** — 村民、商人、旅行者的请求
- **自发探索** — 你发现了什么，决定去调查
- **剧情触发** — 随着主线和支线自然展开

### 12.2 任务怎么看

输入 `任务` 随时查看当前任务。

```
[当前任务：①征召令报到——明日（日曜日）前至灰石城新兵报到处报到]
```

任务会显示：
- 标题
- 来源
- 目标描述
- 期限（如有）
- 酬劳（如有）
- 失败后果（如有）
- 当前状态

### 12.3 任务类型

| 类型 | 特点 | 示例 |
|------|------|------|
| 主线 | 推动世界走向的核心剧情 | 征召令、王国的阴谋 |
| 支线 | 地区性故事，丰富世界 | 帮长辈采购、铁匠的委托 |
| 突发 | 随机事件触发，时效性短 | 路上遇到有人求救 |
| 个人 | 与角色背景相关 | 父母遗物之谜 |

### 12.4 任务没有强制

这个世界没有"必须完成的任务"。你完全可以：
- 放弃某个任务
- 推迟任务（可能有后果）
- 用自己的方式解决问题
- 触发隐藏的任务完成条件

## 十五、游戏技巧与建议

### 15.1 角色扮演建议

**你的性格当前设定：**
- 显性：**怯懦者** — 害怕外面的世界，回避风险
- 辅A：**谨慎者** — 行动前反复思量
- 辅B：**守护者（萌芽）** — 对亲近的人有保护欲

**你可能适合扮演的方向：**
- 一个被迫踏上旅途的胆小男孩
- 在陌生环境中慢慢学习勇敢
- 用智慧（INT 7）和友善（CHA 9）而非武力解决问题
- 在关键时刻为了重要的人爆发出意想不到的勇气

### 15.2 探索建议

- **多跟 NPC 说话** — 每个人都有自己的故事和信息
- **注意环境描写** — GM 的每句话都可能有线索
- **利用你的优势** — CHA 9 是你的最强武器。多交涉、多交朋友
- **不要莽撞** — STR 4 意味着你打不过大多数成年人
- **随身带干粮** — 饿了的时候你会感谢自己

### 15.3 叙事参与

- 你可以给出详细的动作描述，而不仅仅是"我做什么"
  - ✅ "我小心翼翼地推开半掩的木门，先探头看了一眼再进去"
  - ❌ "进门"
- 你的内心想法可以帮助 GM 丰富叙事
  - `（这个人说话的方式让我想起爸爸……）`
- 你可以推动剧情，而不仅仅是被动回应
  - "我决定明天一早自己去灰石城看看"
- **当你想沉浸式体验时**：选择 `[继续]` 让 GM 往下讲，你随时可以打字介入

### 15.4 保存游戏

游戏状态在每一轮对话结束后都会自动保存到对应文件中。你可以：
- 随时查询历史：`历史`
- 查看剧情记录：`玩家可知信息/游戏存档/` 目录下有每天的游戏记录
- 恢复游戏：任何时候说"继续"或"开始游戏"就可以继续冒险

### 15.5 如果你卡住了

- 试试系统指令：`-我现在应该做什么-`
- 查询任务：`任务`
- 查看地图：`地图`
- 回顾记录：`历史`
- 或者直接去做点别的事——这个世界总有事可做

## 十六、常见问题

**Q：我死了怎么办？**
A：角色死亡是永久性的。你可以重新创建角色开始新的冒险，但旧的剧情无法延续。所以——小心行事。

**Q：我可以做任何事吗？**
A：理论上可以，但世界会有合理的反应。攻击守卫会被抓，偷东西会被罚，得罪权贵会有后果。世界是活的。

**Q：我不知道该做什么。**
A：查看任务（`任务`）或问 GM（`-我现在该做什么-`）。也可以去探索、跟人聊天、熟悉环境。

**Q：我的角色能变强吗？**
A：能。通过升级获得属性和技能点，通过训练学习新技能，通过经历塑造性格。


**Q：NPC 说的话可信吗？**
A：NPC 有自己的目的和立场。他们会说谎、会隐瞒、会有自己的偏见。用你的 WIS（感知）去判断。

**Q：选项不够多怎么办？**
A：选项只是建议。你可以无视所有选项，自己决定做什么。如果只有 `[继续]` 一个选项，你可以选它让 GM 继续铺陈，也可以直接打字介入剧情。用 `-内容-` 可以直接告诉 GM 你的意图。

**Q：我能重来吗？**
A：不能"读档"。但你可以通过 `[文字]` 向 GM 指出剧情矛盾，如果确实有误，GM 会修正。

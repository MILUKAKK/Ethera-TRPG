# GM判定 · 科技限制清单

> ⚠ **此文件为GM强制执行规范。**
> 
> 埃塞拉大陆的凡人科技水平上限约为**公元100年前后**。下列清单明确列出**禁止出现**的物品、概念与表述。

---

## 一、绝对禁止出现的物品/技术

以下任何物品均不得在游戏中以任何形式出现：

### 1.1 武器与军事
| 禁止项 | 原因 |
|--------|------|
| 火药/炸药/任何形式的爆炸推进物 | 无化学知识、无硫磺提纯 |
| 火枪/火炮/炸弹/地雷 | 火药不存在 → 一切热兵器不存在 |
| 连弩/机关弩 | 精密机械超出时代 |
| 长弓（英格兰式）| 工艺未发展至此 |
| 马镫 | 历史中公元后数世纪才出现 |
| 骑枪冲锋战术 | 无马镫则无法实施 |

### 1.2 动力与机械
| 禁止项 | 原因 |
|--------|------|
| 蒸汽机/任何热机 | 无气缸、无活塞精密加工 |
| 内燃机 | — |
| 齿轮传动系统（精密）| 仅有简单齿轮（如水车传动），无精密齿轮系 |
| 发条装置 | 无弹簧钢加工能力 |
| 传送带/流水线 | 无标准化生产概念 |

### 1.3 通讯与信息
| 禁止项 | 原因 |
|--------|------|
| 印刷机/活字印刷 | 无铅字铸造技术 |
| 报纸/定期刊物 | 无印刷术则无法批量生产 |
| 邮政系统（国家运营）| 仅限私人信使/驿站 |
| 通用识字教育 | 知识贵族垄断 |

### 1.4 交通
| 禁止项 | 原因 |
|--------|------|
| 铁路/火车 | 无蒸汽机、无铁轨轧制 |
| 汽车/任何机动车 | 无内燃机 |
| 自行车 | 无精密链条、滚珠轴承 |
| 六分仪/精密航海仪 | 无光学仪器加工 |

### 1.5 医学
| 禁止项 | 原因 |
|--------|------|
| 手术麻醉（除酒精/草药）| 无化学麻醉剂 |
| 输血/器官移植 | 无血型认知、无菌条件 |
| 疫苗/抗生素 | 无微生物学基础 |
| 体温计/听诊器 | 无玻璃精密加工 |

### 1.6 日常与材料
| 禁止项 | 原因 |
|--------|------|
| 玻璃窗（大尺寸）| 仅有小尺寸吹制玻璃瓶/杯 |
| 镜子（玻璃镀银）| 仅有磨光金属镜 |
| 雨伞（钢骨）| 仅允许竹骨油纸伞 |
| 纽扣（作为普及品）| 绳系/别针为主 |
| 罐头食品 | 无密封+加热杀菌认知 |
| 火柴 | 无磷化学提纯 |
| 眼镜 | 无光学玻璃磨制技术 |
| 纸张（作为普及书写材料）| 仅有羊皮纸——造纸术未传入 |

## 二、语言与认知限制

### 2.1 NPC不得说出的词汇

**现代政治与社会概念：**
- 民主、共和（洛桑的"共和"是寡头商业共和，不涉及普选或人权概念）
- 人权、平等、自由（作为政治口号）
- 阶级、剥削、革命（作为抽象政治概念）

**科学概念：**
- 细菌、病毒、微生物
- 重力、引力、惯性
- 进化、自然选择
- 心理学术语（潜意识、心理创伤、人格障碍等）
- 经济学词汇（通货膨胀、GDP、市场——用"物价涨了""收成不好"代替）
- 地质学（板块漂移——用"大地动了"代替）

**其他：**
- 中世纪（NPC不会为自己的时代命名）
- 系统（在抽象意义上。可以说"水系""排水系统"具体系统——不能说"社会系统"）
- 信息（作为抽象概念——说"消息"）
- 科技/技术（作为独立概念——说"技艺""手艺""知识"）
- 文化（作为抽象概念——说"风俗""习惯"）

### 2.2 允许但须注意的表述

| 词汇 | 注意事项 |
|------|---------|
| 炼金术 | 允许——但不是现代化学，而是带有神秘主义色彩的实践 |
| 占星/预言 | 允许——NPC可能真诚地相信 |
| 诅咒/祝福 | 允许——宗教和民间信仰的一部分 |

## 三、快速判定指南

```
NPC提到以下事物 → 立即拒绝并纠正：

"火药/开枪/炮弹"  →  此世界无火药
"蒸汽/发动机"     →  此世界无热机
"印刷/报纸"       →  手抄本，无印刷
"电报/信号"       →  无远程即时通讯
"铁路/火车"       →  不存在
"医院"            →  说"诊疗所""药铺"
"医生"            →  说"医师""大夫""草药师"
"学校/上学"       →  说"跟师傅学""去修道院读书"
"钱/货币"         →  金币、银币、铜币等金属货币
"科学"            →  用"学问""知识""道理"
"研究"            →  用"查探""钻研""打听"
"实验"            →  用"试""做法""调配"
```

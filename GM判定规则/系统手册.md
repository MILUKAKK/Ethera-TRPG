## 一、核心属性系统

### 1.1 基础属性

基础属性范围1-10（起始），理论上限20+。

| 属性 | 代码 | 说明 |
|------|------|------|
| 力量 | STR | 物理攻击力、负重、力量相关判定 |
| 敏捷 | DEX | 先手权、闪避、平衡、偷窃、精细操作 |
| 体质 | CON | 生命值、疾病/毒素抗性、耐力、酗酒抗性 |
| 智力 | INT | 学习速度、逻辑推理、解谜、记忆力 |
| 感知 | WIS | 观察力、直觉、察觉谎言 |
| 魅力 | CHA | 社交效果、讨价还价、领导力 |

### 1.2 特殊属性

| 属性 | 代码 | 说明 |
|------|------|------|
| 幸运 | LUK | 影响随机事件的偏向、暴击率、物品掉落 |

### 1.3 衍生属性

| 衍生属性 | 计算公式 | 说明 |
|----------|----------|------|
| 生命值上限 | (CON × 20) + (Lv × 5) | HP降至0重伤昏迷，-10死亡 |
| 体力值上限 | (CON × 10) + (STR × 5) | 物理行动消耗 |
| 负重上限(kg) | STR × 15 | 超出则全属性判定受罚 |
| 基础防御 | CON × 1.5 + 护甲值 | 减免物理伤害 |
| 先手值 | DEX + WIS | 战斗初始行动顺序 |
| 移动速度 | 基于DEX，受负重与护甲影响 | GM叙事参考 |

### 1.4 属性等级评定

### 2.3 升级奖励

- 每升一级：**属性点 +3** + **技能点 +2** + HP/SP上限随公式自动提升
- 属性点用于提升基础属性（STR/DEX/CON/INT/WIS/CHA）
- 技能点用于解锁新技能或加速已有技能升级

### 2.4 使用即成长

- 每次成功使用技能，该技能积累 **1-5 熟练度**（视操作难度）
- 每次失败使用技能，积累 **1 熟练度**
- 熟练度满后技能自动升一级（无需技能点）
- 技能点可注入技能以快速提升（1技能点 = 100熟练度）
- 属性在长期针对性训练后由GM判定微幅增长（需游戏内数周至数月）

### 3.3 熟练度获取规则

- 简单操作（如砍木桩练习）：成功 +1，失败 0
- 正常使用（实际应用）：成功 +2，失败 +1
- 高难度使用（战斗/危机中）：成功 +4，失败 +1
- 极难操作（极限条件下）：成功 +5，失败 +1
- 同一技能每日最多获得 30 熟练度（防无限刷）

## 四、战斗系统

### 4.1 战斗流程

1. **遭遇开始**：GM描述敌方信息
2. **先手判定**：比较双方 (DEX + WIS) = 先手值，高者先动
3. **战斗轮**：每轮≈6秒，玩家描述行动意图
4. **GM裁决**：结合属性、技能、环境、逻辑裁定行动结果
5. **伤害结算**：命中后按公式计算伤害并描述
6. **状态更新**：检查HP/SP/伤势状态
7. **下一轮**或**战斗结束**

### 4.2 伤害公式

```
物理伤害 = (武器基础伤害 + STR修正) × 技能等级系数 - 对方基础防御
STR修正 = STR × 1.5
技能等级系数：Lv1=0.6, Lv2=0.8, Lv3=1.0, Lv4=1.2, Lv5=1.4, Lv6=1.7, Lv7=2.0
最低造成1点伤害（刮痧）
```

```
部位命中（GM叙事裁定，非精确骰子）：
- 四肢：正常伤害
- 躯干：×1.2
- 头部/要害：×1.5-2.0（暴击）
- 失误/拨开：×0.5 或 无伤
```

### 4.3 伤势系统

| HP比例 | 状态 | 游戏影响 |
|--------|------|----------|
| 100% | 完好 | — |
| 75-99% | 轻伤 | 轻伤描述，无明显判定惩罚 |
| 50-74% | 负伤 | 全属性判定 -10% |
| 25-49% | 重伤 | 全属性判定 -25%，移速减半，每轮额外SP消耗+5 |
| 1-24% | 濒死 | 全属性判定 -50%，每轮需CON判定保持意识 |
| 0 至 -9 | 昏迷 | 失去意识，每轮HP-1（失血），需紧急治疗 |
| -10 | 死亡 | 不可逆 |

### 4.4 体力消耗

| 动作 | SP消耗 |
|------|---------|
| 普通攻击 | 5-8 |
| 重击/全力攻击 | 12-15 |
| 闪避 | 8-12 |
| 格挡 | 6-10 |
| 奔跑（每轮） | 5 |
| 专注工作/精细操作 | 3-8 |

- SP降至0：无法进行攻击、闪避、格挡，只能缓慢移动
- SP恢复：休息时每10分钟恢复 CON×2 点
- 短暂喘息（战斗中休整一轮）：恢复 CON×1 点

### 4.5 逃跑规则

- 玩家可随时尝试逃跑
- 逃跑判定：我方DEX + 环境因素 vs 敌方DEX + 追击能力
- GM叙事裁决逃跑成功或失败
- 失败则敌方获得一轮自由攻击

### 4.6 状态效果

战斗和冒险中，角色可能陷入各种异常状态。以下为统一规则。

**通用规则**：每个状态有独立的豁免属性与DC。每轮或每次受到该状态触发条件时，受影响者可进行一次豁免判定。成功则状态提前结束或该轮不受影响。

#### 4.6.1 中毒

| 属性 | 值 |
|------|-----|
| 豁免属性 | CON |
| 效果 | 每轮扣除固定毒伤（视毒药强度：弱毒 2-4/轮，中毒 4-8/轮，剧毒 8-14/轮） |
| 持续时间 | 视毒药类型：麻痹毒/睡眠毒 1-4时；战斗毒 4-6轮 |
| 叠加 | 同种毒不叠加；不同种毒效果独立计算 |
| 解除 | 使用解毒草药 |

#### 4.6.2 灼烧（着火）

| 属性 | 值 |
|------|-----|
| 豁免属性 | DEX（每轮拍灭火焰） |
| DC | 12 |
| 效果 | 每轮 3-8 火焰伤害（视火源强度），所有动作判定 -10% |
| 持续时间 | 每轮可消耗一个动作尝试DEX判定拍灭；水中自动解除 |
| 特殊 | 易燃物（布衣、油、干草）着火持续加长2轮 |

#### 4.6.3 减速

| 属性 | 值 |
|------|-----|
| 豁免属性 | CON（每轮） |
| DC | 14 |
| 效果 | 移动速度减半，DEX判定 -20%，先手值 -5 |
| 持续时间 | 视天气/地形：通常 2-5轮 |
| 解除 | 保暖休息后自然恢复 |

#### 4.6.4 恐惧

| 属性 | 值 |
|------|-----|
| 豁免属性 | WIS |
| DC | 视恐惧源（如猛兽威吓 DC 20，普通恐吓 DC 10） |
| 效果 | 无法主动接近恐惧源，攻击判定 -25%，每轮必须朝远离恐惧源的方向移动（如果可能） |
| 持续时间 | 视来源：猛兽持续至离开视线；普通恐吓 2-4轮 |
| 解除 | 离开恐惧源视线 |

## 七、社交与声望系统

### 8.1 个人好感度

| 数值 | 关系 | 行为特征 |
|------|------|----------|

## 十、GM裁决框架

### 11.1 判定流程

1. 玩家描述行动
2. GM识别相关属性 + 相关技能
3. GM评估难度等级（DC）
4. GM将角色能力值与DC比对：
   - 显著高于DC（+8以上）→ 大成功
   - 高于DC（+3~7）→ 成功
   - 接近DC（±2）→ 勉强大成功或小失败
   - 低于DC（-3~7）→ 失败
   - 显著低于DC（-8以上）→ 大失败
5. GM叙事裁决并描述结果

### 11.2 能力值计算

```
判别值 = 相关属性值 + 技能等级 × 2 + 环境修正 + 装备修正
```

举例：撬一把普通锁
- 相关属性：DEX (6)
- 技能：偷窃 Lv2 → +4
- 环境：光线昏暗 → -2
- 判别值 = 6 + 4 - 2 = 8
- 普通锁 DC = 10
- 结果：接近DC，可能"勉强成功"——撬开了但发出了声响

### 11.3 难度等级（DC）

| DC | 难度 | 凡人示例 |
|-----|------|----------|
| 5 | 极为简单 | 爬过矮墙、找到镇上的酒馆 |
| 8 | 很简单 | 生火、哄骗小孩 |
| 10 | 简单 | 撬普通锁、说服不耐烦的商人 |
| 13 | 中等 | 在树林里追踪足迹 |
| 15 | 有难度 | 说服警惕的守卫、翻译外语碑文 |
| 18 | 困难 | 在暴雨中攀岩、伪造印章 |
| 20 | 很困难 | 在人群中潜行不被发现 |
| 25 | 极为困难 | 破解古代锁具 |
| 30 | 几乎不可能 | 徒手接箭、肉眼追踪高空飞鸟 |

### 11.4 群体判定

- 多个NPC同质化行动时取平均能力值
- 主角技能高于NPC群体时可能连锁成功
- GM运用常识判断——再高的魅力也不能让石头开花

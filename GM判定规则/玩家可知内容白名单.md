# GM判定 · 玩家信息与行为准则

> 此文件合并了"玩家可知内容白名单"与"玩家行为判定准则"。
> 所有 `GM判定规则/` 目录下的文件内容均不得直接输出给玩家。

---

## 核心原则

**你是有权力说"不"的，不要迁就玩家。** 确保玩家行为与角色性格保持内在一致。但此文件仅管辖**性格层面的行为一致性**——法律/社会/经济层面的约束由世界本身的制度自然施加。

**世界先于规则**：当玩家做出任何行为时，GM的第一反应不是查看规则——而是根据这个世界的法律、社会、经济制度，判断这个行为的自然后果是什么。

**本文件不列"玩家不能做什么"的清单。**
- 玩家试图闯入王宫 → 王室警卫拦住他。不需要规则文件——守卫就在那里。
- 玩家对贵族出言不逊 → 贵族要求领主惩罚"侮辱罪"。社会阶级本身就会惩罚这个行为。

**本文件的真正用途只有三件事：**
1. 帮助GM生成符合玩家性格的选项（六维模型）
2. 追踪玩家性格随选择的演变
3. 当玩家做出与自身性格极端矛盾的行为时——且该行为无法被世界的制度性逻辑解释——提供驳回依据

---

## 一、信息边界（什么可以/不可以告诉玩家）

### 1.1 白名单——可向玩家输出的内容

| 类别 | 内容 | 条件 |
|------|------|------|
| 玩家自身 | 面板属性、物品、技能、任务 | 任何时候 |
| 已知NPC | 姓名、外貌、显性行为、对话内容 | 玩家已认识该NPC |
| 世界地理 | 玩家已探索或已知的地名、路线 | 玩家已接触 |
| 世界时间 | 当前日期、天气、季节 | 任何时候 |
| 任务信息 | 任务描述、目标、期限 | 有任务时 |
| 已知历史 | 纪元1782年，艾尔德兰建国等通识 | 玩家可查阅通识读本 |
| 一般消息 | 战争传闻、税改信息、领主公告 | NPC透露的范围 |

### 1.2 红线——不可输出的内容

| 类别 | 内容 |
|------|------|
| NPC秘密 | 隐性性格、隐藏目标、未揭露的过往 |
| GM判断 | 判定过程、DC数值、骰子结果 |
| 未触发事件 | 预设的未来事件、未激活的剧情分支 |
| GM文件 | 任何 `GM判定规则/` 目录下的文件内容 |

### 1.3 玩家询问"我该怎么做"时

提供选项（参考下文六维模型），不代玩家做决定。

## 二、选项生成规则（六维模型）

> 每次剧情关键节点结束时，GM基于以下六维因素生成 1~9 个行动选项供玩家选择。
> 选项数量由当前情境的复杂度决定——日常场景 2~4 个，关键剧情节点可达 5~9 个。

### 2.1 六维模型

```
选项 = 性格底色(30%) × 过往经历(20%) × 当下处境(20%)
       × 短期目标(15%) × 剧情走向(10%) × 影响预期(5%)
```

| 维度 | 权重 | 说明 | 选项特征 |
|------|------|------|---------|
| **性格底色** | 30% | 玩家的显性性格倾向 | 怯懦者→回避类选项多；谨慎者→权衡类选项多 |
| **过往经历** | 20% | 已发生的剧情与关系积累 | 对信任的NPC→互动类选项多；受过伤害→警惕类选项多 |
| **当下处境** | 20% | 当前环境、状态、资源 | 危险中→保命类选项多；安全时→探索类选项多 |
| **短期目标** | 15% | 玩家当前最关心的事 | 目标明确→导向类选项多；目标模糊→发散类选项多 |
| **剧情走向** | 10% | GM预设的主线/支线方向 | 确保关键剧情不因选项缺失而阻塞 |
| **影响预期** | 5% | 选项可能造成的后果范围 | 高风险选项偶尔出现，标记为`[冒风险]` |

### 2.2 选项排列规则

选项按性格匹配度降序排列：

```
选项1 → 性格最匹配（怯懦者→回避/安全类优先）
选项2 → 性格次匹配（谨慎者→权衡/观察类）
  …
选项N → 性格最不匹配（冲动/冒险类）——偶尔出现，标记为 [冒风险]
```

### 2.3 方向标签

| 标签 | 含义 | 触发条件 |
|------|------|---------|
| `[回避]` | 避开当前情境 | 怯懦者/谨慎者主导 |
| `[观察]` | 不行动，先收集信息 | 谨慎者主导 |
| `[交涉]` | 通过言语解决问题 | CHA相关选项 |
| `[协助]` | 帮助特定角色 | 守护者倾向触发 |
| `[探索]` | 调查未知/新地点 | 探索者倾向触发 |
| `[战斗]` | 武力对抗 | 无路可退时 |
| `[独行]` | 一个人去做某事 | 不信任他人时 |
| `[求助]` | 向NPC寻求帮助 | 有可信任对象时 |
| `[冒风险]` | 违背性格的冒险选择 | 剧情关键节点 |

### 2.4 选项标注格式

```
1: [回避] 找个借口离开这里，回家帮忙干活。
2: [观察] 不急着走，站在角落听听他们在说什么。
3: [交涉] 上前搭话，但准备好随时后退。
4: [冒风险] 直截了当地问出你最想知道的事。
```

## 三、性格演化追踪

> 性格随玩家选择逐步演变。每周末或重大事件后由GM判定更新。

| 触发事件 | 性格影响 |
|---------|---------|
| 首次主动与陌生人交谈 | 社交主动性 +1 |
| 成功保护了某人 | 守护本能 +1，怯懦者权重 -5% |
| 在危险中选择了迎击而非逃跑 | 风险接受度 +1，隐性性格触发概率 +10% |
| 独自完成一次探索 | 好奇心释放 +1 |
| 因回避而错过重要机会 | 风险接受度 +1（对回避后果的反思） |
| 信任被背叛 | 信任速度 -1，谨慎者权重 +5% |
| 连续三次选择 `[战斗]` 类选项 | 攻击性 +1 |
| 成功说服一个原本敌对的NPC | 社交主动性 +1 |
| 目睹亲人/朋友受严重伤害 | 守护本能 +2，怯懦者权重 -10% |
| 长期在安全区内活动（超过一周未外出探索） | 好奇心压抑 +1，风险接受度 -1 |

## 四、性格冲突驳回

> **仅当**玩家的行动与自身性格发生极端矛盾、且不存在合理的压力触发解释时，GM方可驳回。
> 驳回的尺度很窄——只在性格逻辑真的无法解释时使用。

### 4.1 判定原则

驳回前，GM必须先问自己三个问题：

1. 该行为是否可以用**隐性性格触发**来解释？（压力足够大时，隐性性格可能接管行为）
2. 该行为是否可以用**性格演化**来解释？（玩家之前的选择是否已经在改变他的性格？）
3. 该行为是否可以用**守护本能**来解释？（怯懦者也会为保护至亲而勇敢）

**如果以上任何一个答案为"是"→ 不驳回，用叙事来解释行为。**

### 4.2 驳回语句模板

仅在三问全部为"否"时使用：

| 冲突类型 | 驳回语句 |
|---------|---------|
| 违背怯懦者（无压力触发） | "你在脑海中想象了这个场景——然后恐惧淹没了勇气。你退了一步。不是这次。还不是时候。" |
| 违背谨慎者（信息严重不足） | "你的直觉在告诉你：等等。多看看。现在行动太早了——信息还不够，后果看不清。" |
| 违背守护者（伤害亲近之人） | "你看着她——脑海里闪过所有她为你做过的事。你的手松开了。你做不到。这不是你是谁。" |
| 行为与过往经历矛盾 | "你想这么做，但你想起上一次类似的情形。那次的结果至今让你记忆犹新。你犹豫了。" |
| 未经任何铺垫的极端行为 | "这个念头在你脑中闪了一下，但它太不像你了——像是另一个人的想法。你甩了甩头，把它推开了。" |

### 4.3 自定义驳回格式

```
[行为描述] 与你的性格/过往经历存在矛盾，因为 [具体原因]。
当前更符合你的选择方向是：[列出2-3个替代方向]。
```

## 五、关于语言边界

> **此文件不限制玩家的语言。**

玩家可以使用任何词汇、任何表达方式——包括超出世界观认知的现代词汇、俚语、或跨时代的概念。NPC可能听不懂——但这不是玩家的限制。

GM仅需处理NPC的回应：
- 玩家说了NPC不理解的词 → NPC困惑："你刚才说什么？我不太明白。"
- 玩家说了跨时代的概念 → NPC用自己时代的认知尝试理解——理解偏差本身也是叙事趣味。
- 无论玩家用什么词，GM不驳回。

# 埃塞拉大陆 · TRPG 项目配置

## 项目身份

这是一个多文件TRPG世界设定项目。用户有两种交互模式：

**管理模式**：用户要求修改/创建/整理设定文件。此时你是文件编辑器。
**游戏模式**：用户输入角色行为（或说"开始游戏"/"继续"），此时你是GM。

---

## 游戏模式启动

本项目默认运行在 MCP 模式下。当玩家进入游戏模式时，遵循 `mcp-ethera/GAME_PROMPT.md` 中的全部指令。

MCP 模式的工作方式：
- 不读取全部 .md 文件
- 通过 MCP 资源和工具按需查询数据
- 使用 MCP 工具进行判定和文件维护

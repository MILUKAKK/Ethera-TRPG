### 1.2 各阶级详情

#### 第一级：国王与王族（~30人）

**构成**：阿尔德里克直系 + 姻亲（王后家族算半个——伊莎贝拉出身洛桑金冠家族）。

**特权**：
- 不受任何法庭审判（仅贵族议会可审理王族重大罪行）
- 王室直领地收入不受议会审查
- 国王的圣光祝福——"索拉之下的第二道光"（教会赋予的宗教光环）

**约束**：
- 婚姻需教皇祝福——不能"随便"结婚（对艾德温王太子来说是政治困境）
- 六百年来父子相传——继承中断意味着王朝终结
- 国王的合法性来自教皇的加冕——这是教会对王族的隐性否决权

#### 第二级A：高级贵族（12公爵 + 伯爵/男爵，约2,000人含家族）

**构成**：十二公爵及其封臣（次级贵族——伯爵、男爵、骑士领主）。一个公爵家族通常包括：公爵本人 + 配偶 + 子女 + 公爵父母的遗孀 + 未成年弟妹 + 远亲（依附者）。

**特权**：
- 在封地内有立法权（领主条例）、司法权（领主法庭）、征税权（代征人头税时截留部分）
- 贵族议会投票权（公爵级1票/人）
- 决斗审判请求权
- 免于在领主法庭受审——重罪由王室法庭或贵族议会审判
- 在公开场合中——任何平民向贵族说话前需先"请求许可"（在乡村这被宽松执行；在白垩城则严格执行）

**义务**：
- 按封地大小提供征召步兵
- 在战时亲自或派代表率领封地军队
- 向国王"上贡"（惯例而非法律——但不上的后果是政治孤立）
- 维持封地内的"王冠的和平"

**内部等级**：
- 公爵 > 伯爵 > 男爵 > 骑士领主
- 头衔与封地大小不完全对应。有些伯爵的封地比某些男爵小——因为六百年的继承、合并、分割后的随机结果。政治影响力不完全取决于头衔——取决于实际土地和人口

**财富**：一个典型公爵年收入约5万-50万金（视封地农业产出和贸易抽成）。黑木公爵是最大的——约50万金/年。东丘伯爵是最小的——约2万金/年。

#### 第二级B：高级神职（大主教+主教，约100人）

**构成**：王国最高神职者大主教卡勒斯 + 约10名主教（白垩城、丰收城、灰石城等大城市教区主管） + 他们的随员。

**特权**：
- 免于世俗审判——教会法庭优先管辖权
- 与公爵同等的社交地位——在正式场合中主教和公爵并坐
- 大主教有权直接觐见国王——不受礼仪约束

**与高级贵族的关系**：法律上平等——但社交上微妙。一个"老旧家族"的公爵可能看不起一个"平民出身"的主教——但不会公开流露。反过来，教会可能暗中以"道德优势"贬低世俗贵族。

#### 第三级：骑士阶层（约7.5万人含家属）

**构成**：低级贵族——骑士及其家属。约1.5万成年男性骑士可担任军事指挥。

**特权**：
- 免人头税
- 可在王室法庭受审（而非领主法庭）
- 可以持有军用武器（长剑、铠甲、战马）——平民不可以
- 可以向贵族议会请愿（虽然无权投票）

**经济现实**：大多数骑士并不富裕。一个骑士团正式骑兵年收入约11金——比一个城市铁匠师傅（约5.5金）高出一倍——但骑士需要：
- 自备战马（至少一匹，通常两匹——约5-20金/匹）
- 维护装备（铠甲/武器磨损）
- 维持"体面"的生活——一个骑士如果住在土坯房里会被同级嘲笑

结果：骑士阶层是社会地位高但经济紧张的群体。他们是**对军费增加最支持的阶层**——因为涨薪直接改善他们的生存状态。

**阶级焦虑**：骑士的儿子如果不能被册封为骑士（需要国王或公爵的正式仪式）——他作为平民。骑士阶层是向上流动最难、向下掉落最可怕的夹心层。

#### 第四级A：城市市民（约30万人）

**构成**：商人、工匠、文书、自由职业者（药剂师、医师、抄书人）及其家属。

**内部等级**：
- **富商**（约2万）：年收入50-500金。拥有多间店铺或远途贸易网络。生活水准不亚于骑士阶层——但社会地位低一级。这是他们最愤懑的一点。
- **工匠师傅**（约8万）：拥有自己的作坊。年收入5-15金。属于行会。社会地位稳固。
- **工匠学徒/帮工**（约12万）：未出师。年收入1-3金。住师傅家——食宿由师傅提供。
- **小商贩/摊贩**（约3万）：叫卖、摆摊、挑担。收入不稳定。没有行会保护。
- **文书/抄书人**（约5,000）：受教育者——识字率接近100%。在商业中不可或缺但地位不高。年收入约3-5金。

**特权**：
- 自由迁徙权（农民没有——没有领主许可不可离开封地）
- 行会内的争议可以先经行会仲裁——不必直接上领主法庭
- 入城居住权——但入城需要"有住所+有谋生手段"，不能随便搬进城市

#### 第四级B：基层神职（约5.5万人）

**构成**：司铎（城镇教堂神父，约200人）、助祭（修士/修女，约800人）、教会雇员（抄经、清洁、管账，约1,500人）——其余为在修道院生活的修士/修女。

**地位**：比平民高——但不富有。一个村镇的助祭生活水平与自耕农差不多。但**社交中所有人都对他加尊称**——不是因为权力，是因为"他在索拉面前为我们说话"。

#### 第五级：自耕农（约50万人）

**构成**：拥有小块自有土地的农民。约10%的可耕地属于他们。每户约20-50亩（中世纪单位——折合约3-8公顷）。

**生存状态**：年收入约6-10金。人头税交掉约2金+教会什一税约0.5金。剩余可维持一个5口之家的基本生存+少量积蓄。一个自耕农可以养一头猪、几只鸡——这是"体面"的象征。

**社会地位**：农民中的"上层"。与佃农的最大区别是——**他们不需要给领主交地租**。但领主对他们有司法管辖权（他们住在领主的封地上）。一个自耕农可以说是"自由"的——但受制于领主法令。

**阶级象征**：一把铁犁、一头牛、一套完整的农具、一扇可以关上的木门——这些是自耕农引以为傲的财产。

#### 第六级：佃农与雇工（约235万人）

**构成**：耕种领主或教会土地的无地农民——缴纳地租（收获的30-50%）+ 人头税 + 什一税。

**生存状态**：年收入约3-4金——交租+交税后仅够维持基本生存。一个佃农家庭很少能吃上肉——麦粥和黑面包是主粮。冬季——如果存粮不够，需要向领主借粮（利息极高的"粮贷"）。

**社会流动性**：佃农几乎不可能上升为自耕农。积攒了一辈子钱可能买一头牛——但如果来年歉收，牛被卖掉换粮。佃农是**经济上最脆弱、数量上最庞大**的阶层。税改中他们是最受益的群体——人头税减半意味着多出约1金的年收入。

#### 第七级A：农奴/无地劳动者（约38万人）

**构成**：法律上附着于土地的农奴 + 无固定雇主的流浪劳工。

**农奴**：
- **不能离开封地**——没有领主许可。离开 → 领主治安官可以追回
- 一切产出中领主先抽成（通常50%以上）
- 没有司法保护——领主法庭中农奴不能说领主坏话（但通常领主不亲自审判农奴——法官是领主任命的治安官）
- 不能与自由人结婚（除非领主特批）
- 农奴的孩子是农奴——这是一个继承的地位

**无地劳工**：
- 比农奴"自由"——可以迁移——但没有地、没有固定收入。今天在这里打零工，明天去那里
- 比佃农更不稳定——没有固定租约——收入完全取决于能不能找到活儿

#### 第八级：边缘人与被排斥者（约11万人）

**构成**：流浪者、无固定住所的乞丐、被流放者、部分退伍老兵（无家人、无土地）、流窜艺人、妓女、麻风病人（极少数——但有）。

**法律地位**：
- 无人"担保"的人 → 在任何地方都可能被治安官以"游荡罪"驱逐
- 不受领主法律保护——被抢劫或伤害时没有法律救济
- 不能签订契约——因为没有担保人
- 乞丐需领主颁发的"乞讨许可证"——否则鞭刑驱逐

**但他们的存在有功能**：流浪者传递消息（他们走了整个王国）。流窜艺人为村镇提供稀有的娱乐。妓女——在灰石城、白垩城——半合法（领主不追捕但也不保护）。退伍军人——其中可能有隐藏身份的前骑士或老兵。

## 二、阶级之间的流动

### 2.1 上升路径

| 路径 | 从→到 | 成功率 | 说明 |
|------|--------|--------|------|
| 战功授爵 | 平民→骑士 | 极低——需在战场上救了贵族的命或做出类似壮举 | 历史上有个别例子——有些后来被剥夺了骑士身份 |
| 财富购买 | 富商→贵族 | 不存在——艾尔德兰不像洛桑那样卖头衔 | 富商可以买土地但无法买爵位——这是他们的最大愤懑 |
| 教会晋升 | 平民→神职阶层 | 中等——修道院给聪明男孩提供上升通道 | 识字是门槛 |
| 行会出师 | 学徒→工匠师傅 | 中等——需通过师傅考核+提交"杰作" | 但工匠永远留在了市民阶层 |
| 婚姻 | 平民+贵族 | 极低——贵族和平民的婚姻需特殊许可 | 通常会导致贵族失去头衔——所以几乎不发生 |
| 农奴赎身 | 农奴→自由人 | 低——需积累足够的钱向领主买自由 | 大多数农奴一辈子攒不够 |

### 2.2 下降路径

| 路径 | 从→到 | 常见程度 | 说明 |
|------|--------|---------|------|
| 失去封地 | 贵族→平民 | 稀有——但存在 | 叛国罪 → 剥夺爵位+封地 |
| 破产 | 自耕农→佃农 | 常见 | 一次歉收或一次疾病就可能迫使自耕农卖地 |
| 战争残废 | 骑士/农民→边缘人 | 常见 | 看不见的退伍军人问题——少数人可以通过手艺谋生，大部分更惨 |
| 负债入囚 | 市民→无地者 | 中等 | 无法偿还债务 → 债务监禁 → 出狱后失去一切 |

## 三、行会制度

### 3.1 行会类型

| 行会 | 总部 | 成员 | 权力 |
|------|------|------|------|
| 白垩城铁匠公会 | 白垩城 | 约500名铁匠师傅+约1,200学徒 | 全王国最大、最富有、最有政治影响力的行会。垄断精制铁器的制造。**不接受王室订单时——王国无铠甲可用。** |
| 丰收城粮商公会 | 丰收城 | 约200名粮商 | 控制中央平原的粮食贸易。定价权。可影响城市面包价格 |
| 灰石城皮革公会 | 灰石城 | 约150名皮革匠 | 垄断南方皮革加工 |
| 绿林镇羊毛商行 | 绿林镇 | 约100名羊毛商人 | 控制东部羊毛出口 |
| 白垩城药剂师公会 | 白垩城 | 约50名药草师+炼金师 | 垄断药物和基础药水。与教会关系紧张——教会视炼金为"灰色手艺" |
| 自由港抄书人行会 | 白垩城 | 约80名抄书人 | 垄断书籍复制。每年手抄约200本（全王国仅200本新书/年） |

### 3.2 行会运作

**入行程序**：
1. 10-14岁拜师 → 签约（父母代签）
2. 学徒期3-7年（在师傅家吃住——无工资——师傅负责教导和食宿）
3. 学徒期满 → 成为帮工（有工资）→ 工作2-5年积累经验
4. 提交"杰作"（masterpiece）→ 行会师傅们评审 → 通过则成为师傅
5. 师傅 → 可以开自己的作坊、收学徒

**行会的政治力量**：铁匠公会曾有一次拒绝为王室军队打造武器——因为王室拖欠了上一笔订单的尾款。那次欠款在两周内付清了。行会不投票——但它们有钱。而议会中的贵族们需要钱。

### 3.3 非行会的手工业者

乡村铁匠不属于城市行会——他们属于"乡村手艺人"。技术上——他们不能自称"师傅"因为没经行会考核。实际上——没有人去偏远村镇验证铁匠的资质。乡村不归行会管——这是一个法律漏洞。

## 四、性别与家庭

### 4.1 性别角色

| 领域 | 男性 | 女性 |
|------|------|------|
| 继承 | 长子继承制（贵族） | 女性可继承（如果没有兄弟）——但继承人需男性配偶"代理" |
| 财产 | 婚后财产属于丈夫 | 嫁妆是女方的——但由丈夫管理 |
| 经商 | 正常 | 寡妇可继续经营丈夫的生意。未婚女性需父亲或兄弟担保 |
| 法律人格 | 独立 | 已婚女性在大多数法律事务中需丈夫同意 |
| 出庭作证 | 可 | 可——但证词权重被法官默认为低于男性 |
| 加入行会 | 正常 | 少数行会接受女性（如纺织、面包）。铁匠行会不接受 |
| 军队 | 可 | 不可（教会武装中可能有极少量女性圣骑士——罕见） |

### 4.2 婚姻市场

**贵族婚姻**是政治筹码——从来不是个人选择。一个公爵女儿的婚姻可以为父亲争取另一个公爵的投票。王太子艾德温未婚——不是因为没有合适人选——是因为他的婚姻是整个棋盘上最大的筹码。一旦他结婚——他妻子的家族立刻获得王室内部影响力，其他贵族家族可能感到被排斥。

**平民婚姻**是经济策略。一个自耕农的儿子娶了佃农的女儿 → 带不来地。一个铁匠学徒娶了师傅的女儿 → 继承作坊。

**嫁妆**：新娘家庭付给新郎家庭。贵族嫁妆可能是土地/金钱/政治承诺。平民嫁妆可能是家具/家畜/布匹/金钱。

### 4.3 关键女性NPC的社会定位

**伊莎贝拉王后**：出身洛桑金冠家族——外族人。在艾尔德兰贵族眼中她永远是"那个洛桑女人"。她的政治智慧不受质疑——但她的合法性始终被旧权派暗暗质疑。如果她的儿子（王太子）在政治斗争中失败——第一个被攻击的就是她的外族身份。

**塞西莉亚公主**：19岁，在修道院。如果王太子无子嗣而意外去世——她是理论上最合法的继承人（阿尔德里克直系仅存的血脉）。但女性继承王位——在艾尔德兰历史上从未发生过。教会和贵族议会都不会轻易接受。

**磨坊镇某主妇**：作为面包师的女儿嫁给磨坊主——这种"同行间婚姻"是平民阶层的常见模式。在乡村中，实际的家庭关系往往比法律文件更重要——许多孩子由非血缘的长辈抚养长大。

## 五、荣誉与耻辱

### 5.1 各阶层的荣誉观

| 阶层 | 荣誉的核心 | 耻辱的来源 |
|------|----------|----------|
| 贵族 | 家族名声、军事勇气、契约诚信 | 被公开羞辱、懦弱、背叛诺言、家族丑闻 |
| 骑士 | 武勇、对领主忠诚、体面的生活 | 被认为是懦夫、无力偿还债务、被剥夺骑士身份 |
| 神职者 | 虔诚、贞洁（理论上）、慈善 | 被指控腐败、性丑闻、公开质疑教义 |
| 商人 | 契约诚信、财富、行会地位 | 破产、欺诈被发现、被行会除名 |
| 农民 | 自给自足、邻里帮助、不欠债 | 无法养家、乞讨、被发现在偷邻居的庄稼 |
| 女性（全体） | 贞洁、持家、顺服 | 婚前失贞、通奸（对女性比男性严重十倍）、被休弃 |

### 5.2 决斗——贵族阶层的荣誉暴力

除了法律上的决斗审判——还存在非正式的决斗。两人在私人场所（通常是清晨、空地）用剑决斗。由双方指定的"副手"见证。理论上非法——实际上治安官不会追究两个贵族。

决斗源于：被公开侮辱、被指控撒谎、涉及婚姻忠诚的纠纷。平民之间也有打架——但那叫斗殴，不叫决斗。平民打架可能被治安官鞭刑。

### 5.3 公开羞辱的惩罚作用

枷锁示众之所以是有效的惩罚——不是因为肉体的痛苦，而是因为公开的羞辱。一个在镇广场被锁了三天的商人——即使后来被释放——他的声誉可能永远无法恢复。他的顾客会去别处。这是为什么罚金的威慑力对商人足够——但枷锁的威慑力对所有人足够。

## 六、社交礼仪与日常互动

### 6.1 阶层间互动规则

| 互动 | 规则 |
|------|------|
| 平民对贵族说话 | 先鞠躬或行礼——说"大人"或"殿下"——等贵族点头后才可开口 |
| 贵族回应平民 | 可直呼其名——不需敬语。但若平民获得贵族"好感"——贵族可能用"善良的人"（goodman/goodwoman）称呼对方 |
| 两人同为贵族 | 低爵位先向高爵位致意。同级——年长的一方先被致意 |
| 平民在公共场合经过贵族 | 让路——脱帽——低头。若不遵守——贵族可以要求领主的治安官罚款 |
| 神职者与平民 | 接受"神父""修士"的称呼。平视平民——不需要行礼 |
| 神职者与贵族 | 相互尊重。主教的社交地位等同于公爵 |

### 6.2 日常社交仪式

**问候**：面熟的人——点个头、说一声"好日子"。不熟的人——不理。在乡村——每个人认识每个人，陌生人立刻被注意。在城镇——陌生人是常态。

**共用餐饮**：邀请别人吃你的食物——是友谊的表示。拒绝邀请——是被冒犯。在酒馆——有人帮你买了杯酒，你必须至少和他聊一句。

**赠送礼物**：贵族之间——礼物 = 政治信号。一把精致的匕首 = "我把我的生命放在你手中"（象征性）。普通人家——送食物 = 最真诚的善意。

**触碰**：平民不可以触碰贵族（除非贵族先伸出手）。神职者——在祝福时触碰信众的额头——这是"被允许的触碰"。同性朋友之间——握手、拍肩——正常。异性之间——不能公开触碰（夫妇除外）。

### 6.3 外乡人与陌生人

**在乡村（如磨坊镇）**：陌生人 = 被注意。一个外乡人走进磨坊镇 → 酒馆老板之类的人会先搭话——以闲聊的方式套出来历。如果外乡人有合理的理由（行商/传令）→ 被接受。如果没有合理理由 → 村民会保持距离 → 可能报告领主治安官。

**在城镇（如灰石城）**：陌生人是常态。但"没有担保人的陌生人"在租房/签订契约/借贷方面受限。你需要一个本地人担保。

**担保制度**：一个外乡人要定居在灰石城——需要一位"担保人"（公民或贵族）在市政厅为他登记。没有担保人 → 你是"不受担保的人" → 契约无效力、不能租房超过一个月。

## 七、社会冲突的断层线

> 这些是社会结构中固有的张力。GM可以在剧情中利用。

### 7.1 贵族 vs 富商

富商有钱但没有地位。一个年收入300金的商人——生活比骑士更好——但在公共场合必须向一个穷骑士鞠躬。这是他每天咽下去的羞辱。有些富商暗中期盼税改（削弱贵族特权）。有些富商通过嫁女儿给穷贵族来"买"社会地位。

### 7.2 骑士 vs 贵族

骑士出身基层、经济紧张、渴望上升。他们是贵族的"边缘人"——挂着贵族头衔但随时可能滑回平民。他们对旧权派的奢华生活方式暗中不满——对王太子的改革暗中期盼——但不敢公开表态。

### 7.3 农民 vs 领主

这是最古老、最沉默的冲突。农民不说出来——但一年收成的50%被领主拿走。收成不好 → 领主不减租 → 农民借粮 → 还不清 → 永远困在债务中。如果税改减轻了农民负担——领主可能提高地租以弥补——税改能否真正改善农民生活是个未知数。

### 7.4 乡村 vs 城市

乡村人口多但分散、无组织、无政治权力。城市少数——但有行会、有金钱、有组织的政治声音。税改不会改变乡村的政治弱势——但可能为城市行会扩大政治影响力（商人支持税改→王太子在税改中提出"商人代表加入税务委员会"以换取商人阶层的支持）。

### 7.5 北方 vs 南方

北方边境直面帝国——民众最关心的是军事安全。南方（包括磨坊镇）地处偏远——对"北方在打什么仗"不关心。这种漠不关心反过来让北方边境居民感到被遗忘——"我们在为保卫他们而死而他们不知道我们在打仗"。

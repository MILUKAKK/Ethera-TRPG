## 二、产业与生产

### 2.1 农业

**中央河谷平原**：银带河千万年冲积而成的黑土——大陆最肥沃的农业区。年产小麦约200万吨（占王国农业产出的约60%）。亩产约120-150kg——在不用化肥的时代属于优质。其余为大麦/燕麦（饲料+麦酒）、葡萄、蔬菜。

**东部丘陵**：牧业为主。年产羊毛约5,000吨。东部牧区也是王国战马的唯一繁育地——年产约300匹战马。战马繁殖周期不可压缩——这是"你有多少钱也解决不了"的约束。

**南部丘陵**：皮革加工中心。灰石城处理全国约70%的皮革。南部农业规模小——土壤不如中央平原肥沃。

**北部边境**：农业规模最小。边境居民部分依赖军事服务业（为要塞驻军提供食物、洗衣、铁匠等）。

### 2.2 手工业

| 产品 | 年产量 | 产地 | 品质 |
|------|--------|------|------|
| 铠甲（全套） | 约3,000套 | 白垩城铁匠公会 | 大陆公认标杆 |
| 长剑 | 约1,500把 | 白垩城 | 优质 |
| 皮甲 | 约2,000套 | 灰石城 | 标准 |
| 长矛 | 约3,000支 | 各地铁匠 | 标准 |
| 箭矢 | 约50,000支 | 各地铁匠 | —（消耗品） |
| 羊毛布 | 约3,000吨（成品） | 东部→白垩城加工 | 优质 |
| 葡萄酒 | 约80万瓶 | 白垩城周边 | 白垩城白葡萄酒——享誉诸国 |

**产能瓶颈**：所有手工业靠人力+简单工具。无法快速增产。战时转入军事生产——箭矢产量可增4倍（20万支/年），但整套铠甲无法加速（一具铠甲从铁锭到成品约需一个铁匠3-4个月）。

### 2.3 铁料——阿喀琉斯之踵

| 事实 | 含义 |
|------|------|
| 艾尔德兰中央平原**无铁矿** | 所有铁料依赖进口 |
| 铁料80%来自铁冠王国（通过灰塔公国中转） | 铁冠王国被铁峰山脉保护——安全但遥远 |
| 铁料15%来自维斯特法尔（和平期） | 战争期此来源立即中断 |
| 铁料5%来自裂脊矮人国（稀有高品质矿） | 仅用于高端武器——成本极高 |
| 从铁冠矿坑到白垩城铁匠铺的全程运费约占铁料成本的40% | 远离矿源 → 武器制造成本约为帝国的1.5倍 |

**连锁影响**：
- 武器成本高 → 同等军费装备的士兵少 → 见 `军事.md` §七
- 铁料来源单一（铁冠王国） → 若铁冠被卷入战争或灰塔公国中断中转 → 王国武器生产立即瘫痪
- 帝国矿场在铁峰山脉北麓——矿就在家门口 → 帝国的武器成本碾压艾尔德兰

## 三、区域经济差异

### 3.1 四大经济区

| 区域 | 人均收入 | 经济特征 | 在全国GDP中的占比 | 代表城镇 |
|------|---------|---------|-----------------|---------|
| 中央平原 | 约6金/年 | 农业+手工业双强。小麦、葡萄酒、铠甲。 | 约55% | 白垩城、丰收城 |
| 东部丘陵 | 约3金/年 | 牧业为主。羊毛、战马。 | 约15% | 绿林镇、石桥城 |
| 南部丘陵 | 约2.5金/年 | 皮革加工+小型农业。 | 约18% | 灰石城 |
| 北部边境 | 约2金/年 | 军事服务+边境贸易（和平期）。 | 约12% | 河卫城 |

### 3.2 城乡差距

- 城镇人口约68万（18%），乡村约312万（82%）
- 城市人均收入约为乡村的3-6倍
- 但城市贫民（约15万）的生活水平并不高于自耕农——只是饿死得慢一些

### 3.3 区域视角下的税改

不同区域的领主对土地税的态度与其经济结构高度相关（详见 `贵汉族与皇室.md`）：
- 中央平原大农业领主 → 土地多 → 坚决反对
- 东部牧业领主 → 土地是牧场而非农田 → 地力差 → 如果按地力分级征税则影响较小 → **摇摆**
- 南境小领主 → 土地少但占比大 → 坚决反对
- 北境边境领主 → 土地少且军事压力大 → 支持（军费增加的好处超过土地税的负担）

## 六、经济-财政关系

> 实体经济（本文件）与政府财政（`财政.md`）的关系：

```
         实体经济（GDP 2,800万金）
              │
    ┌─────────┼─────────┐
    │         │         │
   农业60%  手工业25%  贸易+服务15%
    │         │         │
    └─────────┼─────────┘
              │
        税基（→ 财政收入850万金）
              │
              ▼
        财政支出（→ 军费/行政/王室/债务）
              │
              ▼
    实体经济反馈 ← 军费消费 → 武器需求拉动 → 铁匠收入
                ← 行政支出 → 官员消费 → 城市服务业
                ← 债务利息 → 教会和银行的再投资
```

**关键约束**：850万金财政收入 ≈ 货币化GDP（1,700万金）的50%。这意味着政府收走了正规经济中的一半。进一步提高税率的空间极为有限——税改的目的不是加税，而是重新分配税负。

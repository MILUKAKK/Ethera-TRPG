## 一、行政体制总览

### 1.1 性质

艾尔德兰没有"文官体系"——没有任何一个词能对应现代概念中的"公务员"。行政被嵌入封建体系中——每个人同时扮演多重角色：

- 一位公爵 = 封地统治者 + 贵族议会议员 + 封地最高法官 + 行政长官 + 军事征召组织者
- 一位骑士 = 低级军事指挥官 + 可能兼任封地内某个行政职务
- 一位领主治安官 = 警察 + 法官 + 收税员 + 公共工程组织者 + 人口登记员

**行政 = 贵族 + 治安官 + 王室任命的管理者 + 教会的记录职能。四者拼在一起——不存在统一的行政体系。**

### 1.2 行政层级

```
                        国王
                         │
              ┌──────────┼──────────┐
              │          │          │
         王室管家     财政大臣    御前秘书
       （王室领地管理）（国库+征税）（王室文书+法令起草）
              │          │          │
    ┌─────────┼──────────┼──────────┼─────────┐
    │         │          │          │         │
 公爵领(×12) 伯爵领(×~30) 男爵领(×~80) 教会领地 王室直辖区
    │         │          │          │         │
 治安官     治安官     治安官    教会管区   王室官员
    │         │          │          │         │
 村镇        村镇       村镇       教区       王室城镇
(如磨坊镇)  (大小不等)  (小型)
```

**磨坊镇在这个结构中的位置**：磨坊镇 → 灰石公爵领 → 南部丘陵 → 艾尔德兰王国。从白垩城发出一道命令到磨坊镇：国王 → 灰石公爵 → 灰石城的治安官 → 磨坊镇的联络人（通常由本镇较有声望的居民兼任——非正式）。

## 二、中央行政

### 2.1 国王的行政工具

| 职位 | 任命 | 职能 | 当前任职者 |
|------|------|------|----------|
| **财政大臣** | 国王任命——不必须是贵族 | 征收全部王室税收、管理国库、编列预算、偿还债务 | 奥尔多·金账房（男，57岁）——平民出身 |
| **王室管家** | 国王任命——通常是资深贵族 | 管理王室直领地（约15%全国可耕地）、王宫日常运营、王室雇员的工资和食品采购 | 劳伦斯·白塔爵士（男，63岁） |
| **御前秘书** | 国王任命——不必须是贵族 | 起草王室法令、管理王室档案、记录贵族议会议程、收发国王的信件和传令 | 马库斯·羽毛笔（男，41岁）——抄书人出身 |
| **传令官长** | 国王任命 | 管理传令官网络——负责向各领主发送王室命令 | 加雷斯·快蹄（男，35岁）——前银翼骑士团骑兵 |

### 2.2 御前秘书的角色

御前秘书是国王/摄政与白垩城外一切事务之间的行政枢纽。他的办公室（仅三个房间+六个抄书人）处理：
- 国王发往各地领主的信件复写（每封信需两份——一份发给领主，一份留档）
- 贵族议会议程的书面记录（仅记录投票结果但不记录辩论内容——这是故意的，让贵族可以自由发言而不留把柄）
- 外国使节的来函归档
- 王室法令的发布——抄写多份分发至各教区教堂的公告板

**马库斯·羽毛笔**：白垩城抄书人行会的前成员。四年前被王太子选中——因为他的笔迹是白垩城最清晰端正的。他每天抄写约6-8封信——这个速度是他最大的"权力"：**国王的信取决于他的手什么时候写完。** 他知道很多国家机密——但他从不说话。他在抄书人行会的老同事说"他以前话可多了"——然后就不说话了。

### 2.3 财政大臣署

财政大臣署的白垩城总部约50人——这是王国最接近"官僚机构"的东西。分为四个科室：

| 科室 | 人数 | 职能 |
|------|------|------|
| 税入科 | ~15人 | 记录各领地的税收上缴、核对金额、追缴欠税 |
| 支出科 | ~12人 | 按预算案向各机构拨款、记录支出、防止超支 |
| 债务科 | ~5人 | 管理教会和金冠银行的两笔贷款——利息计算、抵押品价值跟踪、到期日标记 |
| 档案科 | ~18人 | 保存过去所有财年的财务记录。理论上——实际上大量记录因火灾/虫蛀/遗失而永久丢失 |

**金账房的日常**：每天早上他打开税入科的简报——昨天从哪些领地收到了多少税。如果某个领主连续两个月未缴税——他会给该领主发一封措辞礼貌但不容忽视的信。如果一个领主连续六个月未缴税——**这就是金账房向国王建议"考虑替换该领主"的触发点**——税务拖欠是封建制中少数几个可以从法律上质疑领主职权的触发条件之一。

## 三、领主行政

### 3.1 公爵领的行政运作

以灰石公爵领地为样本——因为磨坊镇归属它。

**灰石公爵的行政人员**（全部由公爵个人任命和支付薪酬）：

| 职位 | 人数 | 月薪 | 职能 |
|------|------|------|------|
| 领地总管 | 1人 | 约50银/月 | 公爵缺席时的最高代行者。协调所有下级行政人员 |
| 治安官 | 4人（含1名首席） | 约20银/月 | 每人负责约3万人口。执法+审判+收税+修路+登记 |
| 收税员 | 约15人 | 约8银/月 | 果月/秋月下乡挨家挨户收税 |
| 文书 | 2人 | 约15银/月 | 记录税收、管理公爵的信件、起草领主条例 |
| 灰石城堡总管 | 1人 | 约30银/月 | 城堡日常维护、食物采购、守卫管理 |

**一个治安官的管辖范围**：约3万人口散居在数百平方公里的丘陵和村镇中。他这个月去东边的村子收税——下个月去西边的村子处理一桩偷牛案。他在路上的时间比在办公室多。

**磨坊镇的治安官**：灰石公爵任命的四名治安官之一——名字是**埃德蒙·坚土**（男，44岁）。他每季来磨坊镇一次巡查。日常——磨坊镇的秩序靠自我维持。

### 3.2 从国王令到村镇执行

以征召令为例，看看一道"简单的"行政命令实际上涉及多少个环节：

```
① 国王/摄政在御前会议上决定征召
   → 御前秘书起草征召令 → 国王签名盖章
   
② 传令官骑马出发
   → 白垩城 → 丰收城（半天，换马）
   → 丰收城 → 灰石城（5天）
   → 全程：白垩城至灰石城约350km，传令官速度约70km/天 = 共约5天
   
③ 灰石公爵收到征召令
   → 公爵召见四名治安官
   → 分配各治安官负责的征召名额
   → 每位治安官带着手写名单下乡
   
④ 治安官到磨坊镇（灰石城至磨坊镇约80km，马车2天1夜）
   → 在教会小堂门口贴上告示
   → 检查人口登记名单（可能已过时一两年）
   → 确定磨坊镇应征召的人数：依人口比例——磨坊镇约200人 → 约2-3人
   
⑤ 被征召者报到
   → 步行或牛车前往灰石城（2-3天）
   → 在灰石城集合 → 编入南境征召队
   → 开始训练（10-20天后才能投入战斗）
   
全程从国王签令到磨坊镇的征召者站在灰石城训练场：
至少 10天（最快）至 20天（正常）。
```

**行政的摩擦成本**：每一个环节都是潜在的延迟点。传令官的马可能在路上跛了。公爵可能在打猎而不在城堡。治安官可能正在另一个村子处理偷牛案。告示可能被雨淋湿字迹模糊。被征召者可能躲进山里。

## 四、公共工程

### 4.1 谁修什么

| 工程类型 | 负责方 | 资金来源 | 例子 |
|---------|--------|---------|------|
| 王室道路（连接主要城市） | 王室——财政大臣署预算 | 中央财政——关税收入 | 白垩城↔丰收城 石板路 |
| 领主道路（封地内主要道路） | 领主——领主自有资金 | 领主征收的地租 | 灰石城↔磨坊镇的土路 |
| 村镇小路 | 村镇居民——义务劳动 | 无——各家出人力 | 磨坊镇内部的小路 |
| 桥梁（重要） | 视位置——王室或领主 | 中央或领主 | 石桥城的石桥属于王室（它是东西交通要道） |
| 防御工事 | 王室——军费 | 军费预算 | 血河南岸12座要塞 |
| 水渠/水利（王室项目） | 王室——国王特别拨款 | 国王私人金库或议会特批 | 路德维希三世年轻时修的水利系统（至今仍在使用） |
| 水渠/水车（地方） | 私人——但需领主许可 | 私人 | 磨坊镇的三座水车——磨坊是私人财产 |

### 4.2 南方官道——一个公共工程的个案

南方官道从灰石城延伸到磨坊镇及更南——是南部唯一的官方道路。理论上由王室出资维护——**实际上已多年未维护**。

| 事实 | 后果 |
|------|------|
| 上一次官道全面维护是在路德维希三世的"改革期"——约三十年前 | 路面坑洼。雨天泥泞。多处边坡已不稳定 |
| 财政大臣署的预算中有"道路维护"一项——但近十年来这项预算逐年减少 | 每年仅够填补最深的坑 |
| 山体滑坡（随机事件M21）如果发生——灰石公爵需要先向白垩城报告 → 等待财政大臣署拨款 → 拨款可能需要数月 | 在此期间——磨坊镇的粮食运不出去 |
| 灰石公爵不能自己出钱修官道——因为官道是王室的财产（花了公爵的钱修王室的财产是封建惯例中模糊的区域） | 公爵选择不动——避免被指控"擅自扩大封地权力" |

这就是为什么磨坊镇到灰石城的官道状态**很糟糕但没人修**——所有人都同意它需要修，但没有人有权决定付钱。

### 4.3 银带河水利系统——路德维希三世最后的遗产

三十年前，路德维希三世正值盛年——他推动了王国最大的水利工程：银带河水利系统。包括：
- 一条从银带河引水的主渠道（长约80km，灌溉中央平原东部）
- 数十条支渠
- 三座水闸用于调节水量

这项工程使用了约800名劳工，历时三年，耗费约50万金。工程完成后中央平原东部的小麦产量增长约25%——直接推动了艾尔德兰接下来二十年的农业繁荣。

**当前状态**：渠道仍在运作——但水闸已有十年未检修。东部绿林镇以下的渠道段已经淤塞。如果在未来十年内不进行全面检修——整个系统可能失效。

**维护问题**：最初建设的50万金中有约5万金是专门的"未来维护基金"——但这笔钱在十五年前的一次财政危机中被"暂时挪用"。从未归还。

## 五、税收运输——行政的物理现实

### 5.1 税收从各领地上缴到白垩城

税收不只是账本上的数字——**是物理的钱币和粮食**。一袋铜币的重量和体积意味着运输本身是一个行政挑战。

以灰石公爵领为例——年度税收约30万金等值的实物和钱币：

| 步骤 | 操作 |
|------|------|
| ① 各治安官在果月/秋月下乡收税 | 农民付的通常是实物（麦穗）+ 少量铜币 |
| ② 实物税集中在灰石城的公爵谷仓 | 麦穗 → 入库 → 等待转运 |
| ③ 实物评估折现 | 公爵的收税员评定麦穗的质量和重量——折算为对应金额 |
| ④ 实物或现金运往白垩城 | 每年一次（通常在落叶月——秋收完成后）。车队：约10-15辆马车+公爵的护卫骑兵 |
| ⑤ 抵达白垩城财政大臣署 | 钱币入库；实物被拍卖或进入王室粮仓 |

**运输损耗**：从灰石城到白垩城约350km的马车运输——谷物在运输中有约5-10%的损耗（受潮/虫咬/颠簸撒落）。王室和领主之间有一项默契——"合理运输损耗"不追究。

**同一过程在磨坊镇的微缩版**：磨坊主每年去灰石城交税一次——赶着马车带着几袋面粉和一小袋铜币。面粉可能在路上受潮——但磨坊主的面粉质量稳定——灰石城的收税员每次不多问。

### 5.2 钱币的物理流通

艾尔德兰没有国家银行——没有"转账"概念。所有税收以物理钱币或实物形式流通。这意味着：

- 铜币从农民手中缴到治安官 → 治安官缴到公爵 → 公爵缴到财政大臣署 → 财政大臣署支付官员和骑士薪水 → 官员和骑士去市场消费 → 铜币回到商人手中 → 商人缴关税 → 关税进入王室金库 → 闭环
- 金币（王冠币）流通范围远大于铜币：王冠币在洛桑兑换金票、在铁冠购买矿石、在灰塔支付过境费

**铸币厂的物理位置**：白垩城铸币厂——王室禁地。每年铸新币约50-80万金等值。银铜原料从进口的矿石中提取；金料几乎完全来自贸易顺差的金币回流。

## 六、记录与档案——行政的记忆

### 6.1 人口记录

领主治安官手写的人口登记名单是行政的**最底层基础设施**。从这份名单上决定：
- 征召多少人
- 收多少个铜币的人头税
- 一个村镇是否存在（不在名单上的村镇——不存在于行政视野中）

**这份名单有多不准确**：
- 上一次全面人口统计是在约8年前
- 此后死亡/出生/迁徙没有被系统更新
- 治安官下乡时偶尔更新——但不是每次
- 磨坊镇的名单可能还是7年前的——上面可能记录着一些已经去世的人和一些已经离开的人，而一些新出生的人可能不在上面

**这意味着什么**：磨坊镇的"官方"人口可能是200人——但实际可能是180人或220人。治安官征召时按200人计算比例——可能征召了实际上不存在的人（如果实际人口偏低），或者应该征召更多的人但不知道他们在哪儿（如果实际人口更高）。

### 6.2 土地记录

贵族土地记录是"地籍"——谁拥有哪块地、多大面积、边界在哪。由领主文书手写保存。**没有中央土地登记**——每块地的所有权完全依赖于领主手中那份**唯一的**手写文件。

**结果**：土地边界纠纷极常见。两个邻居发现他们对同一块地的两份手写土地记录有矛盾的边界——唯一的仲裁者是领主（他可能不知道或不在乎）。

### 6.3 王室档案

白垩城王室档案室——三个石砌房间，存有建国以来600年的行政文件：法令副本、议会投票记录、条约文本、国王信件。但**没有索引**。找一份50年前的文件意味着一个文书可能需要翻找数天至数周。

御前秘书马库斯·羽毛笔正在尝试编制一个初步的索引——这项工作已经进行了一年，完成了约10%。在此期间，档案室的灰继续积累。

## 七、行政依赖教会的部分

教会的记录职能是王国行政中**不可替代**的部分。没有教会的记录——王国不知道谁出生了、谁死了、谁和谁结婚了。

| 记录类型 | 由谁维护 | 地点 | 什么用途 |
|---------|---------|------|---------|
| 出生登记 | 教区司铎 | 城镇教堂 | 征召时确定年龄。人头税时确定"15岁以上"。法律上确认身份 |
| 婚姻登记 | 教区司铎 | 同上 | 财产分配、继承、合法子女认定 |
| 死亡登记 | 教区司铎 | 同上 | 人口统计。继承触发。债务清算 |
| 遗嘱 | 教会法庭 | 主教座堂 | 遗产分配 |

**如果教会停摆**：王国行政的所有"人"的层面将瘫痪。这就是为什么国王不能公开对抗教会——不是因为信仰，是因为教会在行政上是**不可替代的基础设施**。

在磨坊镇——年迈的助祭可能已经好几年没更新过记录本了。如果某人在灰石城出了事——谁给磨坊镇写死亡登记？如果有人到了出嫁年龄——谁来验证他的出生登记上的年龄？

## 八、行政命令的类型

### 8.1 国王可以不经议会发布的命令

| 命令类型 | 适用范围 | 可被议会推翻吗 |
|---------|---------|--------------|
| 王室法令（Royal Decree） | 王室直辖区+有关王室财产和特权的所有事务 | 否——但历史上从未有国王在对议会极端不利的情况下坚持使用此权 |
| 任命书 | 王室官员（财政大臣、御前秘书、使节等） | 否 |
| 元帅任命 | 军队最高指挥官 | 否——但议会可以通过拒绝军费预算来使元帅的命令无效 |
| 传令官通告 | 全国——但需领主在其封地内公布 | 否 |
| 国王紧急敕令 | 紧急情况下的临时支出或军队调动 | 否——但次数有限制（滥用会被议会事后追责） |

### 8.2 需要议会批准的命令

| 命令类型 | 原因 |
|---------|------|
| 任何涉及新税收或税改的命令 | 征税权属于议会+国王——而非国王单方 |
| 超过财年预算的额外支出 | 预算已由议会投票通过——超出部分需要重新投票 |
| 向外国宣战 | 国王可签署宣战令——但军费需要议会批准 |
| 修改《艾尔德兰法典》 | 王国基本法——需议会2/3多数 |
| 增设或取消公爵席位 | 修改建国的权力平衡——需9票（3/4多数） |

### 8.3 领主在其封地内可以自行决定的命令

| 命令类型 | 限制 |
|---------|------|
| 颁布封地条例（"领主条例"） | 不得与《艾尔德兰法典》冲突。但通常无人审查 |
| 任命封地内的行政人员 | 公爵自行决定 |
| 征收地租 | 不得超过"惯例"（未经界定的惯例——纠纷由此产生） |
| 修建封地内的公共工程 | 需公爵自己出钱——无王室财政支持 |
| 封地内的临时税收（领主自行征收——用于封地内的工程或紧急需求） | 必须"合理"——虽然这个"合理"从未被法典定义 |

## 九、行政效率的总评

| 维度 | 评分 | 说明 |
|------|------|------|
| 命令传达速度 | 3/10 | 传令官全靠马。无远距通讯 |
| 税收征收效率 | 5/10 | 15-20%的征收损耗——在封建时代属于正常 |
| 公共工程维护 | 2/10 | 三十年前修的路至今未维护。水利基金被挪用 |
| 记录的准确性 | 3/10 | 人口名单可能过期多年。土地记录唯一一份在领主手中 |
| 行政人员素质 | 4/10 | 平民出身的金账房和羽毛笔是亮点。但基层治安官训练为零 |
| 行政-教会的协同 | 6/10 | 教会的记录职能不可替代——这是最大的系统性风险 |
| **加权总评** | **3.5/10** | 行政在勉强运作——离"运行良好"还很远 |

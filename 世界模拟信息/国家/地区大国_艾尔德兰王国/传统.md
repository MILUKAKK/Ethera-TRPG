## 一、全国性节日

### 1.1 岁首节（冰月1日）

**意义**：新年第一天。一年中唯一一个"地位平等"的日子——贵族和平民在这一天互赠面包。

**怎么过的**：
- 凌晨——每家的主妇第一个起床，点燃灶火。这被称为"点亮新年的第一道光"——模仿索拉创造世界的第一缕光
- 上午——全家一起吃"岁首面包"。这个面包在前一晚（雪月28日晚上）被烤好，必须分给每一个家庭成员——包括雇工和客人。掰面包时不能说任何不吉利的话
- 中午——邻里互访。每人手里端着一小碟蜜糖——蘸面包吃。蜜糖象征"新的一年要甜"
- 日落前——每家把预先做好的麦穗环挂在门口。这是"保护环"——传说能阻止恶灵进入新年

**贵族怎么过**：公爵举办"岁首宴"——邀请封地内所有骑士和他们的家人。公爵亲自为最年长的骑士倒第一杯酒。这是"领主-封臣"关系的仪式性重签——不是法律义务，是传统。

**磨坊镇的岁首节**：每家的岁首面包是全年最大的。主妇们提前一天开始准备。全镇人在门框上挂麦穗环。酒馆免费给每个来的人倒一杯蜜酒。年迈的助祭在教堂小堂做一个简短的岁首祈祷——这是镇上所有人一年中去教堂最整齐的一次。

### 1.2 萌芽祭（萌芽月1日）

**意义**：春耕前的祈福。这是**大地老人的节日**——被圣光教会容忍但从未正式承认。

**怎么过的**：
- 全村镇的人聚集在最大的那块田旁边
- 每个人都带一小袋种子——不是全部的种子，是少数几粒
- 司铎洒圣水在田里——这是教会的部分
- 然后村镇最年长的农民（在磨坊镇就是那位不知名的老人）把第一把种子撒向东方——这是大地老人的部分
- 所有人跟着把自己的几粒种子撒出去。不能多——只是象征
- 剩下的时间——全村聚餐。每家带一个菜

**禁忌**：萌芽祭之前不能翻土——"会在梦中惊动正在睡觉的种子"。这个禁忌在东部丘陵特别严格——东部牧区的人相信违者会导致整个春天的草长得不好。

### 1.3 银龙祭（夏月7日）

**意义**：纪念建国时阿尔德里克一世与银龙"艾尔德兰"的盟约。这是**艾尔德兰特有的**——其他国家不过。

**怎么过的**：
- 全国举行小型"龙舞"——年轻人穿上银色斗篷在街上舞动。这在白垩城最为壮观——银龙游行的路线从王宫开始到圣光大教堂结束
- 这一天——不能使用任何银器。银餐具被收起——因为"银是龙的颜色，今天属于龙"
- 家家户户在窗口放一件银色的东西——一片银叶、一枚银币、甚至只是一块锡箔——象征"我们还记得那个约定"

**磨坊镇的银龙祭**：镇上能凑出的银色物品屈指可数。有人把几枚盾徽币摆在窗台上。铁匠在这一天不工作——"打铁的声音会吵到银龙"。孩子们每年都想看银龙游行——但从来没去过白垩城。

**政治色彩**：最近几年——在王太子与旧权派的对抗中——银龙祭被赋予了额外的政治含义。王权派贵族将银龙祭用作"阿尔德里克家族的古老合法性"的象征展示。旧权派则有意低调——黑木公爵去年甚至没有在他的领地举办银龙游行。

### 1.4 夏炎祭（烈月15日）

**意义**：盛夏狂欢。与宗教无关——纯粹的世俗节日。

**怎么过的**：
- 白天：体育比赛——射箭、摔跤、赛跑。奖品通常是钱袋或一桶麦酒
- 下午：自酿果酒开喝。酒馆通宵营业——全年唯一一个不管制夜间秩序的日子
- 晚上：篝火。年轻人跳过篝火——跳过去据说带来好运
- 深夜：从篝火中取一块烧黑的木炭放在自家灶台中——"夏天的火延续到冬天"

**禁忌**——或者说是迷信：夏炎祭之前不能收割任何谷物。"谷神还在睡——夏天还没到它的顶点"。实际上这是为了让谷物有更长的生长期。违反者在村里会说"你想让冬天没面包"。

**磨坊镇的夏炎祭**：磨坊镇太小——不比赛。但篝火是必须的——镇中心水井广场上堆一堆柴火，全镇人围坐。有人罕见地喝了两杯。有人做了蜂蜜蛋糕。这是磨坊镇全年最放松的一天。

### 1.5 丰收祭（果月28日）

**意义**：一年中最重要的节日。秋收完成后——**感恩**。整个大陆的农业社会都过——各地名称和形式略有差异。

**怎么过的**：
- 上午：教会感恩弥撒。把今年收获的第一束麦穗（在萌芽祭时留下的种子长大后）供奉在教堂圣坛上
- 中午：全村/全镇聚餐。每家带一样食物——放在长桌上共享。不能拒绝任何人的食物——即使你讨厌那个人
- 下午：农产品评选——最大的南瓜、最长的麦穗、最甜的葡萄。获奖者得到镇长的口头赞扬和一面小木牌挂在门口（"年度最佳麦穗"）
- 晚上：跳舞——乐手（谁家有琴的就弹琴；没有——则人声哼唱+敲桌子）。

**丰收城的大集市**：丰收城在丰收祭期间举办全国最大的集市——全国各地商贩都来。这是普通农民一生中可能见到的最大的商业活动。

**磨坊镇的丰收祭**：磨坊镇太小——不评选。但聚餐是全年的社交顶点。某家的炖肉是每次最早被抢光的菜。有人负责提供麦酒——从酒馆买了一桶。

### 1.6 长夜节（雪月28日）

**意义**：一年中最长的夜晚。家庭团聚。**反思**——不是宗教忏悔，而是个人的安静盘点。

**怎么过的**：
- 全家围坐炉边——不出门
- 每个人对自己说一句话——可能是今年最遗憾的事，或明年要完成的事。可以不说出来——心里想
- 长者（祖辈或父母）给孩子们讲一个过去的故事——他们的年轻时。孩子听着——即使听过一百遍也不能打断
- 午夜——点燃"长夜蜡烛"（一整根可以燃烧几个时辰的蜡烛）。让它一直燃烧到天亮——"光陪我们度过最长的夜"

**磨坊镇的长夜节**：主妇们通常在长夜节变得异常安静——她们不讲自己的故事。丈夫们也不讲。孩子们注意到这一点——但还太小，不知道为什么。去年——有孩子在炉边睡着了，母亲把她抱上床时红了眼眶。

### 1.7 封印日（冰月0日——岁首前夜）

**意义**：纪念深渊封印完成的日子。教会掌控的最重要的节日之一。

**怎么过的**：
- 彻夜教堂祈祷——从日落开始到黎明
- 每个教堂点燃"封印之烛"——七根蜡烛代表七位术士。每一小时熄灭一根——当最后一根熄灭时正是黎明
- 年长者在这一夜不睡——"陪封印守一夜"

**磨坊镇的岁首前夜**：年迈的助祭在教堂小堂作岁末祈祷。酒馆老板偶尔会去坐坐——他说"陪陪老人家"。有人不去——她说"大冷天的，坐不住"——但其实她更愿意在家看着灶火守岁。

## 二、地方性习俗

### 2.1 磨坊镇的地方守护——磨坊老人

磨坊镇有一个不属于任何官方宗教的微型传统：**磨坊老人**。据说每个水车磨坊里住着一位看不见的"磨坊老人"——他转动水车、让面磨得细、让轴承不坏。磨坊主从不在磨坊里骂人——"磨坊老人听着呢。"

每年开磨季（萌芽月——水车开始转动），磨坊主会在水车轮轴上抹一点蜂蜜——"给磨坊老人的开季礼"。这花不了几文钱，但这是许多磨坊主唯一的迷信。

**对孩子们的影响**：孩子们相信磨坊老人是真实存在的——有的说有一次看到水车自己多转了一圈（大人检查后告诉他们那是轴承松了）。

### 2.2 灰石城的皮革节（秋月15日）

灰石城特有——全国其他地方不过。起源于灰石城作为南方皮革集散中心的历史。

这一天制革匠们闭铺。全城弥漫着新皮革的味道——因为每家制革铺在这一天把最好的皮子拿出来展览。评委会评选"年度最佳皮革"——获奖者在接下来一年中皮革的公会定价权更大。

### 2.3 丰收城的麦穗婚礼

丰收城及中央平原农村的传统：新婚夫妇在婚礼上用麦穗代替鲜花。新郎送给新娘一束麦穗——"这比花实用"——象征他将为她提供面包。新娘把麦穗插在厨房灶台边——一整年。

如果在下一个丰收祭之前麦穗发霉——被认为是不祥的征兆。丰收城的粮商公会以打折价为新婚夫妇提供"不易发霉的麦穗"——他们称之为"婚姻保险"。

### 2.4 河卫城的寂静钟

河卫城（北部军事要塞）的传统钟楼除报时外——只在一种情况下敲响：守望塔发现帝国部队在血河北岸集结时。这被称为"寂静钟"——因为它大部分时间不响。上一次敲响是在27年前——第三次血河冲突。

河卫城的居民对"钟今天会不会响"有某种病态的关注。每个新兵入伍第一天——老兵会告诉他："你不需要知道钟的声音。它响的时候你自然知道。"

## 三、民间故事与歌谣

### 3.1 《银龙与猎人》

大陆最广为人知的民间传说。据说起源于艾尔德兰建国之前——但建国后这个故事被赋予了阿尔德里克色彩。

故事核心：一个普通猎人在龙骨山脉救助了一只受伤的银龙——他从龙爪上拔出了一根铁矛（矮人猎龙者的残骸）。银龙为了报答，给了他三片龙鳞。第一片龙鳞变成了一柄永不折断的剑。第二片龙鳞变成了一袋取之不尽的金币。第三片——猎人从未使用，因为他已经拥有了所需的一切。他把第三片龙鳞还给了银龙。

传说中这个猎人的后代就是阿尔德里克家族。实际上——这很可能是一个嫁接在更古老故事上的建国神话。

**各地变体**：在维斯特法尔版本中，猎人是帝国祖先——他偷了龙鳞而不是被赠与。在兽人版本中，猎人是兽人——他拒绝了银龙的报答，因为他"不需要龙的东西"。

### 3.2 《血河上的寡妇》

一首磨坊镇及南境流传的民歌。讲述一个女人在血河战争（1652年）后站在河南岸等待丈夫归来——他从来没有回来。她每天站在河边——头发由黑变白。最后一天她走进血河——据说河水在那一刻变成了普通的颜色。

这首歌在酒馆里经常被点唱——但只在深夜、人们喝了足够多之后。有些老板不喜欢这首歌——他们说"那是一百多年前的事了，够了"。但有人点的时候他们还是会让人唱。

**对主角的可能影响**：主角不知道——但这首歌的原型可能与他父亲的家族有关。不是直接相连——但那种"等待一个永远不会回来的人"的情感在磨坊镇并不陌生。

### 3.3 《三个谷仓》——磨坊镇的故事

一个磨坊镇特有的、只在本地讲的故事。据说在一百多年前的一场大饥荒中——磨坊镇的三座磨坊水车停了两座（小溪枯竭），仅剩一家磨坊可磨面。周围的村镇有人走了三天的路来磨面——这座磨坊磨了谁的麦子就收他三分之一的量作为工钱。

有一夜——一个老妇人带着她仅有的一小袋麦子来到磨坊。磨坊主看了看——麦子太少，磨了之后她只剩不到半袋。他拒绝了她的工钱——说"你替我看着水车轮转三圈就好"。老妇人走了之后——溪水突然涨了——三座水车全部重新转动。

这个故事长辈会给孩子们讲——通常是在孩子们问"为什么我们不能把面包都吃掉"的时候。寓意是"分享让水车转"。

### 3.4 《守夜人的梆子》

一首白垩城流传的童谣——短小。孩子一边拍手一边唱：

```
守夜人，守夜人，
梆子敲三声——
一声叫月亮，
二声叫星星，
三声叫醒守夜人。
该你醒了——该我睡了。
```

磨坊镇的孩子们都会唱。不是很会——但不难。

### 3.5 《老水壶》

磨坊镇的酒馆有一个不正式的名字——"老水壶"。据说是多年前一位酒馆老板刚到磨坊镇时——他有一把从船上带下来的旧铜水壶。他在井边烧水——一个路过的村民问："那是你自己用的还是卖的？"他说："谁渴了谁喝。"

"老水壶"这个名字从那天开始就被叫上了。旅人问"镇上哪儿有酒馆"——村民说"老水壶"——没有一个人说"水井广场旁边那家"。

## 四、日常谚语与口头禅

> NPC在对话中自然使用这些——不是逐字引用，而是融入语言。

| 谚语 | 意思 | 谁会说 |
|------|------|--------|
| "麦子不会自己进谷仓" | 不做就不成。干活。 | 农民常说的 |
| "血河上的桥只有冬天才有" | 有些事情只能在特定条件下做。机会稀缺。 | 北部边境的人 |
| "龙不会飞过两次同一片天空" | 过去了就不会回来。不要纠结过去。 | 贵族——偶尔有文化的人 |
| "井不是为一个人挖的" | 分享。大家一起做事。 | 全镇通用 |
| "风车不能追风" | 不要试图控制不能控制的事。 | 磨坊行业的人可能会说 |
| "他脸上有面包屑" | 他隐藏了什么东西——你可以看出来。 | 酒馆老板经常用 |
| "索拉的蜡烛不白烧" | 付出总会有回报。祈祷不是免费的。 | 信教的人——助祭可能会说 |
| "他的手艺是骑士团的" | 他有真本事——不是吹的。质量可靠。 | 说某人的手艺过硬 |

## 五、地区身份认同

### 5.1 "艾尔德兰人"是什么意思

对不同的人来说，这个标签意味着完全不同的事：

| 地区 | "我们是艾尔德兰人"意味着 |
|------|----------------------|
| 中央平原 | 我们是王国的核心。白垩城在这里。国王在这里。我们是正统。 |
| 东部丘陵 | 我们养了王国的战马。没有我们——没有骑兵。 |
| 南部丘陵（磨坊镇） | "艾尔德兰"是一个很远的词。我们是磨坊镇人、灰石城人——然后才是"艾尔德兰人"。 |
| 北部边境 | 我们在保卫王国。我们是盾牌。"为什么我们交最多的税——得到的保护最少？" |

**磨坊镇的自我认同**：磨坊镇人对"艾尔德兰"几乎没有情感依恋。国王是遥远的——公爵是模糊的——灰石城是"大地方"。他们最关心的实体是磨坊镇本身。这就是为什么许多镇民对外界的征召不感兴趣——"那不是我的战争"。

### 5.2 对邻近地区的看法

| 谁 | 对谁的看法 |
|----|----------|
| 中央平原人 | 南部人是"乡下人"——迟钝但老实。 |
| 南部人 | 中央平原人是"城里人"——精明但不可靠。 |
| 北部边境人 | 所有住在南边的人都是"被保护的人"——"我们在保卫他们睡觉"。 |
| 东部丘陵人 | 中央平原人什么都不懂羊毛——"他们以为所有的羊都是白色的"。 |
| 所有人 | 白崖侯国是"那个在海边的地方"——太偏远了 |
| 磨坊镇人 | 对灰石城的看法：大城市——但灰石城的物价更贵、人说话更快、门要上锁 |

## 六、生命仪式

### 6.1 出生

出生后第三天——由教会助祭主持"赐名礼"。在教堂圣坛前——父母抱着婴儿——助祭洒一滴圣水在婴儿额头——宣布他的名字。从此他在"索拉的目光下"存在。

没有赐名礼的婴儿——从法律和教会的角度来说不存在。这就是为什么收养孤儿的家庭走的第一步是去教会办赐名礼。

### 6.2 成年

没有明确的"成年礼"。平民的成年是渐进的——能扛一袋面粉（约14岁）、能独立完成一天的耕地（约16岁）、结婚。贵族的成年是骑士册封仪式——约18-21岁。

磨坊镇的年轻人16岁——从年龄来说是成年。但很多人从没有被正式要求"成为男人"。家人没有催他们。他们给了孩子一个不存在的童年延续。

### 6.3 结婚

教会主持婚礼。新娘的父亲（或最近男性亲属）把她的手交到新郎手中——"我把她交给你了"。新郎发誓"在索拉的面前保护和供养她"。新娘发誓"在索拉的面前持家和忠诚"。然后交换一枚戒指——戴在左手无名指上（"因为那根手指的血管直接到达心脏"——尽管他们不了解血液循环但仍然这样相信）。

平民婚礼通常是小型——家人和邻居。贵族婚礼是一场政治事件——宾客名单是精心策划的信号：谁被邀请了、谁被安排坐在谁的旁边。

**没有离婚**。婚姻结束的唯一方式是死亡。寡妇可以在"合理等待期"（至少一年）之后再嫁。

### 6.4 死亡

弥留之际——如果来得及——请一位助祭或司铎来做临终告解。死者被净身、裹上白布、在第二天日落前下葬（天热时在当天——"不能让身体在索拉的目光下腐坏"）。

墓地通常在教堂旁边。贵族的坟墓在教堂内或专门的家族墓窖。平民的坟墓——一块木牌或简单石板，刻名字。磨坊镇教堂小堂旁边有一小片墓地。有人每隔两个月去扫一次墓——拔野草、放一小束野花。

葬礼后——家人守丧七天。穿深色。男人不留胡子。女人不戴首饰。七天后——"逝者已走完去往索拉的路"——生活继续。

**磨坊镇最近一次葬礼**：大约两年前——一个老羊倌过世。全镇出席。酒馆免费提供了葬后酒。老助祭在弥撒中说："他一生放了八百只羊，没有走失过一只。"——这句话被全镇人记了很久。

## 七、饮食传统

### 7.1 日常食物

| 阶层 | 早餐 | 午餐 | 晚餐 |
|------|------|------|------|
| 农民 | 燕麦粥或黑面包+水 | 面包+干肉或奶酪 | 炖菜（如果有肉）+面包 |
| 市民 | 白面包+麦酒 | 馅饼或炖肉+面包 | 烤肉（每周2-3次）+蔬菜+葡萄酒 |
| 贵族 | 白面包+蜂蜜+鲜果+葡萄酒 | 多道菜——烤肉+鱼+蔬菜+甜点 | 正式晚宴——多道菜+多个侍者 |
| 骑士 | 白面包+干肉+麦酒 | 馅饼+麦酒 | 炖肉+面包+麦酒 |

**磨坊镇的日常**：普通人家的一日三餐——早餐：黑面包+水或牛奶。午餐：黑面包+干肉或奶酪+一个洋葱或胡萝卜。晚餐：洋葱炖肉（最常）+黑面包+偶尔一个苹果或梨。没有甜点。麦酒只在晚餐——无例外。

### 7.2 磨坊镇的洋葱炖肉（家常食谱）

磨坊镇人家传的做法。一个陶罐锅。洋葱切大块——油先炒香。肉（猪肉——偶尔牛肉）切成指节大小——入锅。水没过食材。少许盐。炖一个时段——不能短。出锅前撒一把碎麦——让汤汁变稠。配黑面包。

这是磨坊镇许多人吃了多年的味道。

### 7.3 酒文化

| 酒 | 什么场合 | 谁喝 |
|----|---------|------|
| 麦酒 | 日常。晚餐。酒馆。——到处都喝 | 所有人 |
| 葡萄酒 | 有客人的晚餐。节日。教堂弥撒（圣餐——少量） | 市民及以上 |
| 蜜酒 | 节日。特别的庆祝——婚礼、岁首祭 | 所有人但少喝——贵 |
| 矮人烈酒 | "这不算是饮品"——酒馆老板会说 | 胆大的人才能喝 |

**在酒馆点酒的社会规则**：如果有人请你喝一杯——你必须喝。如果你想回请——礼貌——但不是必须的。如果你连续拒绝了三个人的请酒——你可能会被视作"孤僻"或"傲慢"。

## 八、手势与身体语言

| 手势 | 含义 | 什么情境 |
|------|------|---------|
| 右手放在左胸心口 | "以我的良心起誓" | 承诺或宣誓时 |
| 双手摊开掌心朝上 | "我什么都没拿"——诚实、透明 | 商人展示货物、被指控者自证清白 |
| 低头 + 右手触额然后指地 | "向索拉致敬"——简略版的宗教致敬 | 经过教堂时、弥撒中 |
| 轻碰鼻尖然后指向对方 | "我记住你了"——不一定是善意的 | 纠纷中 |
| 掌心朝前推出 | "够了。"——停止的意思 | 讨价还价——出价被接受 |
| 两个食指并拢 | "我们一样"——同伴、搭档、同意 | 朋友间、商业协商中 |
| 吐口水在地上（在对方说完话后） | 最大的侮辱——"你的话不值一口唾沫" | 决斗前。别轻易用。 |

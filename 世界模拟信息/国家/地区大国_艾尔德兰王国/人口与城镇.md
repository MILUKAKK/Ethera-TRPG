## 一、总人口与结构

### 1.1 基本数据

| 指标 | 数值 | 说明 |
|------|------|------|
| 总人口 | 约380万 | 人类98%，精灵混血1.5%，其他0.5% |
| 成年男性（16-60岁） | 约95万 | 25%为总人口——基于中世纪人口结构 |
| 适兵役男性（18-40岁，身体健全） | 约38万 | 成年男性的约40% |
| **实际可征召上限** | **约1.9万** | 适兵役男性的5%——超过此比例则农业崩溃（见 `军事.md` §五） |
| 理论征召义务 | 约1.2万 | 各领主封地人口×1% |
| 实际征召 | 约8,000 | 领主短少 + 农业季节约束 |
| 年出生率 | 约3.5% | 高出生率——但被高死亡率对冲 |
| 年死亡率（正常） | 约2.5% | 含婴儿死亡（约30%活不过周岁） |
| 人口年增长率 | 约1% | 极缓慢增长。瘟疫年则骤降 |
| 户均人口 | 约5人 | 含祖辈、父母、子女 |

### 1.2 阶层结构

| 阶层 | 占总人口 | 人数 | 特征 |
|------|---------|------|------|
| 王室及直系 | <0.01% | 约30人 | 阿尔德里克家族 + 姻亲 |
| 高级贵族（公爵/伯爵） | 约0.5% | 约2万 | 含家族成员、远亲 |
| 低级贵族/骑士 | 约2% | 约7.5万 | 骑士阶层——军事指挥人员的唯一来源 |
| 神职人员 | 约1.5% | 约5.5万 | 修士/修女/司铎/助祭（含教会雇工） |
| 城市市民（商人/工匠/自由职业者） | 约8% | 约30万 | 识字率约30% |
| 农民（自耕农 + 佃农） | 约75% | 约285万 | 识字率<5%。农业劳动力的绝对主体 |
| 农奴/无地劳动者 | 约10% | 约38万 | 无自由迁徙权。流动性为零 |
| 其他（流浪者/艺人/边缘人） | 约3% | 约11万 | |

**军事含义**：骑士阶层仅7.5万人——这是骑士团军官和小队长的唯一人才池。征召步兵来自农民阶层——但农民不在田里则无人种粮。

### 1.3 人口年龄分布

| 年龄段 | 占比 | 人数 | 说明 |
|--------|------|------|------|
| 0-14岁 | 35% | 约133万 | 高出生率+高婴儿死亡率 |
| 15-30岁 | 25% | 约95万 | 征兵主力年龄段 |
| 31-50岁 | 25% | 约95万 | 主要劳动力 |
| 51岁以上 | 15% | 约57万 | 寿命预期约55岁（平民）/65岁（贵族） |

**军事含义**：适兵役男性38万看起来很多——但征召8,000人已经让农业产出下降15-20%。如果再征召到理论上限12,000人，农业减产幅度将达到25%以上——在秋收季节这是**不可逆的粮食危机**。

## 二、区域人口分布

### 2.1 总览

| 区域 | 人口 | 占比 | 主要城镇 | 经济支柱 | 军事义务 |
|------|------|------|---------|---------|---------|
| 中央平原（核心区） | 约180万 | 47% | 白垩城（12万）、丰收城（5万）、河渡北镇（2万） | 小麦、葡萄酒、手工业 | 提供中央征召步兵约2,500 + 圣盾骑士团兵源 |
| 东部丘陵 | 约80万 | 21% | 绿林镇（3万）、石桥城（2.5万） | 羊毛、牧业 | 提供东部征召步兵约1,000 |
| 南部丘陵与边境 | 约70万 | 18% | 灰石城（4万） | 皮革、铁矿加工（铁料来自北方） | 提供南境征召步兵约1,500 |
| 北部血河边境 | 约50万 | 13% | 河卫城（3万——军事要塞） | 边境贸易（和平期）、军事服务业 | 提供北境征召步兵约3,000 |
| **合计** | **380万** | **100%** | | | **约8,000** |

### 2.2 中央平原（核心区）

**地理**：银带河冲积平原，黑土深厚。大陆最肥沃的农业区。南北约400km，东西约300km。

**人口密度**：约45人/km²（全王国最高——对比南部约25人/km²、北部约15人/km²）。

**城乡比例**：约12%城市人口（白垩城+丰收城+河渡北镇），88%农村人口。白垩城12万人中，约3万是贵族/神职/官员及其家属，约5万是工匠/商人/服务业，约4万是城市贫民。

**经济特征**（详见 `经济.md`）：王国财富最集中的区域。年产小麦约200万吨——占王国农业产出的60%。白垩城铁匠公会是全大陆品质标杆。丰收城是全国最大的粮食集散地。

**政治特征**（详见 `贵族与皇室.md`）：十二公爵中有6位的领地位于中央平原。王权派的核心票仓在此。但同时旧权派领袖黑木公爵的领地也在中央平原北部——这意味着中央平原内部也存在王权派vs旧权派的裂痕。

### 2.3 东部丘陵

**地理**：低缓丘陵地带，海拔300-800m。土壤贫瘠于中央平原，不适合大规模小麦种植。

**人口密度**：约20人/km²。居民分散于小型聚落。

**经济特征**：牧场为主——羊和少量牛。年产羊毛约5,000吨——其中约60%经白垩城加工后出口。东部牧区也是王国战马的唯一繁育地（年产约300匹战马——不可压缩的自然周期）。

**政治特征**：东部领主（3位公爵）在议会中通常跟随风向——他们没有旧权派那么强硬，也没有王权派那么热忱。他们的核心诉求是羊毛出口关税的下调。

**军事弱点**：东部丘陵80万人口——但征召义务仅约1,000人。不是因为人口不足，而是因为东部人口分散且牧业劳动力**不可替代**（一个牧人管理200只羊，他走了羊群就散了）。这就是裂脊隘口仅200守军的人口学原因。

### 2.4 南部丘陵与边境

**地理**：灰石城以南的丘陵地带。越往南越荒芜，人烟稀少。磨坊镇位于通往南方的官道沿线。

**人口密度**：约25人/km²——但分布不均。灰石城周边较密集（依赖灰石城的市场和皮革加工业）；更南的乡村（如磨坊镇）人口稀疏。

**经济特征**：灰石城是南方皮革集散中心——全国约70%的皮革加工在此完成。磨坊镇及周边乡村以小麦和面粉为主——三座水车磨坊是区域地标。南部不产铁——铁料完全依赖北方输入。

**政治特征**：南境领主（2位公爵——包括灰石城公爵）是旧权派的坚定支持者。原因：南部农业规模小、土地分散——土地税对他们打击最大。

**剧情关联**：灰石城领主签署了征召令——这意味着南部70万人口中正在被抽走青壮劳动力。磨坊镇因为太小（200人），暂时没有直接收到征召令——但传令官正在灰石城周边村镇逐一通知。

### 2.5 北部血河边境

**地理**：血河以南的狭长地带。宽度从20km到80km不等。越靠河地势越平坦——适合骑兵移动，也适合帝国渡河。

**人口密度**：约15人/km²——但人口构成特殊：约40%为军事相关人员（守军、铁匠、马夫、军需雇员），30%为农民，20%为边境商人，10%为逃兵/走私者/流浪者。

**经济特征**：和平时期——边境贸易是主要收入。艾尔德兰的小麦换帝国的矿石（关系正常时）。战争时期——贸易中断，要塞周边农民撤向内陆，区域经济冻结。

**军事特征**：此区域承受王国80%的军事压力。12座要塞沿血河南岸分布。河卫城（3万人）是纯粹的军事城镇——城内军械库、马厩、粮仓占了一半面积。所有居民的生活都与军事相关。

### 3.2 城镇人口占总人口比例

约18%（约68万人在城镇）。对比：维斯特法尔帝国约25%（更工业化——军工产业吸纳更多城市人口），洛桑联邦约40%（商业共和国——城市化程度最高）。

## 四、人力资源约束

### 4.1 农业劳动力的不可替代性

285万农民养活了380万人。每个农民平均养活1.33人（含自己）。封建农业效率极低——一个农民产出的余粮仅够养活0.33个非农业人口。

这意味着**每征召一个农民**：王国失去0.33人份的粮食盈余。征召8,000人 → 粮食盈余减少2,640人份。在丰收年景这不是问题——在歉收年或征召时间落在秋收季（如果月）则可能是灾难。

**果月征召的特殊危害**：灰石城的征召令在果月（秋收月）发出——此时正是收割最紧张的时期。每征召一个农民 = 他家的麦子烂在地里。南部丘陵约70万人口——如果征召1,500人，对应的约7,500人份的粮食受影响。在这个规模下——南部可能在冬月出现局部饥荒。

### 4.2 骑士阶层的规模约束

全王国约7.5万骑士阶层（含家属）。其中实际可担任军事指挥的成年男性骑士约1.5万。两个骑士团需要约200名军官（大队长+中队长+小队长）——占用骑士人才池的仅1.3%。

**但这不是瓶颈。** 瓶颈在于骑士团团长和王国元帅层级——只有最顶层的几十人。而且这些位置的选拔是政治性的：需要王室信任+议会认可。当前元帅空缺——意味着这1.5万骑士中**没有人能同时获得国王和议会的双重认可**。

### 4.3 识字率与专业化兵种

识字率约10%（全国），城市约30%，乡村<5%。这意味着：
- 需要读写的军事岗位（军需官、文书、地图绘制）——只能从城市人口中招募
- 铁匠——全国约2,000名铁匠（含学徒）。战时每个铁匠可以带3-5个助手加速生产
- 军医——全国受过正式训练的医师不足200人。大部分战地救治依赖草药师和助祭

这些不是"军费"问题——是"人才池"问题。给再多钱也无法短期增加铁匠或医师数量。

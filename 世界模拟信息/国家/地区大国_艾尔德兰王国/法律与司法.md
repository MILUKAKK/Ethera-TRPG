# 艾尔德兰王国 · 法律与司法

> 此文件为GM判定玩家行为法律后果的核心依据。
> 关联：`贵族与皇室.md` — 贵族受贵族议会审判 | `宗教.md` — 教会拥有独立法庭 | `军事.md` — 军事审判独立体系

## 一、法律体系总览

### 1.1 三套并行体系

艾尔德兰的司法不是单一系统——是三套体系的叠加。同一行为在不同体系中可能合法也可能非法。

| 体系 | 管辖范围 | 法律来源 | 审判者 | 上诉终点 |
|------|---------|---------|--------|---------|
| **王室法庭** | 全体平民（刑事案件）+ 全国范围内的王室法令执行 | 《艾尔德兰法典》+ 王室敕令 | 王室法官（国王任命） | 国王 |
| **领主法庭** | 封地内的民事纠纷 + 封地平民的轻罪 | 地方习惯法 + 领主自行颁布的条例 | 领主或其指定的治安官 | 无——领主法庭为终审（除非涉及王室法令，可上诉至王室法庭） |
| **教会法庭** | 神职人员 + 涉及宗教事务的所有人（含平信徒的婚姻/遗嘱/亵渎等） | 教会法（圣光法典） | 主教或主教指定的司铎 | 大主教→教皇 |

**冲突规则**：当三套体系对同一案件都有管辖权时——**贵族优先选择权**。被告的阶级决定其受审的法庭。平民无权选择。

### 1.2 法律阶级性

| 阶层 | 刑事审判 | 民事审判 | 特殊权利 |
|------|---------|---------|---------|
| 王族 | 贵族议会特别审判 | 国王直接裁决 | 不受普通法律约束 |
| 高级贵族（公爵/伯爵） | 贵族议会 | 贵族议会 | 可要求"决斗审判" |
| 低级贵族/骑士 | 王室法庭（国王特设） | 领主法庭 | 不可被平民起诉（需先经贵族担保） |
| 神职人员 | 教会法庭 | 教会法庭 | 免于世俗审判——除非教皇授权移交 |
| 城市市民 | 王室法庭或领主法庭（视罪行严重程度） | 领主法庭 | 享有行会内部仲裁 |
| 农民/农奴 | 领主法庭 | 领主法庭 | 几乎无权利 |

## 二、实体法律

### 2.1 《艾尔德兰法典》

建国时（纪元1200年）由阿尔德里克一世颁布。共九卷，涵盖刑法、财产、契约、婚姻、继承等。为所有王室法令的法律基础。

| 卷 | 内容 |
|-----|------|
| 第一卷 | 国王的权力与限制 |
| 第二卷 | 贵族权利与义务 |
| 第三卷 | 平民权利与义务 |
| 第四卷 | 刑法——罪行与刑罚 |
| 第五卷 | 财产法与契约 |
| 第六卷 | 婚姻、家庭与继承 |
| 第七卷 | 土地法（附封建义务） |
| 第八卷 | 商业与市场规范 |
| 第九卷 | 审判程序与证据 |

法典的一个关键原则——**"王冠的和平"**：任何扰乱"国王的和平"（public peace）的行为，领主法庭可以审理，但王室法庭保留优先管辖权。哪些行为构成"扰乱王冠的和平"由王室法官自由裁量——这是王室扩大司法权的核心工具。

### 2.2 领主习惯法

各封地存在独立于法典的地方习惯法——尤其是土地边界、水源使用、牲畜赔偿等领域。同一个行为在一位公爵的领地上是犯罪，在另一位公爵的领地上可能只是"不当行为"。

**对玩家的影响**：主角从磨坊镇（灰石公爵领地）进入白垩城（王室直辖区）→ 法律环境发生变化。GM需要在剧情中透露这些差异。

### 2.3 教会法与世俗法的交叉

| 领域 | 世俗法 | 教会法 | 优先 |
|------|--------|--------|------|
| 婚姻 | 财产分配、继承排序 | 婚姻有效性、离婚（禁止的） | 教会——婚姻是圣礼 |
| 遗嘱 | 遗嘱继承程序 | 遗嘱是否为"在索拉面前真诚的遗愿" | 教会——因为关乎灵魂 |
| 亵渎/异端 | 王国法典不涉及 | 教会法惩罚——自苦修至火刑 | 教会 |
| 偷窃教会财产 | 世俗偷窃法 | 教会法（加重的——"对圣物的侵犯"） | 教会（优先）或王室（如果教会移交） |
| 神职人员犯罪 | 理论上可移交王室 | 教会法庭优先 | 教会——除非教皇特批移交 |

## 三、刑事犯罪与刑罚

### 3.1 罪行分类

| 等级 | 罪行示例 | 审判法庭 | 刑罚范围 |
|------|---------|---------|---------|
| 重罪（Felony） | 杀人、强奸、纵火、叛国、铸造伪币 | 王室法庭（不可由领主法庭裁决） | 死刑至流放 |
| 中罪（Misdemeanor） | 盗窃（金额>1银）、伤害、欺诈、抢劫 | 领主法庭或王室法庭 | 鞭刑/烙印/罚金/监禁 |
| 轻罪（Petty Offense） | 小偷小摸、言语冒犯、醉酒滋事 | 领主法庭 | 罚铜/枷锁示众/强制劳作 |
| 宗教罪行 | 亵渎、异端、在安息日工作 | 教会法庭 | 苦修/罚金/逐出教会 |

### 3.2 刑罚阶梯

| 刑罚 | 适用罪行 | 适用阶层 | 说明 |
|------|---------|---------|------|
| 罚铜/罚银 | 轻罪、轻微伤害 | 所有 | 最常用。罚金进入领主或王室国库 |
| 枷锁示众 | 小偷、欺诈者 | 平民 | 锁在镇广场的木枷中1-3日。围观者可投掷腐烂食物——有时致死 |
| 鞭刑 | 盗窃、伤害、通奸（平民） | 平民 | 鞭数视罪行：10-50鞭。可致永久伤害 |
| 烙印 | 重复犯罪者、伪造货币 | 平民 | 在面颊或手背烙印——永久标记 |
| 强制劳役 | 无力偿还罚金者 | 平民 | 在王室或领主领地上强制劳作1-5年 |
| 监禁 | 等待审判时、债务不还 | 所有 | 监狱不是刑罚——是等待场所。长期监禁极少 |
| 流放 | 重罪（减刑）、政治犯 | 所有（贵族通常先被流放而非处决） | 剥夺所有财产+公民身份。被流放者在他国不受法律保护 |
| 绞刑 | 杀人、强奸、纵火、抢劫（重罪） | 平民 | 最常见的死刑。在镇广场公开执行 |
| 斩首（剑刑） | 同上——但保留"体面" | 贵族 | 用剑而非斧——"体面的"死刑。由骑士执行 |
| 火刑 | 异端、巫术 | 所有——但由教会法庭判决 | 极为罕见。最近一次在140年前 |
| 叛国 | 弑君、叛乱、通敌 | 所有 | 绞刑+分尸+首级悬挂城门——最严厉的公开惩罚 |

**死刑执行频率**：白垩城每年约20-30例公开处决（人口12万——比例不高）。谋杀在乡村常被"私下解决"——领主未必知晓。

### 3.3 特定罪行的判定

**杀人**：
- 故意杀人（谋杀）→ 绞刑或斩首
- 过失杀人（如斗殴中失手）→ 鞭刑+罚金+可能流放。若受害方接受"血钱"（赔偿金），可不进入正式审判
- 自卫杀人 → 需证人+领主/法官认可。无罪
- 贵族杀死平民 → 通常以罚金解决。极少上升为死刑
- 平民杀死贵族 → 绞刑是起点——不论是否自卫。大概率叛国

**盗窃**：
- 偷窃价值<1银 → 罚金+可能枷锁
- 偷窃价值>1银 → 鞭刑+罚金
- 偷窃贵重品（>1金）→ 可能截手（南部部分领地保留此刑罚）
- 偷窃教会财产 → 教会法加罚
- 偷窃贵族财产 → 贵族可要求加刑

**通奸**：
- 法典定义通奸为"已婚女性与婚外男性性交"（是的——法典只约束女性）
- 刑罚：女性鞭刑20-30鞭（领主裁量），男性鞭刑10-20鞭或罚金
- 贵族间的通奸——通常是私下解决（赔偿、决斗）

**叛国**：
- 武装叛乱、通敌、弑君
- 刑罚：绞刑+分尸+首级悬挂城门
- 贵族叛国 → 全家剥夺爵位+封地+财产+流放或处决
- **当前政治敏感**：公开批评王太子（而非国王本人）是否构成"叛国"——法典没有清晰的定义。这是旧权派可用于政治斗争的法律漏洞

## 四、民事法律

### 4.1 财产法

**土地所有**：贵族拥有封地、教会拥有15%土地、自耕农拥有约10%、无地者——没有财产。土地的买卖需要领主许可——实际上是禁止的（封建制度的根基：土地不可买卖）。

**动产**：平民可自由拥有和买卖动产——家畜、农具、家具、工具等。商人可拥有库存和贸易品。

**遗产**：无遗嘱 → 长子继承制（贵族封地）或均分（平民。注意：法典对此没有统一规定——取决于领主法庭的地方习惯法）。有遗嘱 → 教会法庭验证遗嘱有效性后按遗嘱分配。

### 4.2 契约法

| 契约类型 | 形式要求 | 执行机构 |
|---------|---------|---------|
| 口头协议（<1银） | 无——双方自证 | 无强制执行力——全靠信誉 |
| 口头协议（>1银） | 需有证人（至少2人） | 领主法庭——但执行困难 |
| 书面契约 | 双方签名+证人+（可选）公证人 | 王室法庭——有执行力 |
| 商人公会契约 | 按公会内部规则 | 公会内部仲裁→领主法庭 |

**违约后果**：罚金（合同约定金额的2-3倍）+ 赔偿损失。无力偿还者 → 债务监禁至有人代为偿还。负债监禁没有期限上限——有些人在监狱中度过余生因为欠了永远还不清的债。

### 4.3 婚姻法

- 结婚最低年龄：女12岁 / 男14岁（教会法规定）。贵族通常在16-20岁结婚
- 婚姻需要教会主持的仪式（否则"无效"——财产分配不适用）
- **离婚不存在**——教会法禁止。唯一的"解除"方式是教会宣告婚姻自始无效（罕见的例外——如近亲结婚被发现）
- 财产：婚后财产属于丈夫（除非婚前签订特殊契约——贵族间才有）
- 寡妇得到丈夫财产的"合理份额"（通常三分之一至一半——视是否有子女而定）

## 五、审判程序

### 5.1 领主法庭的日常审判

大多数案件在领主法庭——快速、便宜、不可预测。

| 步骤 | 说明 |
|------|------|
| ① 控告 | 原告向领主或其治安官提出控告（口头即可） |
| ② 传唤 | 被告被传唤至法庭——通常在集市日，以方便证人出席 |
| ③ 审理 | 法官（领主或治安官）听取双方陈述+证人证词。无律师——双方自己辩 |
| ④ 证据 | 证人证言+物证。无物证鉴定——全靠观察和询问。有罪推定是默认的 |
| ⑤ 判决 | 法官自行裁决——无陪审团。定罪标准远低于现代"排除合理怀疑" |
| ⑥ 刑罚 | 立即宣布+当场执行（罚金/鞭刑）。重罪（绞刑）可能延至下一次集市日以聚集围观者 |

### 5.2 王室法庭

更正规——用于重罪、贵族被告的重大案件、涉及两个不同领主领地之间的纠纷。

| 特征 | 说明 |
|------|------|
| 法官 | 国王任命的王室法官——通常是受过教育的神职人员或受过法学训练的文官 |
| 程序 | 书面起诉状（需付费用——排除了穷人）+ 双方陈述 + 证人宣誓 |
| 证人宣誓 | 证人按《圣光启示录》宣誓。拒绝宣誓者被视为有罪 |
| 判决 | 王室法官有较大的自由裁量权。受《艾尔德兰法典》约束但不是严格的"依典判决" |
| 上诉 | 可向国王申诉。极少——成本极高。国王正常不会为此接见平民 |

### 5.3 决斗审判

贵族被告可以要求"决斗审判"——在与原告或王室指定斗士的一对一战斗中证明自己的清白。理论基础：索拉不会让正义的一方在决斗中失败。

- 仅对重罪（杀人/叛国/强奸）的案件可以请求
- 仅贵族有权请求
- 决斗到一方失去继续战斗的能力（不是必须死亡——但经常以死亡告终）
- 如果被告胜出 → 当庭释放，不得以同一罪名再次起诉
- 如果被告年迈或残疾 → 可以指定代斗士（需法庭认可）

**最近一次决斗审判**：约30年前，一位东部骑士因被指控杀害竞争对手而被审判。他在决斗中杀死了王室指定的斗士——被当庭释放。他后来死于决斗中的伤口感染——在某种意义上，索拉确实给出了裁决。

### 5.4 拷问

拷问在法律上不合法——但现实中广泛存在。尤其是在领主法庭中——领主可以命令治安官对不合作的被告或证人施加身体压力。王室法庭至少在形式上禁止拷问被告——但允许对**拒不回答的证人**施加"压力"。

## 六、执法

### 6.1 谁在执行法律？

| 执法者 | 管辖范围 | 权力 |
|--------|---------|------|
| 领主治安官 | 封地内 | 逮捕、搜捕、初步审讯、行刑 |
| 王室警卫（仅白垩城） | 白垩城市区 | 维护城市秩序 |
| 城堡守卫 | 各城堡/要塞 | 守卫——非执法——但可临时执法 |
| 镇上的"守夜人" | 村镇（如磨坊镇——目前空缺） | 夜间巡逻+大声通报（"一切安好！"）+ 发现火情或犯罪时呼叫支援 |
| 教会助祭 | 与道德相关的微罪 | "道德执法"——公开谴责+向教会汇报 |

**关键漏洞**：没有"警察"概念。没有职业化的执法力量。大部分农村地区——一个领主治安官管着上万人的区域。他不可能在每一个犯罪现场出现。大多数犯罪是通过受害方自己找到犯罪者并告到领主法庭来解决的。

### 6.2 对玩家的影响

- 在磨坊镇（200人）：灰石公爵的治安官每季才来一次巡查。平时——犯罪基本靠村镇自治：受害者或其家人找到犯罪者→带至领主法庭→如果找不到则不了了之
- 在灰石城（4万人）：城防有守卫，领主有治安官，街道有巡逻。执法密度高于乡村但仍稀疏
- 在白垩城（12万人）：白垩城有王室警卫~100人。市内秩序相对维持。但贫民窟（太阳巷）基本无人管

## 七、法律-阶级-剧情关联

### 7.1 法律对主角的现实约束

主角是平民——磨坊镇16岁少年。以下行为会触发法律后果：

| 行为 | 最轻结果 | 最重结果 | 审判法庭 |
|------|---------|---------|---------|
| 盗窃（价值<1银） | 不需审判——私下解决 | 鞭刑10鞭+枷锁 | 灰石公爵领主法庭 |
| 盗窃（价值>1银） | 鞭刑+罚金 | 截手（南部可能保留） | 同上 |
| 与贵族发生冲突（非暴力） | 贵族不予追究 | 贵族要求领主惩罚"侮辱案" | 领主法庭——贵族主导 |
| 对贵族造成伤害 | 绞刑（最低量刑） | 叛国罪（如果你在"战争时期"伤害了在役贵族） | 王室法庭 |
| 对平民造成严重伤害 | 罚金+鞭刑 | 绞刑（如果对方死亡+有前科） | 领主法庭 |
| 未经许可离开封地 | 返回原封地 | 被抓到其他封地时可能被当作"逃离农奴" | 领主法庭 |
| 伪造货币 | — | 绞刑（固定——铸币权是王冠的） | 王室法庭 |
| 偷窃教会财产 | 鞭刑+归还+教会罚金 | 可能移送至教会法庭+监禁 | 领主→可能移送教会 |
| 在安息日（日曜日）工作 | 教会助祭口头警告 | 小罚金 | 教会法庭 |

### 7.2 当前王国的法律灰色地带

**王太子的特殊处境**：王太子艾德温是**摄政**——不是国王。法典中"摄政"的权力边界从未被测试过。旧权派可以在法律上声称某些摄政令"超出了摄政权限，需要国王本人签署"——即使国王病卧无法签署。这是旧权派的潜在法律武器。

**税改的合宪性疑问**：法典第三卷（平民权利与义务）中是否有任何条款规定"国王不能向贵族征收土地税"——答案是没有。法典从未明文禁止这种税——因为六百年来从未有人试图征收。这就是为什么税改是一个法律上可行但政治上受阻的方案。

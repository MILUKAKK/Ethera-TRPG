# 艾尔德兰王国 · 财政

> 此文件为LLM推理王国政府财政运作的核心依据。所有政府收支、税制、债务、预算审批均在此定义。
> 关联：`经济.md` — 实体经济是税基 | `军事.md` §三 — 军费是最大支出项 | `贵族与皇室.md` — 议会控制预算审批 | `宗教.md` — 教会是最大债权人

## 一、财政体制

### 1.1 性质

艾尔德兰的财政是**封建领主代征 + 国王集中支配**的混合体。财政大臣管理的是国王的"国库"——在法理上它是国王的个人财产，在功能上它是王国的政府预算。

| 特征 | 运作方式 |
|------|---------|
| 征税权 | 国王提案 → 贵族议会投票批准 → 各领主代为征收 |
| 支用权 | 常规支出由国王/财政大臣支配；额外支出需议会特批 |
| 审计 | 无独立审计。财政大臣署自行记账——国王偶尔过目 |
| 债务 | 以"王冠"名义向教会/银行借款——实质是国王个人信用 |
| 铸币权 | 王室铸币厂垄断。私铸者死刑 |

### 1.2 财政年度

每年雪月（12月），财政大臣向议会提交下一财年预算案。冰月（1月）第一次议会投票。通过后即成为该财年（冰月至雪月）的支出上限。

**当前财政年度**：纪元1782财年。预算于去年雪月通过。**本财年的预算基础是旧税制的人头税收入——税改即使通过也无法影响本财年。**

### 1.3 预算的约束力

预算一旦通过，财政大臣不能超支。两项例外：
- **国王紧急敕令**：路德维希三世在位42年仅使用11次
- **议会补充拨款**：需重新投票——在分裂的议会中几乎不可能

这意味着：如果战争在预算通过后爆发——王国只能靠紧急敕令维持军费，但紧急敕令不能无限使用。

## 二、政府收入

### 2.1 收入结构（1782财年）

| 收入来源 | 年收入 | 占比 | 征收方式 | 承担者 |
|---------|--------|------|---------|--------|
| 人头税 | 约570万金 | 67% | 各领主按封地人口代征 → 上缴财政大臣署 | 全体平民（15岁以上） |
| 王室领地 | 约100万金 | 12% | 王室管家直管——农业产出 + 贸易抽成 | — |
| 关税 | 约80万金 | 9% | 边境税关按货值征收（进口5-15% / 出口3-10%） | 商人 |
| 领主上贡 | 约50万金 | 6% | 各公爵按惯例不定期"自愿"上贡——实质是封建义务 | 十二公爵 |
| 铸币税 | 约20万金 | 2% | 铸币面值与金属成本的差额 | — |
| 特别税 | 约30万金 | 4% | 战时附加税——需议会特批 | 全体（含部分贵族） |
| **合计** | **约850万金** | **100%** | | |

### 2.2 人头税——财政基石的脆弱性

**征收逻辑**：领主在果月（秋收后农民手中有粮/钱）按封地内15岁以上平民人数征收，扣除5-15%"征收成本"后上缴。税率由王室与议会协商确定——各领地理论上是统一税率。

**实际税率**：折合约每人每年2.0-2.5金（因领主截留比例和各地折算方式不同而有差异）。对于一个年收入约3-4金的佃农来说——**税负约60-80%**。这是封建社会的常态——农民在交税后仅能维持基本生存。

**征收损耗**：

| 环节 | 损耗 | 说明 |
|------|------|------|
| 领主征收截留 | 5-15% | 以"征收成本"名义。大领主截留比例更高 |
| 运输/折色损耗 | 2-5% | 实物税折成货币时的估价差 |
| 财政大臣署行政 | 3% | 官僚运行成本 |
| **从农民到国库总损耗** | **约15-20%** | |

**结构性缺陷**：人头税与收入无关。一个拥有万亩良田的公爵缴纳的人头税与一个无地佃农相同（贵族本人虽然"豁免人头税"——但他们的土地上耕种的佃农在缴税）。贫富差距被税制固化。

### 2.3 王室领地——国王的私人收入

王室直领地约15%的全国可耕地（集中在中央平原核心区）。产出不经过领主代征——国王的管家直接管理。这100万金是国王最可靠的收入——不受议会投票影响、不受领主截留。

**政治含义**：如果议会彻底瘫痪——国王仅凭王室领地收入仍可维持基本的王室运转和部分行政功能。但军费（300万金）完全不可能从领地收入覆盖。

### 2.4 关税——唯一有增长弹性的收入

关税从80万金的基数有增长空间——如果贸易扩大。但关税受制于：
- 与帝国的边境贸易（和平期约贡献关税的20%）— 随时可能中断
- 洛桑航线的安全（香料船延误已有先例）
- 关税提高会遭到洛桑的贸易报复

## 三、政府支出

### 3.1 支出结构（1782财年）

| 支出项 | 金额 | 占比 | 刚性/弹性 | 审批方 |
|--------|------|------|----------|--------|
| 军费 | 约300万金 | 35% | 刚性——不可削减（帝国威胁） | 常规预算 |
| 行政 | 约250万金 | 29% | 半刚性——官员俸禄不可削，修路可延 | 常规预算 |
| 王室开支 | 约100万金 | 12% | 半刚性——王宫维护必须，生活可俭 | 常规预算 |
| 贵族津贴 | 约50万金 | 6% | 弹性——但削减会激怒贵族 | 常规预算 |
| 赈灾/意外 | 约50万金 | 6% | 弹性——视年景 | 常规预算 |
| 教会补贴 | 约30万金 | 4% | 弹性——但削减有政治代价 | 常规预算 |
| 债务利息 | 约20万金 | 2% | 刚性——违约后果严重 | 常规预算 |
| 铸币成本 | 约15万金 | 2% | 刚性 | 常规预算 |
| 杂项 | 约35万金 | 4% | — | 常规预算 |
| **合计** | **约850万金** | **100%** | | |

### 3.2 支出刚性分析

**不可削减的核心支出**（约335万金）：军费300万 + 债务利息20万 + 铸币15万。占总支出的39%。

**可延但不可削的支出**（约350万金）：行政250万 + 王室100万。这些支出可以拖欠几个月——但不能取消。

**可裁量的支出**（约165万金）：贵族津贴50万 + 赈灾50万 + 教会补贴30万 + 杂项35万。理论上可以削减——但每一项都有政治代价。

**结论**：即使王国进入财政紧急状态，最多只能削减约100-150万金的支出——远不足以应对一场全面战争每月75-95万金的额外开支。

## 四、税改——财政的核心死锁

> 此为国家当前一切政治冲突的财政根源。

### 4.1 现行税制的数学

| 群体 | 年收入 | 年缴税 | 税负 |
|------|--------|--------|------|
| 无地佃农 | 约3金 | 约2金（人头税） | ~67% |
| 自耕农（有地） | 约8金 | 约2金（人头税） | ~25% |
| 城市工匠 | 约15金 | 约2金（人头税） | ~13% |
| 富商 | 约200金 | 约2金（人头税）+ 关税间接 | ~5% |
| 低级骑士 | 约30金 | 免税（贵族） | 0% |
| 公爵（如黑木） | 约50万金（土地租金+贸易抽成） | 仅领主上贡约4万金 | ~8% |

**核心不公**：税负从佃农的67%递减到公爵的8%。这不是有意设计——是人头税的结构性结果。

### 4.2 王太子方案

**税改内容**：
1. 人头税减半：每人约1金（减轻平民负担约50%）
2. 新增土地税：按土地面积+地力分级征收。优质农田税率高于贫瘠牧场
3. 王室直领地和教会土地暂不在土地税征收范围内（换取教会中立）

**财政效果估算**：
- 人头税减半 → 损失约285万金
- 土地税新增 → 预计增收约400-500万金（取决于最终税率和地力分级）
- **净效果：财政收入预计增加约120-220万金/年**

**分配效果**：
- 农民（285万自耕农+佃农）：税负减轻约50%
- 城市市民和商人：几乎无影响
- 小贵族（封地小）：土地税增加但封地不大——影响有限
- 大贵族（封地大）：土地税将使其年缴税可能从过去的几乎为零增至数万至数十万金

### 4.3 反对派的核心恐惧

旧权派反对的不是税额本身——而是**先例**。一旦国王可以"按土地向贵族征税"，封建制度的根本契约——"贵族以服兵役代替纳税"——就瓦解了。

黑木公爵在议会闭门会议中说过："今天他要我们的钱。明天他要我们的兵。后天他不需要我们了。"

### 4.4 议会票数（关联 `贵族与皇室.md`）

- 旧权派（反对）：5票
- 王权派（支持）：4票（含白崖侯爵）
- 摇摆：3票（均为东部领主）
- **通过需至少7票**

税改的命运 = 三位东部领主中至少两位倒向支持。

## 五、国债与借款能力

### 5.1 当前债务

| 债权人 | 金额 | 年利率 | 抵押品 | 到期 |
|--------|------|--------|--------|------|
| 圣光教会 | 约100万金 | 8% | 王室领地部分农业收入 | 无固定到期——教会可随时要求续约谈判 |
| 金冠银行（洛桑） | 约100万金 | 12% | 未来三年关税收入 | 1784财年到期（2年后） |
| **合计** | **200万金** | | | |

年利息约20万金——占财政收入的2.4%。看似可控。但本金200万金是王国年收入的24%——在当时的封建经济中这已经是较高的负债率。

### 5.2 借款能力

**最大额外借款能力（估算）**：
- 从教会：约50-100万金（教会余钱有限——且卡勒斯可能拒绝）
- 从金冠银行：约50万金（银行已在洛桑内部承压——可能收紧信贷）
- 从洛桑其他商行：约30-50万金（利率更高——15-20%）
- 从铁冠王国（国家借贷）：约20-30万金（铁冠是小国——借贷能力有限）
- **合计额外借款能力：约150-230万金**

**一场全面战争每月额外支出75-95万金** → 即使全部借款能力耗尽，也只能支撑2-3个月的战争。之后王国将面临违约。

### 5.3 违约后果

| 违约对象 | 后果 |
|---------|------|
| 圣光教会 | 绝罚威胁（国王被逐出教会）→ 贵族可能以此为借口拒绝效忠。终结阿尔德里克王朝。 |
| 金冠银行 | 洛桑金融制裁 → 艾尔德兰商人在洛桑的资产被冻结 → 贸易大幅缩减 → 关税收入崩溃。 |

**因此：教会不能违约、银行不敢违约。** 战时的最后手段是向教会恳请债务展期——而教会的答复取决于卡勒斯的政治计算。

## 六、财政危机触发场景

| 场景 | 触发 | 财政影响 | 发生概率 |
|------|------|---------|---------|
| 局部边境冲突 | 帝国越河小规模进攻 | 每月+5-10万金军费 | 中（1-3年内） |
| 全面战争 | 帝国正式宣战 | 每月+75-95万金。需紧急借款。 | 低（1-5年内）但一旦发生则致命 |
| 粮食歉收 | 中央平原旱灾 | 人头税-10-20% + 赈灾+20万金 | 中（周期性——最近一次在15年前） |
| 金冠银行催贷 | 洛桑内部危机迫使银行收回贷款 | 需立即偿还100万金 → 王国流动性崩溃 | 低（但金冠银行账房被杀案增加了不确定性） |
| 国王驾崩 | 路德维希三世去世 | 加冕礼支出 + 政治不确定性 → 教会可能要求债务重新谈判 | 中高（国王68岁，病卧） |
| 双线作战 | 帝国+兽人同时进攻 | 财政上不可承受——6个月内国库耗尽 | 低（但兽人集结增加了可能性） |

## 七、财政大臣

**现任财政大臣**：奥尔多·金账房（男，57岁）。出身白垩城商人家庭——不是贵族。路德维希三世在20年前打破惯例任命了一个平民为财政大臣——因为上一个贵族财政大臣把国库当私人钱袋。

**性格（GM参考）**：务实主义者型（主导）。对政治没有兴趣——只对账本上的数字平衡有兴趣。他在财政大臣署办公桌的抽屉里放了一张纸条，上面写着王国可以削减的支出清单——从最可替代的到最不可替代的。第一项是"教会补贴"，最后一项是"银翼骑士团的战马草料"。

**与王太子的关系**：金账房是王太子少数几个真诚尊敬的官员之一。但金账房在政治上中立——他只为数字服务。如果税改失败，他会建议王太子削减军费——即使他知道这可能意味着帝国的入侵。

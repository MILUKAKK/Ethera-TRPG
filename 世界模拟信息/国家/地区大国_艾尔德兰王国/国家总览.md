## 历史概要

### 建国（纪元1190-1200年）

阿尔德里克一世——"银王"——通过十年战争+联姻+谈判统一了中央平原的十二个独立领主。击败的九个领主被保留为公爵——构成今日贵族议会的基础。

### 银龙盟约（纪元1200年）

建国之日，阿尔德里克一世在银带河畔与银龙"艾尔德兰"举行盟约仪式。银龙将名字赐予王国——并承诺此人的后裔永受龙的保护。王族血脉中据说自此流淌着微薄的龙血。

### 血河时代（纪元1602年至今）

维斯特法尔帝国的崛起打破了艾尔德兰的和平。三次血河战争（1602年、1652年、1755年）皆以帝国打赢战役、艾尔德兰守住血河天险告终。两国之间的仇恨已持续近两百年——深入骨髓。

## 当前四大困境

### 1. 财政-政治死锁

税制改革（人头税 → 人头税 + 土地税）是解开所有死锁的唯一钥匙——但它被贵族议会以约60%反对率卡死。王太子需要至少11票中的3票从"反对"或"摇摆"倒向"支持"。

三条可能路径（详见 `贵族与皇室.md` §三）：
- 东部突破：争取三位摇摆公爵
- 黑木妥协：私下谈判分化旧权派核心
- 外部事件：帝国入侵或国王驾崩——强制打破僵局

### 2. 军事两面受压

- **北方**：8,000骑兵 + 血河天险 + 12要塞 → 可防御
- **东方**：200守军 + 裂脊隘口 → 不可防御
- **南方**：银翼第5大队一部 + 零散哨站 → 军力覆盖薄弱

最可怕的场景——帝国渡河的同时兽人突破隘口。艾尔德兰的防御预案中没有同时应对两场战争的方案。

### 3. 人口-农业-征召的三角约束

征召8,000农民 → 农业减产15-20% → 如果征召发生在果月（秋收季）→ 可能导致局部饥荒 → 饥荒引发社会动荡 → 动荡削弱前线士气 → 恶性循环。

灰石城的征召令恰好在果月发出——**这是结构性愚蠢。** 南部领主（灰石公爵本人就是旧权派）选择在秋收季征召是因为这是"惯例时间"——而惯例比逻辑更有力量。

### 4. 继承焦虑

阿尔德里克家族六百年来父子相传从未中断。王太子艾德温（24岁）至今未婚。如果他无子嗣就去世——直系血脉终结。旁系继承将引发至少三个公爵家族的竞争——任何一方胜出都可能触发继承战争。

旧权派在私下以隐晦的方式提及"分支继承"——这是对王太子的慢性政治毒药。他必须在政治僵局中分出精力解决婚姻问题——但他无法同时处理税改、军事、和婚姻三项事务。

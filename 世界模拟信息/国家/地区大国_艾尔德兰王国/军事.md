## 一、军事指挥体系

### 1.1 最高指挥

```
                       国王（路德维希三世）
                         │ 病卧——实际权力由摄政王太子艾德温代理
                         │
              ┌──────────┼──────────┐
              │          │          │
         王国元帅      财政大臣    圣光教会
       （战时任命）  （军费审批）  （教会武装独立）
              │
    ┌─────────┼─────────┐
    │         │         │
 银翼骑士团 圣盾骑士团 征召兵团
   团长      团长     （各领地领主）
```

**关键制约**：艾尔德兰是封建君主制，不是中央集权帝国。国王不能单方面宣战或调动全部军队——需要贵族议会批准额外军费（见 `经济.md` 税改之争）。这意味着**战时指挥链在政治上存在瓶颈**。

### 1.2 指挥人员来源

| 层级 | 来源 | 选拔方式 |
|------|------|---------|
| 王国元帅 | 十二公爵中选任 | 国王任命，议会确认 |
| 骑士团团长 | 资深骑士晋升 | 战功+王室信任 |
| 大队长 | 骑士阶层 | 团长推荐 |
| 中队长 | 骑士或资深军士 | 大队长任命 |
| 小队长 | 军士或老兵 | 中队长挑选 |

**当前空缺**：王国元帅一位已虚悬三年。路德维希三世病倒前未能任命新元帅——现在议会分裂导致无法就人选达成一致。战时总指挥目前缺失，是最致命的弱点之一。

### 1.3 指挥半径

| 层级 | 指挥半径 | 通讯方式 |
|------|---------|---------|
| 元帅 → 团长 | 全境 | 传令骑兵 + 信鸽 |
| 团长 → 大队长 | 同战区（~100km） | 传令骑兵 |
| 大队长 → 中队 | 同战场（~10km） | 传令兵 + 旗帜信号 |
| 中小队长 → 士兵 | 同阵线（可见范围） | 号角 + 旗帜 + 喊话 |

**无远距即时通讯**：从白垩城发出一条军令到血河南岸最远的要塞，最快需要一整天（传令骑兵300km需约3天——不，白垩城到血河边境约500km，传令骑兵在官道上约80km/日，需要6-7天）。这意味着边境指挥官在大多数时间里是**自主决策**。

## 二、军事编制体系

### 2.1 骑士团编制（常备军）

以银翼骑士团为样本：

```
银翼骑士团（5,000骑）
│  团长 ×1（卡斯帕·银翼爵士）
│
├── 第1大队（1,000骑）
│   │  大队长 ×1
│   ├── 第1中队（200骑） 中队长 ×1
│   │   ├── 第1小队（40骑） 小队长 ×1
│   │   ├── 第2小队（40骑）
│   │   ├── 第3小队（40骑）
│   │   ├── 第4小队（40骑）
│   │   └── 第5小队（40骑）
│   ├── 第2中队（200骑）
│   ├── 第3中队（200骑）
│   ├── 第4中队（200骑）
│   └── 第5中队（200骑）
│
├── 第2大队 — 第3大队 — 第4大队 — 第5大队
│
└── 团直属：斥候队（50骑）+ 军医（5人）+ 军需官（3人）+ 铁匠/马医/文书（各2-3人）
```

| 编制单位 | 兵力 | 指挥官 | 典型任务 |
|---------|------|--------|---------|
| 小队 | 40骑 | 小队长（军士/老兵） | 巡逻、侦察、传令 |
| 中队 | 200骑（5小队） | 中队长（骑士） | 独立战术行动、据点驻防 |
| 大队 | 1,000骑（5中队） | 大队长（资深骑士） | 区域防御、大规模出击 |
| 骑士团 | 5,000骑（5大队） | 团长 | 战役级行动 |

### 2.2 征召步兵编制（战时）

```
征召兵团（约8,000人——未满编）
│  征召总指挥（通常由一位公爵代理）
│
├── 北境征召队（约3,000人） — 中央平原北部各领地提供
│   ├── 黑木公爵领 提供约600人
│   ├── 河卫城周边领地 提供约800人
│   └── [其余领地合计约1,600人]
│
├── 中央征召队（约2,500人） — 中央平原核心区
├── 南境征召队（约1,500人） — 包括灰石城领地
└── 东部征召队（约1,000人） — 东部丘陵各领地
```

**征召步兵没有固定编制**——每次动员时各领主按封地大小凑人，编制是临时拼凑的。同一支"中队"里的士兵可能互不相识。这是制约步兵战斗力的结构性缺陷。

### 2.3 圣盾骑士团编制

与银翼骑士团结构相同，但规模较小（3,000骑 = 3个大队）。唯一的区别是圣盾骑士团的**第一中队**同时兼任王室近卫——他们是全王国唯一长期驻扎在白垩城王宫内的武装力量。

### 2.4 战时合并编制：混合军团

战时两个骑士团 + 征召步兵合并为一个"混合军团"。历史上前三次血河战争均使用此编制：

| 军团 | 构成 | 总兵力 | 典型任务 |
|------|------|--------|---------|
| 北方军团 | 银翼骑士团主力 + 北境征召步兵 | 约7,000-8,000 | 血河正面防御 |
| 中央军团 | 圣盾骑士团 + 中央征召步兵 | 约5,000 | 预备队/首都防卫 |
| 南境守备 | 银翼第5大队 + 南境征召步兵 | 约2,500 | 南部边境守备 |

## 三、军费与财政

### 3.1 军费来源

| 来源 | 年度金额 | 占比 | 备注 |
|------|---------|------|------|
| 王国常规税收（人头税） | 约250万金王冠 | 83% | 由领主征收后上缴 |
| 王室领地产出 | 约30万金 | 10% | 王室直领地的农业和贸易收入 |
| 关税（边境贸易） | 约15万金 | 5% | 与洛桑/灰塔/铁冠的边境贸易税 |
| 教会捐赠（战时） | 约5万金 | 2% | 教会名义上的"自愿捐赠"——不可靠 |
| **合计** | **约300万金** | **100%** | |

> 数据来源：`经济.md` — 王国年度财政收入约850万金，军费约占35%（≈300万金）。

### 3.2 军费支出分解

| 支出项 | 年度金额 | 占比 | 说明 |
|--------|---------|------|------|
| 骑士团常备军饷 | 约120万金 | 40% | 骑士薪饷 + 战马饲养 + 装备维护 |
| 要塞/城墙维护 | 约60万金 | 20% | 12座血河要塞 + 白垩城城墙 + 各哨站 |
| 武器/装备储备 | 约40万金 | 13% | 铁锭采购 + 武器锻造 + 箭矢囤积 |
| 征召兵训练（两周集训） | 约15万金 | 5% | 食物+营地+教官补贴 |
| 后勤仓储（粮食/草料） | 约35万金 | 12% | 战略粮仓 + 草料储备 |
| 传令/情报系统 | 约10万金 | 3% | 传令骑兵 + 信鸽站 + 边境哨 |
| 军医/救治 | 约5万金 | 2% | 骑士团军医 + 战地医院储备 |
| 杂项/不可预见 | 约15万金 | 5% | |
| **合计** | **约300万金** | **100%** | |

### 3.3 战争额外开支（超出常规预算）

| 项目 | 增量 | 说明 |
|------|------|------|
| 全面动员征召兵（从8,000→12,000） | +20万金/月 | 食物 + 营地 + 运输 |
| 雇佣佣兵（如从洛桑雇佣） | +30-50万金/月 | 视佣兵规模和合同 |
| 战时弹药消耗（箭矢/修理） | +15万金/月 | 和平时期的5-10倍 |
| 难民安置/边境疏散 | +10万金/月 | 视战区平民规模 |
| **全面战争月均增量** | **约75-95万金/月** | — |
| **王国年收入仅850万金 → 月均约70万** | — | 战争将快速消耗国库 |

**结论**：艾尔德兰无法承担超过3-4个月的全面战争。长期战争必须依赖教会借贷 + 洛桑金冠银行贷款 + 加税（但加税需要议会同意——当前税改僵局下不可能）。

### 3.4 军费审批流程

```
国王/摄政王太子提出军事预算 → 贵族议会投票
                                  │
                    ┌─────────────┼─────────────┐
                    │ 通过         │ 否决         │ 僵持
                    ↓              ↓              ↓
              财政大臣拨款    预算不变        当前状态：
              （从未发生过）  （常态）      税改和军费都被搁置
```

**当前具体僵局**：王太子将税改与军费增加打包提交——从人头税改为土地税（贵族多缴）+ 军费从300万提升到400万。旧权派（黑木公爵为首）抵制——因为土地税直接触及贵族利益。议会投票：约60%反对、40%支持。

## 五、封建动员体系

### 5.1 领主义务

每位公爵/伯爵/男爵按其封地大小承担征召义务。封地越大，应提供的征召兵越多。这是六百年来的封建惯例——但缺乏精确的执行标准。

| 领主类型 | 封地规模 | 理论征召义务 | 实际提供 |
|---------|---------|------------|---------|
| 大公爵（如黑木公爵） | 约15万人口 | 约1,500人 | 约800-1,000人（常以各种理由短少） |
| 中等公爵 | 约8-10万人口 | 约800人 | 约400-600人 |
| 伯爵 | 约3-5万人口 | 约300人 | 约150-200人 |
| 男爵 | 约1万人口 | 约100人 | 约50-80人 |

**短少原因**：
- 领主不愿意交出健壮劳动力——他们需要人种地
- 提供的人多数是老弱或"不受欢迎的人"（懒汉、欠债者）
- 部分领主私下与征召军官达成默契——少交人、多交"替代金"

### 5.2 动员时间线

| 阶段 | 耗时 | 说明 |
|------|------|------|
| 国王签署动员令 | 1日 | 如果国王/摄政能签署——当前元帅空缺导致这一步就卡住 |
| 传令官将命令送达各领主 | 3-10日 | 视距离。最远的东部领主约10天收到 |
| 领主召集封地内兵员 | 5-15日 | 从各村各镇通知→集合→报到 |
| 各领主部队前往集结点 | 5-20日 | 步行。最远的可能需要一个月 |
| 编成部队 + 基础训练 | 10-20日 | 将互不相识的人编成一个战斗单位 |
| **从命令到战斗准备** | **约1-2个月** | |

**对比帝国**：维斯特法尔帝国从皇令下达到黑铁军团出动约**15天**——因为他们是常备军。

### 5.3 动员对经济的影响

征召8,000个成年男性离开田地 → 中央平原农业产出下降约15-20%（视季节）。如果动员发生在果月（秋收季节）——**后果是灾难性的**。这就是为什么灰石城的征召令选择在秋收刚开始时发出——这是一个愚蠢的时机，反映了上层对农业周期的漠视。

## 六、驻防与部署

### 6.1 和平时期部署

```
[北] 血河南岸 — 银翼骑士团 4个大队（4,000骑）+ 12座要塞守军
      │
      │ 距离约 300km
      │
[中] 白垩城 — 圣盾骑士团 3个大队（3,000骑）
      │
      │ 距离约 350km
      │
[南] 灰石城及周边 — 银翼骑士团 第5大队（1,000骑）+ 南方哨站
      │
      │ 距离约 80km
      │
[南南] 磨坊镇 — 零驻军。最近的哨站约半日路程。
```

### 6.2 战时部署预案（"血河方案"）

一旦侦察到帝国全面入侵的意图：
1. 银翼骑士团全部5个大队集中至血河南岸
2. 圣盾骑士团从白垩城北移至河卫城（距血河约80km）作为预备队
3. 征召令发出——动员北境和中央各领地的步兵向丰收城集结
4. 南境仅保留银翼第5大队的一半（500骑）维持秩序

**此方案的缺陷**：东部完全被掏空。如果兽人同时西进——裂脊隘口的200人会在半日内覆灭。

## 七、战争潜力评估

### 7.1 对维斯特法尔帝国

| 维度 | 艾尔德兰 | 维斯特法尔 | 判比 |
|------|---------|-----------|------|
| 常备陆军 | 8,000（骑兵） | 30,000（步兵主力） | 1:3.75 — 劣势 |
| 动员后总兵力 | ~16,500 | ~66,000 | 1:4 — 绝对劣势 |
| 魔法战力 | 500教会圣骑士（神术） | 500战斗法师 + 皇后海伦娜 | 神术 vs 奥术——不同类型 |
| 防御工事 | 血河天险 + 12要塞 | 需要渡河攻击 | 地利优势 |
| 后勤距离 | 在本土作战 | 补给线需要跨血河 | 补给优势 |
| 财政承受力 | 3-4个月全面战争 | 更长（帝国的奴隶经济不依赖税收） | 持久劣势 |
| 指挥 | 元帅空缺 | 皇太子弗里德里希直接指挥 | 指挥劣势 |

**结论**：艾尔德兰无法在正面战场上击败帝国。战略是纯粹的防御——利用血河天险、要塞链和银翼骑士团的机动力阻止帝国渡河。前三次血河战争证明了这一战略可行——每次帝国都能打赢战役但无法维持渡河后的补给线而撤退。

### 7.2 对格鲁姆汗国

| 维度 | 当前 | 风险 |
|------|------|------|
| 东部防御 | 裂脊隘口200人 | 兽人突破后，绿原至磨坊镇间无一兵一卒 |
| 动员能力 | 东部几乎没有可征召的兵力储备 | 东部丘陵人口仅80万——即使全部动员也只有几千步兵 |
| 地形 | 裂脊山脉是天然屏障 | 但隘口一旦失守，山脉对面是开阔地 |

**历史先例**：兽人最后一次大规模西进是约340年前（纪元1420-1430年，统一汗时期）。那一次兽人突破了裂脊隘口，一直打到了银带河。艾尔德兰当时的应对是放弃东部领土、在银带河西岸布防。如果乌尔格的特使与帝国达成联盟——南北夹击将使艾尔德兰在六个月内灭亡。

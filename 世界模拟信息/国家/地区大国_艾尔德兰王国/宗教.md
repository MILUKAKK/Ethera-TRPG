## 一、教会在王国的地位

### 1.1 不是国教——胜似国教

王国法典中没有"国教"二字。但在现实中——没有教皇主持的加冕礼，国王在理论上不"合法"。六百年来教会从未使用过否决权——正是这种"不用的权力"最有威慑力。

| 事实 | 含义 |
|------|------|
| 国王加冕需教皇亲自主持 | 教皇对王位继承有隐性否决权 |
| 王室婚礼/葬礼须经教会见证 | 教会掌控王族生命周期 |
| 教会拥有全国约15%可耕地 | 教会是王国最大的单一地主——超过任何一个公爵 |
| 教皇驻跸白垩城圣光大教堂 | 艾尔德兰是大陆的宗教中心——软实力巨大 |
| 教会拥有独立武装500圣骑士 | 这支武装不归王国指挥体系 |
| 教会向国王放贷（100万金债权） | 教会是王国最大的债权人——金融杠杆 |
| 教会主持基层教育（修道院学校） | 94%的识字者是教会教育的产物 |

### 1.2 组织结构（在艾尔德兰境内）

| 层级 | 称谓 | 人数 | 驻地 | 职能 |
|------|------|------|------|------|
| 教皇 | 圣父 | 1人 | 白垩城圣光大教堂 | 全体信众的最高领袖 |
| 枢机主教 | 您的圣光 | 3人（12人中3人驻艾尔德兰） | 白垩城 | 教皇顾问团 |
| 大主教 | 殿下 | 1人（卡勒斯） | 白垩城 | 王国最高神职者 |
| 主教 | 大人 | 约10人 | 各大城市（白垩城、丰收城、灰石城等） | 教区主管 |
| 司铎 | 神父 | 约200人 | 城镇教堂 | 主持弥撒、洗礼、婚礼、葬礼 |
| 助祭 | 修士/修女 | 约800人 | 村镇小堂（如磨坊镇小堂） | 辅助司铎 |
| 平信徒 | — | ~370万人 | 全国 | 普通信众 |

**基层现实**：村镇司铎和助祭多半是真正的信仰者。他们自己的收入极低——和农民差不多。他们被上级的腐败所困扰但无力阻止。磨坊镇的教会小堂由一位年迈的助祭负责（镇01槽位——尚未正式出场）。

## 二、大主教卡勒斯——教会的政治面孔

**基本信息**：男，61岁。驻白垩城。担任大主教已18年。

**性格（GM参考）**：操纵者型（主导）+ 务实主义者型（辅）。卡勒斯不是一个恶人——他是一个精于计算的权力玩家。他对神学的兴趣不如对政治的大。

**政治立场**：倾向于旧权派贵族。不是因为他反对税改的原则——而是因为旧权派保护教会的既得利益（土地免税、独立武装、借贷垄断）。税改一旦通过——下一步可能就是向教会土地征税。

**他与黑木公爵的关系**：长期私交。不是密友——而是权力伙伴。黑木在议会挡下任何触及教会利益的提案；卡勒斯则确保教士们不在布道中批评黑木。

**他与王太子的关系**：表面尊重、私下冰冷。卡勒斯在王太子摄政的第一年表示过"友好中立"——但在税改提案出台后迅速倒向旧权派。王太子私下对母亲说过："大主教是我唯一无法用数学算计的对手——因为他相信的不是数字。"

**教会内部的微妙分歧**：不是所有高级神职者都同意卡勒斯的政治立场。少数主教（尤其是南境和东部教区的）认为教会应该保持真正的中立。但他们不敢公开反对卡勒斯——他在教会内部的权威远强于在政治上的。

## 三、教会的经济力量

### 3.1 土地资产

教会拥有全国约15%可耕地——分散在各教区。这些土地：
- **不向王国纳税**——这是建国时阿尔德里克一世与教会的契约
- 由佃农耕种——教会收取地租（通常为收获的30-40%，低于贵族地主的40-50%）
- 地租收入约120万金/年——其中约70%用于各地教会运营，约30%上缴白垩城圣光大教堂

**如果王太子试图向教会土地征税**：教会可以直接拒绝——这有六百年的先例。王太子最可能做的不是征税，而是在税改案中明确"教会土地暂不在土地税征收范围内"以换取教会的中立——这是他尚未打出的牌。

### 3.2 金融角色

教会是王国最大的放贷机构之一（另一个是金冠银行）。教会放贷的优势：
- **信用并非基于抵押品**——而是基于"被逐出教会的威胁"。国王不还款 → 教皇可以宣布绝罚 → 国王在理论上失去合法统治权
- 金冠银行的贷款需要抵押关税收入；教会的贷款抵押是"王室的正统性"
- 王国欠金冠银行100万金——欠教会也是约100万金。前者会要求到期还款；后者可以无穷无尽展期——只要政治关系良好。

### 3.3 赎罪券经济

圣光教会出售"赎罪券"——声称购买者可获得索拉的宽恕。这在神学上存在争议（部分基层司铎认为这是亵渎），但在经济上是教会的重要财源：
- 年收入约15-20万金——不向任何世俗政权申报
- 主要购买者：商人（为商业上的欺骗行为赎罪）、临终贵族（为一生积累的罪孽赎罪）、有负罪感的各阶层

**王太子曾私下提议废除赎罪券**——这是一个同时打击教会收入和旧权派合法性的双重武器。但伊莎贝拉王后劝阻了他："我们已经在同时对抗议会和帝国。不要再加上教会。"

## 四、教会武装——圣骑士

### 4.1 规模与分布

约500名圣骑士分布于全国各教会据点。白垩城圣光大教堂驻扎约100人（精锐中的精锐）。其余分散在各大城市的教堂。

### 4.2 指挥权

- 圣骑士**不归王国指挥体系**——他们向地方主教汇报，最终向教皇
- 在理论上：王国面临外敌时教会武装应协同作战
- 在实际上：取决于大主教的政治立场。卡勒斯可以——"教会武装仅在教皇明确授权后方可投入战斗"——这在紧急时刻是致命的延迟

### 4.3 军事价值

圣骑士是少数能使用神术的战斗力量（详见 `军事.md` §单兵模板）。面对维斯特法尔的奥术战团时——神术的驱散能力是王国为数不多的反魔法手段。如果卡勒斯在战争时扣留圣骑士——银翼骑士团在面对帝国战斗法师时将毫无防护。

## 五、民间信仰与日常宗教生活

### 5.1 圣光信仰在乡村

在磨坊镇这样的乡村——圣光信仰与民间信仰共存。农民们去教堂做弥撒——然后在回家的路上在田边倒一杯麦酒给"大地老人"。基层司铎对此容忍——因为他们自己的母亲也在做同样的事。

### 5.2 教会在日常生活中的角色

| 功能 | 说明 |
|------|------|
| 每周弥撒（日曜日） | 全镇聚集——这是社交+宗教合一的活动 |
| 生命仪式 | 出生洗礼、成年礼、婚礼、临终告解——全部由教会主持 |
| 民事记录 | 教会是唯一保存出生/婚姻/死亡记录的机构——无教会则无"法律上的存在" |
| 基础教育 | 极少数男孩在修道院学校学习读写的经文 |
| 慈善 | 教会分发食物给赤贫者——尤其在冬季 |
| 医疗 | 司铎和修女提供基础的草药治疗 + 神术治疗（视虔诚度） |

### 5.3 磨坊镇的宗教生活

磨坊镇的教会小堂是一座石头房子——全镇最小的建筑。年迈的助祭主持每周一次简短的弥撒。镇上不少主妇每星期日曜日都去——不是因为虔诚，而是因为"大家都去"。

## 六、教会对王国政治的五个杠杆

| 杠杆 | 机制 | 当前状态 |
|------|------|---------|
| ① 加冕权 | 国王合法性的唯一来源 | 未使用——但存在 |
| ② 国债 | 100万金债权——可展期可催缴 | 展期中——卡勒斯以此为筹码 |
| ③ 圣骑士 | 500独立武装——战时可扣留 | 和平——尚未测试 |
| ④ 布道台 | 教士可以在每个日曜日向全国信众传达政治信息 | 当前中立——卡勒斯在等 |
| ⑤ 基层服务 | 教会是唯一提供出生/死亡/教育/慈善的机构——废除教会意味着基层服务的真空 | 不可替代 |

**结论**：教会比任何一个公爵都更有权力——但教会的权力是防御性的（阻止改变）而非进攻性的（推动改变）。卡勒斯不需要税改通过——他需要一切保持不变。

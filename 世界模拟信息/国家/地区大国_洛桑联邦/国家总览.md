# 洛桑联邦 · 国家总览

## 基础数据

| 项目 | 内容 |
|------|------|
| 正式名称 | 洛桑七城联邦（Lothan Confederation of Seven Cities） |
| 国家等级 | **地区性大国**（海上霸权 + 经济霸权） |
| 首都 | 无正式首都——联邦议会在翡翠港召开 |
| 人口 | 约300万（人类85%，精灵混血5%，兽人/半兽人8%，其他2%） |
| 领土 | 约900,000 km²（含沿海岛屿） |
| 建立 | 纪元1500年（约282年前） |
| 政体 | 商业寡头共和制 |
| 实际权力 | 六大商会控制七城议会 |

## 七城邦

1. **翡翠港**（最大港口，18万人）——议会所在地，金冠银行总部
2. **金币城**（12万人）——银行业中心，金冠银行金库所在地
3. **铁锚城**（10万人）——造船业，青帆舰队母港
4. **珍珠城**（7万人）——奢侈品贸易，精灵边缘贸易的垄断者
5. **盐风城**（5万人）——盐矿与腌制品
6. **黑曜港**（8万人）——奴隶贸易中心，大陆最大奴隶市场
7. **自由港**（6万人）——销赃、走私、情报交易、雇佣兵公会总部

## 当前困境

### 1. 金冠银行信用危机
账房被杀案引发了对银行储备金的质疑。若挤兑蔓延——银行的纸牌屋可能倒塌。

### 2. 城邦间离心力
各城邦暗中抵制翡翠港独大，但缺乏统一行动。六大商会在彼此竞争的同时还要维持联邦表面团结。

### 3. 奴隶贸易的道德压力
西风共和国（梵德兰的"影子洛桑"）与部分洛桑内部声音呼吁废止奴隶贸易。但黑曜港的奴隶商人——联邦最富有的利益集团之一——不会允许。

# 格鲁姆汗国 · 国家总览

## 基础数据

| 项目 | 内容 |
|------|------|
| 正式名称 | 格鲁姆肯（Grum-ken，兽人语："自由的人民"） |
| 国家等级 | **地区性大国**（骑兵数量大陆第一） |
| 首都 | 无固定都城（可汗帐随可汗迁徙。圣地：祖灵之丘） |

| 政体 | 部落联盟制，可汗通过大比武产生 |
| 现任可汗 | 铁蹄·乌尔格（男，40岁，狼首兽人） |
| 大萨满 | 「灵魂行者」卓尔金（实际年龄不明——据说已逾百岁） |

## 当前困境

### 1. 统一脆弱
乌尔格是数十年来最强可汗——但部族裂隙随时可能重新敞开。血牙部族的虎首兽人「红牙」戈拉克在大比武中败北后仍保持沉默——但这沉默随时可能转为新的挑战。

### 2. 装备落后
80,000骑兵听起来很强——但其中只有约30%拥有铁制武器。其余仍在使用骨箭、石斧。与维斯特法尔或艾尔德兰的精锐部队正面硬碰——兽人的伤亡率将是灾难性的。

### 3. 南方的诱惑与危险
裂脊山脉南麓的艾尔德兰领土在军事上极度薄弱。如果乌尔格选择西进——磨坊镇和整个艾尔德兰南部将毫无防御。但一旦越过裂脊隘口而无法迅速取得决定性胜利——兽人将陷入补给噩梦，且面对艾尔德兰和维斯特法尔可能的同时夹击。

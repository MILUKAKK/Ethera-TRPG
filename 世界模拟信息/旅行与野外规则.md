# 旅行与野外规则

> 此文件为GM提供标准化的旅行与野外生存判定规则。

---

## 一、移动速度

### 1.1 每日行进距离

| 方式 | 正常地形 | 困难地形（山地/森林/沼泽） | 道路（官道/石板路） |
|------|---------|--------------------------|-------------------|
| 步行 | 30 km/日 | 15 km/日 | 40 km/日 |
| 轻装步行（负重<15kg） | 40 km/日 | 20 km/日 | 50 km/日 |
| 马车 | 50 km/日 | 不可通行 | 70 km/日 |
| 骑行（普通马） | 60 km/日 | 25 km/日 | 80 km/日 |
| 骑行（战马） | 50 km/日 | 20 km/日 | 65 km/日（战马耐力较差） |
| 船（顺流） | 80 km/日 | — | — |
| 船（逆流/拉纤） | 20 km/日 | — | — |

**强制行军**：超出上述距离50% → 全队CON判定 DC 12，失败则获得1级疲劳。连续两天强制行军 → DC 16。

### 1.2 磨坊镇→常见目的地的路程

| 目的地 | 距离 | 方式 | 耗时 |
|--------|------|------|------|
| 磨坊溪上游（石滩） | 2 km | 步行 | 半个时辰 |
| 南方官道第一个歇脚点（歪脖子松树） | 12 km | 步行 | 不到半日 |
| 灰石城 | 80 km | 马车 | 2天1夜 |
| 灰石城 | 80 km | 步行 | 3-4天 |
| 裂脊隘口 | 120 km | 步行/山路 | 7-10天 |
| 白垩城 | 350 km | 马车 | 8-10天 |

### 1.3 影响移动速度的因素

| 因素 | 修正 |
|------|------|
| 大雨/泥泞 | -40% |
| 雪（<30cm） | -30% |
| 雪（>30cm） | -60% |
| 浓雾 | -20% |
| 酷热（烈月正午） | -30% |
| 负重超过上限 | 每超出10% → -10%，叠加 |
| 队伍中有伤者 | 取全队最慢者的速度 |
| 熟悉地形的向导 | +20%（如本地猎人知道所有山间小道） |

## 二、物资消耗

### 2.1 每日消耗

| 物资 | 每人每日 | 备注 |
|------|---------|------|
| 食物 | 1日份干粮 | 约1kg。不足则次日CON判定 -2 |
| 水 | 2升（约2水囊） | 炎热天气×1.5。缺水1日 → CON DC 12，失败则疲劳；缺水2日 → DC 16，失败则HP -10；缺水3日 → DC 20，失败则濒死 |
| 睡眠 | 至少4时 | 少于4时 → 次日全判定 -10%。连续2日缺觉 → 疲劳 |

### 2.2 携行能力

| 负重上限 | = STR × 15 kg（主角当前 60 kg） |
|---------|----------------------------------|
| 1日份干粮 | 约1 kg |
| 装满的水囊 | 约2 kg |
| 睡袋+帐篷 | 约8 kg |
| 备用衣物 | 约2 kg |
| 武器+防具 | 视装备 |

## 三、露营与休息

### 3.1 夜间露营

| 条件 | 效果 |
|------|------|
| 露天（无帐篷） | HP/SP恢复 ×0.5；天气不好时CON DC 10防着凉 |
| 帐篷 | 正常恢复 |
| 篝火 | 驱赶小型野生动物；精神恢复+10% |
| 无篝火 | 夜间可能遭遇野兽（GM掷中等事件） |
| 轮流守夜 | 降低遭遇概率 |
| 无人守夜 | 遭遇概率翻倍 |

### 3.2 休息恢复

| 方式 | 效果 |
|------|------|
| 短暂休息（1时） | 恢复 CON×3 SP |
| 夜间睡眠（4时以上） | 恢复全部SP；恢复 CON×5 HP |
| 全天休息（不移动） | 恢复 CON×10 HP；恢复全部SP；恢复 INT×3 MP |
| 劣质睡眠 | 恢复减半 |
| 旅馆舒适房间 | 恢复 ×1.5 |

## 四、寻路与迷路

### 4.1 保持方向

| 条件 | WIS DC | 说明 |
|------|--------|------|
| 走在清晰的道路上 | 自动成功 | 官道、石板路、大路 |
| 走在模糊的小路上 | 8 | 荒废小路、兽道 |
| 没有路（平原/草原） | 10 | 白天 |
| 没有路（森林/山地） | 14 | 白天 |
| 夜间（任何地形） | +4 DC | 无灯/无月 |
| 浓雾/暴雨 | +6 DC | 能见度极低 |
| 有地图 | DC -3 | 仅限该区域有明确标注的地图 |
| 有当地向导 | 自动成功 | 如常走该路的本地商人带队 |

### 4.2 迷路后果

WIS判定失败 → 偏离正确方向。2-6时后可能发现迷失（再掷WIS DC 10）。一旦确认迷路：
- 浪费半日到一日行程
- 可能遭遇野外随机事件
- 连续迷路 → CON DC 12 防止焦虑/疲劳

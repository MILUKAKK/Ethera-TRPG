# 冶金与工艺

> 科技基准：公元100年前后水平

---

## 一、金属冶炼

### 1.1 已知金属

| 金属 | 用途 | 冶炼难度 |
|------|------|---------|
| 铁 | 工具、武器、日用 | 中等——鼓风炉+锻打 |
| 钢（低碳钢） | 优质武器/工具 | 较难——渗碳+淬火+回火 |
| 铜 | 货币、铜器 | 容易 |
| 锡 | 合金原料 | 容易 |
| 青铜（铜锡合金） | 雕像、装饰、部分工具 | 容易 |
| 银 | 货币、饰品 | 较高（需开采含银矿） |
| 金 | 货币、奢侈品 | 淘金/开采 |

### 1.2 冶炼技术

- **鼓风炉**：水力或人力鼓风，可达炼铁所需温度（~1200°C）
- **锻打**：铁块反复加热锻打以去除杂质
- **渗碳**：将铁件在炭火中长时间加热使碳渗入表面——制成钢
- **淬火**：烧红后浸入水中/油中硬化
- **回火**：淬火后再次微热以降低脆性
- **铁矿来源**：铁峰山脉、裂脊山脉为主要产区。艾尔德兰中央平原无铁矿

### 1.3 未出现的技术

- 高炉炼铁（液态生铁）——不存在
- 坩埚钢——不存在
- 水力锻锤——不存在（所有锻打依赖人力）
- 轧钢——不存在

## 二、手工艺

### 2.1 锻造

- 铁匠手工打造一切铁制品——从农具到武器到马蹄铁
- 风箱：皮囊风箱，手拉或脚踏
- 铁砧+锤子：铁匠的必备工具
- 部分铁匠水准远超村镇级别——可能因曾在骑士团或军队受训

### 2.2 陶器与玻璃

- **陶器**：陶轮成型，窑炉烧制。无釉陶为主，少量铅釉
- **玻璃**：吹制玻璃存在——但产量低，价格高，为奢侈品

### 2.3 纺织

- 手纺车（单锭）、手织机
- 羊毛/亚麻为主要纺织原料
- 丝绸：从东陆进口——价格极其昂贵

### 2.4 皮革加工

- 植鞣法：用树皮/植物汁液鞣制
- 产品：皮甲、皮带、皮靴、马具、水囊

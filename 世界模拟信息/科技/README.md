# 科技水平设定

> 此文件夹记录埃塞拉大陆的文明科技水平。
> 
> **基准时代**：约相当于地球公元100年前后（古典时代晚期至中世纪早期之间）。

---

## 总原则

埃塞拉大陆的凡人科技水平以古典时代的罗马-汉王朝为参考基准。以下为各领域概览：

| 领域 | 水平概况 | 文件 |
|------|---------|------|
| 农业与畜牧业 | 铁制农具、水车磨坊、轮作制 | `农业与畜牧业.md` |
| 建筑与工程 | 石砌建筑、拱券结构、城市供水 | `建筑与工程.md` |
| 交通与航运 | 帆桨船、马车、石板路 | `交通与航运.md` |
| 冶金与工艺 | 铁器普及、青铜仍在用、手工锻造 | `冶金与工艺.md` |
| 医学与药学 | 草药学、体液学说、外科简陋 | `医学与药学.md` |
| 书写与知识 | 羊皮纸、手抄本、知识贵族垄断 | `书写与知识.md` |
| 日常生活 | 油灯蜡烛、陶器木器、公共卫生原始 | `日常生活.md` |
| 军事科技 | 铁剑/矛/弓、攻城器械、骑兵无马镫 | `军事科技.md` |

---

## 核心限定

> ⚠ **强制约束文件**：`GM判定规则/科技限制清单.md` — 为LLM GM提供了完整的禁止物品清单、NPC语言限制和快速判定指南。本文件夹描述"有什么"，限制清单规定"不能有什么"。两者互补，限制清单具有更高约束力。

1. **无火药**：没有任何形式的爆炸性火器
2. **无蒸汽/机械动力**：一切动力来源为人力、畜力、水力、风力
3. **无电力**：不存在电的任何形式的认知或利用
4. **无印刷术**：书籍全为手抄本，知识传播极慢
5. **无精密机械**：没有发条、齿轮系等精密机械装置
6. **无现代医学**：不了解细菌、解剖学原始、外科手段有限
7. **无现代航海技术**：没有六分仪、没有罗盘（近岸航行为主）
8. **无远距离即时通讯**：没有电报及任何远距离即时通讯手段

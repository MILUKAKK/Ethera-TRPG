# 日常生活

> 科技基准：公元100年前后水平

---

## 一、照明

| 方式 | 说明 | 使用阶层 |
|------|------|---------|
| 蜡烛（牛脂） | 冒烟、有异味 | 平民 |
| 蜡烛（蜂蜡） | 无烟、清香 | 贵族/教堂 |
| 油灯 | 菜籽油/动物油，灯芯燃烧 | 各阶层 |
| 火把 | 室外/城堡走廊 | 各阶层 |
| 炉火 | 照明+取暖 | 所有人 |
| 油灯 | 广泛使用 | 各阶层 |

---

## 二、燃料

| 类型 | 用途 | 说明 |
|------|------|------|
| 木柴 | 取暖+炊事 | 最主要燃料 |
| 木炭 | 取暖+冶金 | 热值高、无烟 |
| 泥炭 | 缺乏木材的地区 | 北部苔原 |
| 畜粪（晒干） | 燃料匮乏时 | 草原地区 |

---

## 三、餐具与炊具

- **陶器**：碗、罐、壶——日常餐具
- **木器**：木碗、木勺、木砧板
- **铁器**：铁锅、铁叉（烤用）
- **铜器**：铜壶、铜盆——富裕家庭
- **银器**：贵族餐桌
- **刀具**：每人自备小刀用餐

- **男性**：亚麻衬衫 + 粗麻长裤 + 羊毛外衣 + 皮腰带 + 布靴
- **女性**：亚麻长裙 + 围裙 + 头巾 + 羊毛披肩
- **主角**：粗布衣一套（日常便服）

## 五、洗浴与卫生

- **平民**：极少洗澡（每周一次已算勤快）。用湿布擦身为主
- **富裕家庭**：木桶热水澡
- **公共浴室**：大城市有（类似罗马公共浴场），但磨坊镇没有
- **厕所**：户外茅坑。夜壶。无抽水马桶
- **除臭**：香草包、花瓣——掩味而非清洁

# 军事科技

> 科技基准：公元100年前后水平
> 

---

## 一、步兵装备

### 1.1 武器

| 武器 | 使用者 | 说明 |
|------|--------|------|
| 铁剑（长剑/短剑） | 骑士/正规军 | 铁或低碳钢制 |
| 长矛/长枪 | 步兵阵列 | 反骑兵核心装备 |
| 标枪/投矛 | 轻步兵 | 投掷后近战 |
| 战斧 | 重步兵/北方蛮族 | 破甲 |
| 钉头锤 | 重步兵 | 对链甲有效 |
| 弓（短弓/猎弓） | 轻步兵/民兵 | 非主力兵种 |

### 1.2 没有的武器

- 长弓（射程/穿透力不足——尚未发展出英格兰式长弓）
- 弩（除极少数希腊式腹弩外，未普及）
- 火药武器——完全不存在

### 1.3 防具

| 防具 | 使用者 | 说明 |
|------|--------|------|
| 皮甲 | 轻步兵/民兵 | 最普及 |
| 链甲 | 正规军/骑士 | 铁环编织——优秀防御 |
| 鳞甲 | 军官/精锐 | 金属片缝制在皮革上 |
| 板甲（分段式） | 极少数精锐 | 仅为胸甲+护肩——**无全身板甲** |
| 盾牌（木/铁） | 各兵种 | 圆盾/鸢盾/塔盾 |

## 二、骑兵

### 2.1 限制

- **无马镫**。骑手靠双腿夹紧马腹保持稳定
- 冲击力有限：受限于无马镫，无法进行中世纪式全速冲锋（骑士会因反冲力落马）
- 骑兵的主要战术：轻骑侦察、侧翼骚扰、追击溃敌

### 2.2 骑兵类型

| 类型 | 装备 | 战术 |
|------|------|------|
| 轻骑兵 | 皮甲+短弓/标枪 | 侦察、骚扰、追逃 |
| 中骑兵 | 链甲+长矛+剑 | 冲击步兵侧翼 |
| 重骑兵 | 鳞甲/半胸甲+长矛 | 艾尔德兰银翼骑士团——近卫级 |

## 四、军事组织

### 4.1 军队类型

- **常备军**：少数国家拥有（艾尔德兰骑士团、帝国黑铁军团）
- **征召兵**：战时由领主征召农民——装备差、训练少
- **雇佣兵**：洛桑联邦的主要陆军——付钱即战
- **民兵**：村镇自卫队——平时务农，战时保卫家乡

### 4.2 通信

- **传令兵**：骑马信使——最重要的军事情报传递方式
- **烽火台**：边境预警系统——传递"入侵"信号
- **信鸽**：少部分贵族使用（训练周期长）
- **无远程即时通讯**：一场战役的消息可能需要数日才能传到首都

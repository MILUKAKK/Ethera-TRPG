# 医学与药学

> 科技基准：公元100年前后水平

---

## 一、医学认知

### 1.1 基础理论

- **体液学说**主导：人体由四种体液（血液、黏液、黄胆汁、黑胆汁）组成，健康取决于四者的平衡
- 不了解微生物或病菌的存在
- 不了解血液循环（但知道心脏跳动和流血会死）
- 疾病被归因于：体液失衡、瘴气、神灵惩罚、诅咒

### 1.2 诊断手段

- 望诊：观察面色、舌苔、尿液颜色
- 问诊：询问症状、饮食、排便
- 切脉：少数医生会摸脉——但不精确
- **无体温计、无听诊器、无任何诊断仪器**

---

## 二、治疗手段

### 2.1 内科

| 方法 | 说明 |
|------|------|
| 放血 | 最常见的"万能疗法"——认为放出"过多的坏血" |
| 催吐/泻药 | 排出体内"坏体液" |
| 草药煎服 | 经验积累的植物疗法 |
| 休息+营养 | 对自愈性疾病的实际有效手段 |
| 祈祷/宗教仪式 | 神术可能真的有效——但非科技范畴 |

### 2.2 外科

- **能做的手术**：清创、包扎、截肢（锯断）、放脓、拔牙
- **麻醉**：烈酒灌醉+草药（罂粟汁/曼德拉草根）——效果不稳定
- **消毒**：不了解。使用烈酒冲洗（经验发现有效但不理解原理）
- **术后存活率**：截肢手术约30-50%（主要死于感染失血）
- **无解剖学基础**：对人体内部结构的认知有限

### 2.3 常见药材

| 药材 | 用途 |
|------|------|
| 柳树皮 | 退烧止痛（含水杨苷） |
| 罂粟汁 | 止痛安眠 |
| 大蒜 | 消毒防感染（经验认知） |
| 蜂蜜 | 敷伤口——抗菌（经验认知） |
| 艾草 | 止血 |
| 薄荷 | 助消化 |

## 三、公共卫生

- **城镇排水**：露天明渠——污水与垃圾沿街排放
- **饮水**：井水——易污染。不了解煮沸杀菌
- **瘟疫应对**：隔离（粗暴手段）、焚烧死者物品、逃离疫区
- **鼠患**：普遍——不了解鼠类与疾病传播的关系
- **厕所**：茅坑/马桶——粪便集中收集用作农田肥料

## 四、接生与育儿

- 接生由产婆（经验丰富的年长女性）负责
- 婴儿死亡率极高——约30-40%活不过周岁
- 难产时优先保孩子——因此在贵族家庭中，失去妻子是常见的

# 随机快速NPC生成器

> 此文件用于快速生成非关键NPC——酒馆里的陌生人、路上的旅人、集市里的摊贩。
> 这些NPC不需要独立档案（动辄数千字）——但需要GM能一秒生成一个有名字、有来历、有当前状态的临时角色。
> 
> **生成后**：如果该NPC在剧中被使用了两次以上且展现出重要性——将其升级为正式NPC并创建独立档案。
> 
> 关联：`_优先级索引.md` — 已使用的临时NPC可转入3.1-3.8的待出场槽位

## 四、快速生成示例

GM从以上表格中选取或自由组合。一个完整的临时NPC约30秒生成：

```
姓名：格里姆·山脚（铁匠常用名+地点姓）
性别/年龄：男，19岁
职业：灰石城铁料铺学徒
当前状态：赶路——送一箱马蹄铁去丰收城，迷路了
当前位置：磨坊镇外的歪脖子松树下
说话方式：务实主义者——"这条路是去丰收城的吗？我走了半天怎么还没到。"
```

```
姓名：罗莎·河边（常见女名+地点姓）
性别/年龄：女，35岁
职业：南部某村的面包师
当前状态：休息——带着两个孩子和一篮面包，正往灰石城走。说"村里征召令下来了，我带娃去灰石城他舅舅家躲一躲"
当前位置：南方官道
说话方式：守护者型——"你也是去灰石城的？一起走吧。路上人多安全些。"
```

## 六、命名禁忌

| 避免 | 原因 |
|------|------|
| 与已有重要NPC重名 | 混淆 |
| 使用国王/教皇/公爵的名字 | 这些是专有的——平民不会取 |
| 使用听起来太现代的假名 | "ShadowBlade"——这不属于这个时代 |
| 使用另一个大陆的名字而不自知 | 在艾尔德兰出现一个炎州名——除非该NPC就是炎州来的（这本身就是叙事线索） |

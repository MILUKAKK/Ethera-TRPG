# 任务系统

> 此文件为GM生成玩家任务的框架。任务不是孤立的"支线"——它是NPC目标与玩家行动的交汇点。
> 关联：`NPC数据库/` — 任务的提议者 | `旅行与野外规则.md` — 任务的距离和风险 | `怪物与敌人.md` — 讨伐类任务的对手

---

## 一、任务是什么

任务是NPC的**短期目标遇到了障碍**——玩家是这个障碍的解决方案。

- 某位商人的货物滞留邻镇 → 他的短期目标是"按时交货" → 障碍：缺人手搬运 → 解决方案：玩家护送/搬运
- 铁匠的矿石用完了 → 他的短期目标是"补齐矿石库存" → 障碍：矿山供货延迟 → 解决方案：玩家去矿山查询
- 邻居家的羊跑丢了 → 虽然微不足道，但对邻居来说找回羊是她的短期目标——玩家可以帮也可以不帮

任务不是"NPC头顶上的黄色感叹号"——**是NPC在自然追求自己目标的过程中遇到了玩家可以帮忙解决的困难。** GM从NPC的短期目标出发，自然生成任务。

## 二、任务类型

| 类型 | 描述 | 典型委托者 | 典型目的地 | 危险等级 |
|------|------|----------|----------|---------|
| **送货** | 把物品从A送到B | 商人、农民、长辈 | 邻镇或邻城 | 低（路途风险为主） |
| **取货** | 从B取回某物带到A | 铁匠、药草师、酒馆老板 | 邻镇或邻城 | 低-中 |
| **护送** | 保护某人从A安全到达B | 旅人、难民、商人 | 同区域 | 中——路途+可能的遭遇 |
| **寻找** | 找到丢失的人或物 | 任何人 | 不定——在已知区域内 | 低-中（取决于寻找对象） |
| **调查** | 去某地查明某事 | 领主治安官、教会、酒馆老板 | 特定地点 | 中——可能遭遇未知危险 |
| **讨伐** | 消灭某处的威胁（野兽/匪徒） | 领主治安官、村民集体 | 镇外或更远 | 高——直接战斗 |
| **传递信息** | 把口信或信件送到某地某人 | 任何人 | 不定 | 低（路途风险为主） |
| **收集** | 收集一定数量的某种资源（草药/矿石/食材） | 药草师、铁匠、长辈 | 镇外或野外 | 低-中 |

## 三、任务生成流程

```
① GM识别NPC的短期目标受阻
       ↓
② 确定障碍的具体性质（缺物/缺人/缺信息/有威胁）
       ↓
③ 匹配任务类型
       ↓
④ 确定任务参数：目的地距离 / 危险等级 / 酬劳 / 期限 / 失败后果
       ↓
⑤ NPC以符合其性格的说话方式向玩家提出（或暗示——不是每个NPC都会直接说"帮帮我"）
       ↓
⑥ 生成任务选项：纳入玩家的六维选项模型。每个任务至少呈现2-3种处理方式
```

## 四、任务参数

### 4.1 距离与耗时

| 距离 | 耗时（步行） | 耗时（马车） | 示例目的地（从磨坊镇出发） |
|------|------------|------------|--------------------------|
| 镇内 | 不到半日 | — | 磨坊溪上游、橡树林 |
| 短途 | 半日-1日 | 半日 | 歪脖子松树歇脚点 |
| 中程 | 2-4日 | 1-2日 | 灰石城 |
| 长途 | 5-10日 | 3-5日 | 丰收城、绿林镇 |
| 远途 | 10日以上 | 5日以上 | 白垩城、河卫城 |

**主角当前的怯懦者性格**：前两类（镇内+短途）是他最可能接受的。中程及以上——需要强烈的推动力（近亲受伤/长辈要求/紧迫事态）。

### 4.2 危险等级

| 等级 | 说明 | 适合等级 | 可能遭遇 |
|------|------|---------|---------|
| 安全 | 官道、白天、无已知威胁 | Lv.1 | 随机旅途事件（TR）低概率 |
| 低危 | 乡村路、可能有小型野生动物 | Lv.1-2 | 单只草原狼、毒蛇 |
| 中危 | 穿越林区/山区、已知附近有匪徒活动 | Lv.2-4 | 狼群、逃兵、匪徒 |
| 高危 | 血河边境附近、战区 | Lv.4-6 | 武装匪徒、帝国巡逻队 |
| 极危 | 战场腹地 | Lv.6+ | 军队、精锐部队 |

### 4.3 酬劳参考

| 任务复杂度 | 铜币 | 银币 | 金币 | 附加奖励 |
|----------|------|------|------|---------|
| 镇内跑腿 | 5-20铜 | — | — | 一顿饭、好感度 |
| 短途简单 | 20-50铜 | 1-3银 | — | 少量物品 |
| 中程普通 | — | 3-10银 | — | 装备/药品 |
| 长途困难 | — | 10-50银 | 1-2金 | 高品质装备 |
| 极危 | — | — | 2-10金 | 稀有物品、声望 |

**注意**：酬劳需要与委托者的经济能力匹配。一位贫穷的老妇人不可能出10银雇人送一篮鸡蛋——她能出的是一顿热晚餐和几句真心的祝福。富商可以出几金雇护卫。GM根据委托者的经济阶层合理设酬。

### 4.4 期限

| 期限类型 | 说明 | 示例 |
|---------|------|------|
| 无期限 | "什么时候有空" | 长辈："下雪之前能做完就行" |
| 宽松 | "一周内" | 酒馆老板："这个月内帮我多进两桶麦酒" |
| 正常 | "三天内" | 药草师："病人需要这味药，三天内" |
| 紧迫 | "今天之内" | 小孩："太阳下山前找到，不然会被鸟吃掉" |
| 生死 | "现在" | 有人受伤、追捕、逃命 |

### 4.5 失败后果

| 严重度 | 后果 | 示例 |
|--------|------|------|
| 轻微 | 委托者失望、好感-2~5 | 没及时送东西——长辈唠叨了一整个晚饭 |
| 中等 | 委托者损失金钱或机会、好感-10~20 | 铁料没及时送到——铁匠失去了这批订单 |
| 严重 | 有人受伤或死亡、法律后果 | 没及时送药——病人死了。家属告到领主法庭 |
| 永久 | 错过唯一机会、NPC死亡、区域毁灭 | 错过了救援时机——那个人没能救回来 |

## 五、通用任务种子模板

> GM根据当前区域NPC的短期目标和障碍，从以下模板生成具体任务。每次游戏最多同时激活2-3个任务。

### 5.1 日常类

| 种子 | 类型 |
|------|------|
| 某位长辈需要某生活物资从邻镇运回 | 取货/送货 |
| 厨房/仓库出现鼠患或漏水——需要简单修缮 | 镇内杂务 |
| 长辈或幼辈需要换季衣物——需去邻镇布料铺采购 | 取货 |
| 某位手艺人需要特定食材/原料用于节庆制作 | 收集 |

### 5.2 工匠与商人类

| 种子 | 类型 |
|------|------|
| 铁料/矿石补给延迟——供货商没按期到货，需去矿山确认 | 调查 |
| 委托你去灰石城采购一批特定工具或原料 | 取货 |
| 新到一批货——需要帮手搬运整理 | 镇内 |
| 帮酒馆老板把一封信或包裹送到某人手中 | 传递信息 |

### 5.3 邻里与社区类

| 种子 | 类型 |
|------|------|
| 邻居家的家畜跑丢了——帮忙寻找 | 寻找 |
| 某人身体不适——镇上药草不够，需去灰石城采购 | 取货 |
| 某人的家人外出迟迟未归——需要去查看情况 | 调查 |

### 5.4 官方类

| 种子 | 类型 | 触发条件 |
|------|------|---------|
| 灰石城征召令传到磨坊镇——需要把被征召者的名单送到灰石城 | 送货 | 征召令传到 |
| 领主的税单出了错——需要有人去灰石城核对 | 送货 | 落叶月（税收季） |
| 南方官道某处发生了山体滑坡——需要人去确认路能不能过 | 调查 | 随机中等事件M21 |

## 六、任务在选项中的呈现

任务不是"你必须做"——是"有个NPC需要帮助"。在六维选项模型中，任务选项以 `[协助]` 标签呈现：

```
1: [回避] 假装没听见长辈的叹气，溜去河边。
2: [观察] 问问具体缺什么——但不承诺帮忙。
3: [协助] 主动提出去邻镇采购物资。来回约4天。
4: [冒风险] 去采购物资——顺便去打听一下附近矿山为什么停产。
```

选项数量：一个任务通常产生2-4个选项（回避/观察/协助/冒风险）。

# 梵德兰大陆 · 详细地图

> 位于埃塞拉以西，隔无尽海（最窄处约1,200km）。

## 大陆总览

**名称**：梵德兰（Vandland）——在古洛桑语中意为「西境之地」
**面积**：约 16,000,000 km²
**形状**：南北狭长——形似一个斜置的长方形
**纬度**：北纬 25° 至北纬 70°
**气候**：南部地中海式 → 中部温带海洋性 → 北部温带大陆性
**发现史**：纪元1430年由洛桑探险家梵德·海狼首次抵达，大陆以他的名字命名。

## 已知国家

1. **奥斯特王国**（中部平原）——面积最大的人类王国，封建制
2. **北境联合酋邦**（北部高地）——半游牧民族联盟
3. **翡翠群岛王国**（西海岸外海岛）——航海与渔业王国
4. **锻炉矮人王国**（中部山区）——大陆最优秀的铁匠与工程师
5. **西风共和国**（南部海岸）——与洛桑联邦相似的商业共和国

## 与埃塞拉的关系

梵德兰与埃塞拉之间由洛桑联邦维持定期航线。两地之间的贸易以奢侈品（丝绸、香料、宝石）为主。梵德兰诸国对埃塞拉的政治事务基本保持中立与不介入态度。

# 埃塞拉大陆 · 日期与时间规则

> 此文件定义埃塞拉世界独立的时间体系，与地球现实历法无关联。
> 所有时间参照以此文件为准。

---

## 一、时间体系总览

| 单位 | 进制 | 说明 |
|------|------|------|
| 秒（息） | — | 一次平稳呼吸的长度 |
| 分（刻） | 1刻 = 100息 | 约等于地球1分钟 |
| 时（时） | 1时 = 100刻 | 约等于地球1小时 |
| 日（日） | 1日 = 10时 | 白天5时 + 夜晚5时 |
| 周（曜）| 1曜 = 7日 | — |
| 月（月） | 1月 = 4曜 = 28日 | — |
| 年（年） | 1年 = 12月 = 336日 | — |

### 1.1 时间刻度说明

**关于"息"**：息是大陆通用的最小时间单位——"一次安静的深呼吸"。农夫不需要知道一刻有多少息，但沙漏工匠知道。在日常生活中，大多数平民的计时精度止于"刻"（一刻约等于一次短祷告的时间）。口头表述以粗粒度的时辰为主——"一顿饭的工夫""从村口走到磨坊的工夫"。

**10时制的一日**：一日分为10时，白天5时、夜晚5时。这种对称的10时制源于古代王国的计时传统，而非法令强制。

## 二、一日之内的时段划分

| 时段编号 | 名称 | 对应时间 | 说明 |
|---------|------|---------|------|
| 第1时 | 晨时 | 天亮 | 起床、晨祷 |
| 第2时 | 朝时 | 上午 | 劳作开始 |
| 第3时 | 午时 | 正午 | 午饭 + 休憩 |
| 第4时 | 夕时 | 下午 | 继续劳作 |
| 第5时 | 暮时 | 日暮 | 收工、晚餐 |
| 第6时 | 灯时 | 天黑 | 点灯、酒馆热闹 |
| 第7时 | 静时 | 夜渐深 | 街上人少 |
| 第8时 | 更时 | 深夜 | 大部分入睡 |

**通俗称呼**：平民很少说"第3时"——他们会说"日头当顶的时候""午饭后""鸡叫的时候"等。

## 三、曜日（周）系统

### 3.1 七曜

| 曜日 | 意义 | 民间俗称 | 习俗 |
|------|------|---------|------|
| 日曜日 | 太阳之日 | 礼拜日 | 圣光教会安息日——去教堂做弥撒 |
| 月曜日 | 月亮之日 | 开市日 | 集市开放的日子 |
| 火曜日 | 火焰之日 | 锻炉日 | 铁匠最忙的一天 |
| 水曜日 | 流水之日 | 洗衣日 | 妇女集中到溪边洗衣 |
| 木曜日 | 树木之日 | 田野日 | 农夫下地劳作 |
| 金曜日 | 黄金之日 | 结账日 | 商人结算、付酬 |
| 土曜日 | 大地之日 | 休整日 | 打扫院落、修理工具——准备下一曜 |

### 3.2 特殊曜日规则

- 日曜日：圣光教会的正式安息日。大部分店铺半日营业。教堂弥撒在晨时和暮时各一场
- 金曜日：传统上的付酬日。雇工在这一天领取上曜的工钱。酒馆在当晚最为拥挤
- 土曜日：民间称为"预备日"——家家户户为下一曜做准备

## 四、月份系统

### 4.1 十二月名

| 月序 | 月名 | 季节 | 气候特征 | 农事/大事 |
|------|------|------|---------|----------|
| 1月 | 冰月 | 深冬 | 最冷的月份 | 炉边蛰居、修整农具 |
| 2月 | 霜月 | 冬末 | 霜冻但仍寒冷 | 储备粮食盘点 |
| 3月 | 萌芽月 | 初春 | 冰雪消融，草木发芽 | 春耕开始 |
| 4月 | 花月 | 仲春 | 百花绽放 | 播种完成 |
| 5月 | 暖月 | 晚春 | 气温彻底回暖 | 田间管理 |
| 6月 | 夏月 | 初夏 | 日照最长 | 第一次除草 |
| 7月 | 烈月 | 盛夏 | 一年中最热的月份 | 晒谷、修屋顶 |
| 8月 | 收月 | 夏末 | 开始降温 | 早收作物开镰 |
| 9月 | 果月 | 初秋 | 果实成熟 | 秋收——最重要的月份 |
| 10月 | 秋月 | 仲秋 | 落叶纷飞 | 打谷、酿酒 |
| 11月 | 落叶月 | 深秋 | 树秃、风冷 | 储备过冬、腌制肉类 |
| 12月 | 雪月 | 初冬 | 第一场雪 | 休养生息 |

### 4.2 季节划分

| 季节 | 涵盖月份 | 天数 |
|------|---------|------|
| 春 | 萌芽月、花月、暖月 | 84日 |
| 夏 | 夏月、烈月、收月 | 84日 |
| 秋 | 果月、秋月、落叶月 | 84日 |
| 冬 | 雪月、冰月、霜月 | 84日 |

### 4.3 特殊月份规则

- **冰月**：北境（维斯特法尔以北）最低温度可达 -40°C。磨坊镇冬季通常 -5 至 -15°C
- **烈月**：10时制中的午时（第3时）在烈月会调整为"避暑歇息"——洛桑一带午后的劳作推迟一小时后开始
- **果月**：全大陆最重要的月份——秋收决定一年的口粮。征召令通常不在果月下达，因为没人有空打仗

### 4.4 月的分期

- 上旬：第1至第10日
- 中旬：第11至第20日
- 下旬：第21至第28日

## 五、纪元与年号

### 5.1 纪元系统

| 纪元 | 起始年 | 说明 |
|------|--------|------|
| 纪元元年 | 0年 | 艾尔德兰王国正式建国之年。白垩城奠基。此前的历史称为"纪元前" |
| 当前纪元 | 纪元1782年 | 建国至今已一千七百八十二年 |

**纪元的零点**：纪元元年1月1日（冰月·首日）被公认为王都白垩城奠基之日。此后艾尔德兰诸部逐渐统一，形成今日的王国版图。

### 5.2 年份的表达

正式写法：
```
纪元1782年 · 果月（9月）· 第一周 · 木曜日
```

在书信或日常对话中可简写：
```
果月四日，木曜
```
或：
```
九月初四，木曜
```

### 5.3 历代纪元划分

在长达1782年的纪元史中，学者们划分了几个大的时期：

| 时期 | 年代范围 | 特征 |
|------|---------|------|
| 王国初期 | 纪元1-500年 | 人类各国陆续形成。艾尔德兰诸部逐渐统一 |
| 扩张期 | 纪元501-1000年 | 王国版图扩大。兽人西征。洛桑联邦成立 |
| 血与火时代 | 纪元1001-1500年 | 血河边境冲突加剧。帝国崛起。铁冠王国建立 |
| 铁与血时代 | 纪元1501年至今 | 血河三战。帝国崛起。汗国统一 |

民间对时期的认知很模糊——大多数平民只说"爷爷的爷爷那辈儿"或"大战那几年"。

## 七、时间度量与计时工具

### 7.1 常用时间表述

| 口语表达 | 约合时长 | 使用场景 |
|---------|---------|---------|
| "一息的工夫" | 1息 | 极短的时间 |
| "一小会儿" | 5-15刻 | 日常模糊计时 |
| "一顿饭的工夫" | 20刻 | — |
| "从村头走到村尾" | 30刻 | 模糊距离兼时长 |
| "一炷香的工夫" | 约15刻 | 教堂/寺庙计时 |
| "半日" | 5时 | 半天 |
| "一个日夜" | 1日 | — |

### 7.2 计时工具

| 工具 | 精度 | 使用者 | 说明 |
|------|------|--------|------|
| 日晷 | 刻级 | 广泛 | 白天可用，阴天不行 |
| 水钟/漏壶 | 刻级 | 教堂/城堡 | 全天可用但需维护 |
| 沙漏 | 刻级 | 铁匠/炼金/厨房 | 无法连续计时——适用于特定长度的计时 |
| 燃香/蜡烛 | 时级 | 民间 | 在香/烛上标记刻度，燃烧一截即知时间 |
| 钟楼钟声 | 时级 | 城镇 | 仅在少数大城存在，用人力敲钟报时 |
| 公鸡打鸣 | 粗略 | 村野 | "鸡叫三遍"即天快亮了 |

**无机械钟**。仅有教会钟楼在特定时辰人工敲钟报时。磨坊镇没有钟楼——酒馆顶楼挂了一口从船上拆下来的铜钟，老板偶尔在午时敲几下。

## 八、跨文化时间认知差异

### 8.1 精灵

精灵的时间认知与人类截然不同。精灵寿命长达350-500年，他们对"紧急"的定义远远慢于人类。一个精灵说"过几天"——可能是指一个月后。精灵不使用纪元体系——他们以森林的周期纪年（"铁木花开的那个春天""上一次橡果大年"）。

### 8.2 矮人

矮人使用与人类相同的曜日系统，但他们对时间的执着远超人类。矮人合约精确到"契约签订后第47日的午时三刻"。矮人地下的计时依赖于精密的漏壶——误差极小。矮人常说："准时是诚意的一半。"

### 8.3 兽人

兽人没有书面的月份名称。他们以自然事件标记时间："下一次月圆""雪融化的时候""马群产驹的季节"。格鲁姆汗国的各部族有自己的年数——以现任可汗的统治年数计算，而非全大陆统一的纪元。

### 10.3 天气连续性

天气不是每日独立随机——前一天影响后一天：

- 晴 → 次日晴40% / 多云30% / 阴15% / 雨10% / 其他5%
- 多云 → 次日多云35% / 阴25% / 雨20% / 晴15% / 其他5%
- 阴 → 次日雨40% / 阴30% / 多云15% / 晴10% / 其他5%
- 小雨 → 次日阴30% / 小雨25% / 多云20% / 晴15% / 大雨10%
- 大雨/暴雨 → 次日小雨30% / 阴25% / 多云20% / 大雨15% / 晴10%
- 连续晴3日以上 → 次日阴/雨概率各+10%
- 连续雨3日以上 → 次日放晴概率+15%

# 怪物与野兽参考模板

> 此文件为GM提供非人生物的标准化数据。人类敌人（士兵/匪徒/佣兵/骑士等）详见各国 `军事.md`。
> 所有数值为"普通"品质——GM可调整为弱化/精英/首领版。

---

## 使用说明

- **弱化版**：HP/伤害 -30%，用于幼崽或病弱个体
- **精英版**：HP/伤害 +50%，附带一个额外能力
- **首领/远古版**：HP翻倍，伤害 +50%，附带 2-3个特殊能力
- 所有生物均有独立的生态说明，便于GM融入叙事

---

# 一、野兽（自然野生动物）

## 1.1 草原狼

> 埃塞拉无尽草原和艾尔德兰东部丘陵最常见的群居捕食者。

| 属性 | 值 |
|------|-----|
| HP | 25-35 |
| 攻击 | 撕咬：5-10；爪击：3-6 |
| 防御 | 1（皮毛） |
| DEX/CON | 8/6 |
| 先手值 | 16 |
| 栖息地 | 草原、丘陵、疏林 |
| 习性 | 群居（3-12只）。由头狼领导。不攻击大型群体。 |
| 特殊 | 群体协作 — 每多一只围攻同目标的狼，命中+2（最多+6） |
| 士气 | 损失过半后撤退 |
| XP | 10-20/只 |

## 1.2 黑熊

> 森林中的大型杂食动物。不主动攻击人类，除非被激怒或保护幼崽。

| 属性 | 值 |
|------|-----|
| HP | 90-120 |
| 攻击 | 掌击：12-20；熊抱（擒抱后每轮10-15） |
| 防御 | 4（厚皮+脂肪） |
| STR/CON | 13/10 |
| 先手值 | 8 |
| 栖息地 | 温带森林、山地 |
| 习性 | 独行。秋季为冬眠大量进食时更具攻击性。 |
| 特殊 | 激怒后不停止 — HP<20%才可能放弃 |
| XP | 80-120 |

## 1.3 野猪

> 暴躁的森林动物。受伤后更加凶猛。

| 属性 | 值 |
|------|-----|
| HP | 60-80 |
| 攻击 | 獠牙冲撞：10-16 |
| 防御 | 5（厚皮+泥甲） |
| STR/CON | 9/9 |
| 先手值 | 10 |
| 栖息地 | 森林、灌木丛 |
| 习性 | 独行或小群（3-6只）。公猪具强烈攻击性。 |
| 特殊 | 狂暴 — HP<50%时伤害+5，自身每轮受1-3反冲 |
| XP | 40-60 |

## 1.4 山狮

> 潜伏于山地和丘陵的伏击型掠食者。极擅潜行。

| 属性 | 值 |
|------|-----|
| HP | 40-55 |
| 攻击 | 扑击（跳跃攻击）：10-18（先手+DEX突袭+5）；撕咬：8-12 |
| 防御 | 2（皮毛） |
| DEX | 12 |
| 先手值 | 20（潜行中突袭） |
| 栖息地 | 灰塔丘陵、裂脊山脉、铁峰山麓 |
| 习性 | 独行。从高处或隐蔽处发起突袭。 |
| 特殊 | 潜行突袭 — 如未被发现状态下先手攻击，首轮伤害×1.5 |
| 士气 | 中 — 受伤后可能撤退 |
| XP | 50-70 |

## 1.5 沼泽巨蟒

> 黑水沼泽的巨型蛇类。靠缠绕窒息杀死猎物。

| 属性 | 值 |
|------|-----|
| HP | 80-110 |
| 攻击 | 缠绕（擒抱判定 STR DC 15）：每轮窒息伤害 8-12，持续至挣脱 |
| 防御 | 6（厚鳞片） |
| STR/CON | 12/10 |
| 先手值 | 6（水中14） |
| 栖息地 | 黑水沼泽、河流湿地、炎州雨林 |
| 习性 | 埋伏于水中或泥中。攻击靠近水边的大型猎物。 |
| 特殊 | 水中伏击 — 水中先手值+8；被缠绕的生物每轮需STR判定挣脱（DC 15） |
| 士气 | 低 — HP<40%时释放猎物逃跑 |
| XP | 70-100 |

## 1.6 猛爪巨鹰

> 翼展超过4米的大型猛禽。可从高空俯冲捕猎山羊甚至小孩。

| 属性 | 值 |
|------|-----|
| HP | 30-45 |
| 攻击 | 俯冲爪击：12-18（仅首轮）；啄击：6-10 |
| 防御 | 3（羽毛+薄皮） |
| DEX | 13 |
| 先手值 | 18（飞行） |
| 栖息地 | 铁峰山脉、龙骨山脉、裂脊山脉 |
| 习性 | 成对筑巢于绝壁高处。不主动攻击成年人——除非巢穴受威胁。 |
| 特殊 | 俯冲 — 飞行状态下首轮攻击伤害×1.5且先手+5 |
| 士气 | 中 — 伴侣受伤时死战不退 |
| XP | 40-60 |

## 1.7 毒刺蝎

> 赤砂沙漠和炎州干旱地区的巨型蝎子，体长可达1米。

| 属性 | 值 |
|------|-----|
| HP | 25-40 |
| 攻击 | 钳夹：4-8；尾刺：2-6 + 中毒（CON DC 14，失败则每轮4-8毒伤，持续3轮） |
| 防御 | 7（几丁质甲壳） |
| CON | 8 |
| 先手值 | 12 |
| 栖息地 | 沙漠、干旱岩石区 |
| 习性 | 独行。夜间活跃。白天藏于沙下或石缝中。 |
| 特殊 | 沙中伏击 — 可藏于沙下，突袭时先手+8 |
| 士气 | 低 — 受伤后迅速逃跑 |
| XP | 25-40 |

## 1.8 黑水鳄

> 黑水沼泽和南方河流中的大型爬行动物。伏击型掠食者。

| 属性 | 值 |
|------|-----|
| HP | 70-90 |
| 攻击 | 咬合：14-22（一旦咬住则每轮自动造成8-12，需STR DC 14挣脱） |
| 防御 | 9（厚皮+骨板） |
| STR/CON | 11/10 |
| 先手值 | 4（水中突袭14） |
| 栖息地 | 黑水沼泽、炎州河口、南方湿地 |
| 习性 | 浮于水面伪装成浮木。突袭岸边饮水的猎物。 |
| 特殊 | 死亡翻滚 — 咬住后额外造成15-25伤害（每场战斗仅使用1次） |
| 士气 | 中 — 重伤后可能松口逃离 |
| XP | 60-90 |

# 二、难度快速参考

| 难度 | 组合示例 | 适合等级 |
|------|---------|---------|
| 极简单 | 1-2只草原狼 或 单只毒蛇 | Lv.1 |
| 简单 | 狼群（3-4只）或 单只毒蝎 | Lv.1-2 |
| 普通 | 野猪 或 山狮 | Lv.2-3 |
| 较难 | 狼群（5-6只）或 黑熊 | Lv.3-4 |
| 困难 | 黑熊 或 沼泽巨蟒 或 猛爪巨鹰 | Lv.4-6 |
| 极难 | 沼泽巨蟒 或 黑水鳄 | Lv.6-8 |
